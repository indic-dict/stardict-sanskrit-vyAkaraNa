हर्षक्षयो धातुक्षयः
( ग्लास्यति जग्लौ जग्लतुः जग्लाथ जग्लिथ जग्लौ जग्लिव जग्लिम ग्लाता ग्लास्यति ग्लायतु अग्लायत् ग्लायेत् ) आशिषि ( ग्लायात् ग्लायेत् ) वान्यस्य संयोगादेःइति घुमास्थादेरन्यस्य संयोगादेरार्द्धधातुके क्ङिति लिङि वा एत्वम् ( अग्लासयीत् ) सगिटौ ( जिग्लासति जाग्लायते गाग्लेति जाग्लीतः ) ई इल्यघोःइति क्ङिति सार्वधातुके श्नाप्रत्यस्य चाघोरभ्यस्तस्य चकारस्य ईकारविधानादीत्वं लोटि हेरपित्त्वात् ङित्वादीत्वं, ( जाग्लीहि अजिग्लपत्प्रग्लापयति ) ग्लास्त्रावनुवमां चइति अनुपसर्गस्य वा मित्वादीत्वं, ( जाग्लीहि अजिग्लपत्प्रग्लापयति ) ग्लास्त्रावनुवमां चइति अनुपसर्गस्य वा मित्त्वमुक्तम् सुग्लायतीति ( सुग्लः ) "आदेच'' इत्यात्वं प्रागेव प्रत्ययोत्पत्तेरनैमित्तिकमिति "आतश्चोपसर्गे'' इति उपसृष्टादाकारान्तात् विहितः को भवति ( ग्लानः ) "संयोगादेरातो धातोर्यण्वतः''इति निष्ठानत्वम् ( ग्लास्नुः )"ग्लाजिस्थश्च क्स्नुः'' ( सुग्लानं भवता ) आतो युचइति ईषदादिषु कृच्छ्राकृच्छ्रार्थेषूपपदेषु युच् स्त्रियां ( ग्लानिः ) "ग्लाज्याहात्वरिभ्यो निः'' ( ग्लौः) "ग्लानुदीभ्यां डौः, डित्त्वात् टिलोपः "च्विरव्यम्'' एषां च्वयन्तानामेवाव्ययत्वमिति नियमात् "कृन्मेजन्तः'' इत्यव्ययत्वाभावत् ( ग्लावौ ग्लावः ) इत्यादि यथासम्भवं नेयम् ग्लै गात्रविमान इति क्क चित्पठ्यते, गात्रविमानः कानिहानिः 888
