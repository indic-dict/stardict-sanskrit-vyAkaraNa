प्रक्षेपणे
 ( मिनोति मिनुते ममौ मिम्यतुः मिम्युः ममाथ ममिथ मिम्यथुः मिम्य ममौ मिम्यिव मिम्यिम ) "मीनातिमिनोतिदीङां ल्यपि च'' इत्येज्विषये ल्यपि च प्रागेव प्रत्ययोत्पत्तेरात्वं, तस्य चानैमित्तकत्वान्न "द्विर्वचनेऽचि'' इति स्थानिवत्त्वं, थलीट्पक्षे "आतो लोप इति च'' इत्याल्लोपः ( मिम्ये मिम्यिषे मिम्यिवहे माता मास्यति मास्यते मिनोतु मिनु मिनवानि मिनुताम् मिनुष्व मिनवै अमिनोत् अमिनुतम् मिनुयात् मिन्वीत ) आशिषि ( मीयात् मासीष्ट ) यासुटि कित्त्वान्नात्वम् "अकृत्सार्वधातुकयोः'' इति दीर्घः ( अमासीत् अमासिष्टाम् ) आत्वे "यमरमनमाताम्'' इति सगिटौ परस्मैपदे अन्यत्र ( अमास्त अमासाताम् मित्सति मित्सते ) "इको झल्'' इति सनः कित्त्वात् "अज्झनगमां सनि'' इति दीर्घे "सनिमिमा'' इतीसादेशे "सस्यार्द्धधातुके'' इति सस्य तत्वे "अत्र लोपोऽभ्यासस्य'' इत्याभ्यासलोपः ( मेमीयते मेमेति मेमयीति ) "मिनोति'' इति श्तिपा निर्द्देशान्नात्वम् ( मापयति अमीमपत् ) आत्वे पुक् ( प्रमाप्य ) "ल्यपि च'' इत्यात्वम् ( प्रमयः इषत्प्रमयः ) [ निमिमीलियां खलचोः प्रतिषेधः ] इत्यात्वप्रतिषेधः ( मित्रिमम् ) "ड्वितः क्त्रिः'' "क्त्रेर्मम्नित्यम्'' ( मीरः ) "शुषिचिमिञां दीर्घश्च'' इति क्रन्दीर्घौ मयते इति शपि प्रणिदाने मातीति लुकि माने मिमीते इति च श्लौ माने शब्दे च मीयत इति श्यनि 4
