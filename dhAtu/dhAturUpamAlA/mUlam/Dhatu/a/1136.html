लक्ष
 
दर्शन,अङ्कनयोः
 ( लक्षति ) इत्यादि ( लक्ष्मीः ) "लक्षेर्मुद् च'' इतीकारप्रत्ययो मुडागमश्च ( लक्ष्मीवान् लक्ष्मणः ) "लक्ष्म्या अच्च'' इति मत्वर्थीयो नकारप्रप्रत्यय इकारस्य चाकारः ( लक्ष्यम्, लक्षणम् ) अचो यति ल्युटि च णेर्लोपः लक्ष्यमधीतेवेद वा ( लाक्ष्यिकः ) लाक्षणिकोऽप्येवम् उञ्छादौ लक्ष्य लक्षणे इति पाठाठ्ठक् शम लक्ष आलोचन इत्याकुस्मीयो भविष्यति 5
