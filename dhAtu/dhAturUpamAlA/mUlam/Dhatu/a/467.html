व्यथ
 
भये,सञ्चलनयोः
 (व्याध्रात व्यथते ) भीत्रार्थानां भयहेतुःइति बिभेत्यर्थानां त्रायत्यर्थानां च प्रयोगे भयहेतुः कारकमपादानमिति व्याघ्रस्यापादानत्वात्पञ्चमी ( व्यथते विव्यथे ) "व्यथो लिटि'' इत्यभ्यासस्य सम्प्रसारणं, न सम्प्रसारणे सम्प्रसारणम्इति पूर्वस्य यणो यकारस्य सम्प्रसारणपरत्वात्सम्प्रसारणं न भवति, सम्प्रसारणइति च सम्प्रसारणभाविनीत्यर्थः अस्मादेव वचनसामर्थ्यात्पूर्वं परस्य सम्प्रसारणं क्रियते पश्चादन्यस्य, अत्र च वेलायामस्ति सम्प्रसारणपरत्वम् अथ वा "व्यथो लिटि' इति सम्प्रसारणं हलादिशेषापवाद इति परस्यैव प्रसज्यते न पूर्वस्येति न चोद्यावकाशः ( व्यथितेत्यादि विव्यथिष्यते वाव्यथ्यये वाव्यथीति वाव्यति व्यथयति अविव्यथत् ) घटिवच्चिण्णमुलोरुदाहार्यम्_ न व्यथत इति (अव्यथ्यः) "राजसूर्य'' इत्यादिना निपातितः (अव्यथी ) "जिदृक्षि'' इतृयादिना नञ्पूर्वात्ताच्छीलिक इनिः ( व्यथा ) पित्त्वादङ् ( विथुरः ) "विदिभिदि'' इत्यत्र व्यथेः सम्प्रसारणं चेति वचनात्कुरचि सम्प्रसारणम् ( विधुरः ) "व्यथेः सम्प्रसारणं धः किच्च'' इति वचनादुरचि सम्प्रसारणं थकारस्यधकारः व्यथ दुःखचलनयोरिति दुर्गः भयचलनयोरिति प्रकाशः तवर्गचतुर्थान्तोयं दिवादौ 752
