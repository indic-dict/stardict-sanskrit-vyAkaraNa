चिन्तायाम्
 ( मातुः स्मरति ) इत्यादि ह्वरतिवत् "अधीगर्थ'' इति कर्मणिशेषे षष्ठी ( सुस्मूर्षते ) "ज्ञाश्रुस्मृ'' इति तङ् स्मरयतीत्याध्याने घटादिपाठात्, अन्यत्र ( स्मारयति असस्मरत् ) "अत्स्मृदृत्वर'' इत्याभ्यासस्याकार इत्वापवादः अयमनेकार्थत्वात् विस्मरणे चित्तादपक्रमणे वर्त्तते अनन्तरं द्वृ इति क्वचित्पठ्यते ( द्वरतीत्यादि ) स्मरतिवत् द्वर्यते संह्रियते इति ( द्वारम्, ) द्वारे नियुक्तो ( दौवारिकः ) "तत्र नियुक्तः'' इति ठकि "द्वारादीनां च'' इति वृद्धिप्रतिबन्ध एजागमश्च द्वारादिषु तदादिविज्ञानात् द्वारपालस्येदं, ( दौवारपालम् ) इत्यत्रापि वृद्धिप्रतिबन्ध एजागमश्च भवति ( द्वाः ) इति ण्यन्तात् क्विपि 916
