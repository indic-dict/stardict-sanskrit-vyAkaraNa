उदाता अनुदात्तेतः।
टुभ्राजृ
 
टुभ्राशृ
 
टुभ्लाशृ
 
दीप्तौ
 उदाता अनुदात्तेतः अत्र स्वामी आद्यावधिकृत्य दुरुभयसङ्गीत्येके इत्याह त्रयोपि ट्वित इति मैत्रेयः वृत्तिकारोप्यत्रैवानूकूः यदाह "वाभ्राश'' इत्यत्र टुभ्राशृ टुभ्लाशृ इति तथा "फणां च सप्तानाम्'' इत्यत्र नासपदमञ्जर्योः त्रयोपि ट्वितः पठ्यन्ते तस्मादयमेव पाठो बहूनां संमतः भ्राजतेः पठितस्यापीह पाठः फणादित्वेन किति लिट्येत्वाभ्यासलोपविकल्पार्थः, ट्वित्त्वं तु न फलं भवति तत्रापि टुशब्दस्यासङ्क्तुं शक्यत्वात् अनेनैव सिद्धे तस्य पाठो व्रश्चादिषनिवृत्त्यर्थ इति इति तत्रैवोक्तं, ( भ्राजते बभ्राजे भ्रेजे भ्राजितेत्यादि बाभ्राष्टि अबाभ्रट् ) इत्यादौ व्रश्चादिषत्वम् ( भ्राजथुः) "टितो ऽथुच्'' न भ्राजयतइति ( नभ्राट् )"नभ्राण्नपात्'' इति नञः प्रकृतिभावः चङ्परे णौ " भ्राजव'' इत्यादिना उपधाह्रस्वविकल्पनात् (अबिभ्रजत् अबभ्राजत् ) अस्यापि ऋदित्त्वं पूर्ववत् अनुदात्तेत्त्वमात्रुलम् ( भ्राष्ट्रम् ) ष्ट्रन् भ्राष्ट्रमिन्ध इति (भ्राष्ट्रमिन्धः)कर्मण्यम् "भ्राष्ट्राग्न्योरिन्धे'' इति मुम् ( भ्रशते बभ्राशे भ्रेषे भ्राशिता भ्राशिष्यते भ्राशतम् अभ्राशत भ्राशेत अशषि भ्राशिषीष्ट अभ्राशिष्ट ) सार्वधातुके "वा भ्राश'' इत्यादिना पक्षे शपोपवादे श्यनि ( भ्राश्यते भ्राश्यताम् अभ्राश्यत भ्राश्येयेति भवति बिभ्राशिषते बाभ्राष्टि भ्राशयति अअबभ्राशत् ) ऋदित्त्वान्नोपधाह्रस्वः ( भ्लाशति ) सर्वं भ्राशतिवत् ( भ्राशथुः भ्लाशथुः ) द्वित्त्वात् थुच् 810
