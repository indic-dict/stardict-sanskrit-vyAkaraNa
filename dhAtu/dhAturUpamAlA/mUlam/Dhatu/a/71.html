आदाने
 ( कोकते चुकुके कोकितेत्यादि चुकुकिषते चुकोकिषते कुकित्वा कोकित्वा ) "रलो व्युपाधात्'' इतिकित्त्वविकल्पः ( कुकितमनेन, कोकितमनेन प्रकुकितः प्रकोकितः प्रकुकितवान् प्रकोकितवान् ) "उदुपधात्'' इति कित्त्वविकल्पः ( प्रकोकनम् प्रकोकणम् ) हलश्चेजुपधात् हलादेरिजुपधात्परो यः कृत् ततस्थस्य नकारस्य उपसर्गस्थान्निमित्तात्परस्य वा ण इति णत्वविकल्पः "कृत्यचः'' इत्यस्यापवादः ( कोकः ) देवसेवमेषादयः पचादिषु द्रष्टव्या इत्यादिशब्दस्य प्रकारार्थत्वादिदुपधात्कं बाधित्वा ऽच् प्रत्ययः "सलिकल्यनिमहिब( 1 ) ( 1 ) वडीतिः पञ्चपादीवृत्तौ नास्ति डिभडिभण्डिशण्डिपिण्डितुण्डिकुकिभूभ्य इलच्'' इति इलच् (कोकिला) जातिलक्षणङीषं बाधित्वाऽजादित्वात् टाप् वृकः( वर्कते ववृके वर्कितेत्यादि विवर्किषते वरीवृक्यते ) रीगृदुपधस्य चॠत्वतोभ्यासस्य रीगिति रीगागमः, ( वरीवृकांचक्रे ) इत्यदावल्लोपस्य स्थानिवत्त्वात् "न धातुलोपः'' इति वा रुगभावः ( वर्वक्ति वरिवार्क्ति वरीवर्क्ति वर्वृकीति वरिवृकीति वरीवृकीति ) रुग्रिकौ च लुकिइति ऋत्वतोभ्यासस्य रुग्रिकौ चकारद्रीक् ईटि "नाभ्यस्तस्याचि पिति'' इत्यगुणत्वम् एवं वर्वृक्त इत्यादि वर्वकां चकारेत्यादौ यङ्लुकोऽनैमित्तकत्वात् "न धातुलोपः'' इति गुणनिषेधाभावः ( अवर्वृ कीत् अवर्वर्क् स्थातेत्यत्र ) "स्कोः'' इति कलोपो भिन्न्पदत्वान्नेति सीकतावुक्तम् लुङि सार्वधातुकाश्रये गुणे "नाभ्यस्तस्य'' इति निषिद्धेपि सिजाश्रयो भवत्येव प्रत्यलक्षणेन, ( अवर्वर्कीत् अवर्वर्किष्टामित्यादि ) हलन्तलक्षणा वृद्धिः "नेटि'' इति निषिध्यते ( वर्कयति अववर्कत् अवीकृकत् ) उरृत्चङ्परे णावुपधाया ॠकारस्य वा ऋकार इत्यृकारः इररा रामपवादः वृकः इगुपधलक्षणः कः मृगविशेष आयुधजीविसंघविशेषश्च तत्र द्वितीयार्थाभिधायिनो "वृकाट्टेण्यण्'' इति स्वार्थे टेण्यणि वर्किण्यःञ्यादयस्तद्राजाः"पूगाञ्ञ्योग्रामणीपूर्वात्'' इत्यारभ्यापादपरिसमाप्तेर्वक्ष्यामाणाः प्रत्ययास्तद्राजसंज्ञा इति टेण्यस्य तद्राजत्वे "तद्राजस्य बहुषु तेनैवास्त्रियाम्'' इति लुकि बहुषु वृकाः तेनैवेति वचनाद्यदि बहुत्वं तद्राजकृतमेवेत्यवगतेर्वार्केण्यमतिक्रान्ता अतिवार्केण्या इत्यादावर्थान्तरकृते बहुत्वे लुग् न भवति तथा अस्त्रियामितिवचनात् स्त्रियां टित्त्वात् ङीवि हलस्तद्धितस्य हल उत्तरस्य तद्धितयकारस्य लोपो भवतीति परत इति "आदेः परस्य'' इति यकारलोपे च इत्यकारलोपे ( वार्केणीवार्केण्यौ वार्केण्य ) इति भवति वर्करस्तरुणः पशुः बाहुलकादरच् 92
