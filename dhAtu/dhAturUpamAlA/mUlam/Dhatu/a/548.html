उबुन्दिर
 
निशामने
 
निशामनं चक्षुर्विज्ञानम्
( बुन्दति बुबुन्द बुन्दितता अबुन्दत् अबुन्दीत् बुबुन्दिषति बोबुन्द्यते बोबुन्ति ) हौ ङित्वादनुनासिकलोपे ( बोबुन्दि बुन्दते बुबुन्दे इतृयादि ) उदितृवात् क्त्वायामिङ्विकल्पनात ( बुन्दित्वा बुत्त्वा ) निष्ठायां ( बुन्नम् ) नलोपे "रदाभ्याम्" इति निष्ठानत्वम् (बुबुदः) [घञर्थे कञादीनांके द्वेभवतः] इति द्विवचनम् पृषोदरादित्वान्न इलादिशेषः 861
