बीजसन्ताने
 ( वपति उवाप उवप्थ उवपिथ ) उवपिवेत्यादि यजिवत् प्रणिवपति "नेर्गद''इति णत्वम् ( प्रण्यवपत् ) "अङ्व्यवायेऽपीष्यत' इति णत्वम् (प्रवपाणि ) "आनि लोट्'' इति उपसर्गस्थान्निमित्तात्परस्य लोट्सम्बन्धिन आनेर्नकारस्य णत्वम् ( वाप्यम्, ) "आसुयुवपि'' इति ण्यत् ( वापी ) ऋदृसृवपीनामिति ग्रहादिपाठादिनिः (उप्त्रिमम्) "ड्वितः क्त्रिः'' सम्प्रसारणं, "क्त्रेर्मम्नित्यम्'' इति मप् ( वापिः ) इञ् "स्वपिवपिभ्य''इति इतीञ् (वापी) "कृदिकारादक्तिनः''इति डष (वप्रः) "वृधिवपिभ्यां च रन्" इति रप्रत्ययः बीजसन्तानं क्षेत्रे बीजविकरणम्, अयं गर्भाधानेऽपि दृश्यते, यथा मा वः क्षेत्रे परबीजान्यवाप्सुः छेदनेऽपि दृश्यते केशान्वपतीति983
