आञ्छि
 
आयामे
 ( आञ्छति आञ्छ ) अत्र लिट्यभ्यासह्रस्वे "अत आदेः''इत्यत्रतपरकरणं स्वाभाविकह्रस्वपरिग्रहार्थमिति न्यासकारहरदत्तादिभिः प्रतिपादितत्वाद्र दीर्घाभावाद् नुडभावे सवर्णदीर्घः अन्ये तु तपरकरणं मुखसुखार्थमिति आनाञ्छेत्युदाहरन्तीति मैत्रेयः ( आञ्चितेत्यादि आञ्छिच्छिषति ) "न न्द्राः'' इति नकारवर्जं द्वितीयैकाच् द्विरुच्यते ( आञ्छ्यति आञ्छिच्छत् ) 208
