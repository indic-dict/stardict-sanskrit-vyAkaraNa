ञिधृषा
 
प्रागल्भ्ये
 ( धृष्णेति दधर्ष दधृषतुः दधर्षिथ दधृषिव धर्षिता धर्षिष्यति धृष्णोतु अधृष्णोत् धृष्णुयात् आशिषि ( धृष्यात् अधर्षीत् दिधर्षिषति दरिधृष्यते दरीधृषीति, दरिधृष्टि, ) इत्यादि ( धर्षयति अदीधृषत् अदधर्षतदधर्षत्) "उरृद्वा'' ( धृष्यः ) "ऋदुपधात्'' इति क्यप् दधृक् दधृषौ ) "ऋत्विक्'' इत्यादौ निपातनात् क्विनि द्वित्वम् पदान्ते च कुत्वम् ( धृष्णुः ) "त्रसिगृधि'' इत्यादिना धृषेः क्नुः ( धृष्णक् धृष्णजौ ) [ धृषेश्चेति वक्तव्यम् ] इति नजिङ् (दुर्धर्षणः) "भाषायां शासियुधिधृषिमृषिभ्यो युच्'' ( धर्षित्वा ) "न क्त्वा सेङ्'' इत्याकित्वाद्गुणः ( धृष्टो वृषलः ) "धृषशसी वैयात्ये'' इत्यनिट्त्वम् "आदितश्च'' इत्यनिट्त्वे सिद्वे वचनमिदं नियमार्थं, तेन वैयात्यादन्यत्र धर्षित इतीड् भवति भावादिकर्मणोर्न वैयात्ये धृषिरस्तीति तत्र नित्यनिषेधार्थं न भवति वैयात्यादन्यत्र भावादिकर्मणोर्विकल्प एव भवति ( धॄष्टं, धर्षितम् प्रधृष्टः, प्रधर्षितः ) इति, तत्रेट्पक्षे "निष्ठा शीङ्'' इत्यादिना कित्त्वाभावाद्गुणः ञीत्त्वं वर्तमाने क्तार्थम् धृष प्रसहन इति युजादौ 23
