 ( चिल्लति चिल्लः ) तिल गतौ इति दुर्गमैत्रयौ तिल्लेत्यन्ये ( तेलतीत्यादि तिल्लः ) इगुपधलक्षणः कः तिल स्नेहनइति तौदादिकाद्वा तिलानां भवनं क्षेत्रं (तिल्यं तैलीनम् ) "विभाष ल'' इति यत्खञौ तिलेभ्यो हितं (तिल्यम् ) "खलयवमाषतिलवृषबह्मणश्च इति चतुर्थ्यान्ताद्धितार्थे यतृ निष्फलस्तिलः ( तिलपेजः ) तिलपिञ्जः ) "तिलान्निष्फलात्पिञ्जपेजौ'' इति तौ (तिलकम् ) संज्ञायां क्वुन् तिल स्नेहनइति चुरादौ च 527
