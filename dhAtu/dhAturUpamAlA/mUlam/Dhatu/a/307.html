सेचने
 शेवृ खेवृ मेवृ इत्यप्येकइति मैत्रेयः (सेवते सिषेवे सेवितेत्यादि सिसेविषते) सिसेविषते ) "स्तौतिण्योरेव'' इत्यषत्वम् ( सेषेव्यते सेवयति असिपेवत् ) एषाम् ऋदित्त्वात् "ण्ण्चङि'' इति न ह्रस्वः ( परिषेवयते ) "परिनिविभ्यः सेवसितसयसिवुसहसुट्स्तुस्वञ्जाम्'' इति षत्वं "सात्पदाद्योः'' इति निषेधापवादः "प्राक् सिताददव्यवायेपि'' इति वचनात्पर्यषेवतेत्यादावपि भवति ( परिषिषेविषते ) इत्यषणीति नियमः "स्थादिष्वभ्यासेन खभ्यासस्य'' इत्यर्नेन बाध्यते एवं विनिभ्यामप्युदाहार्यम् ( सेवित्वा सेवितः क्विप् सयः ) अत्र मैत्रेयादयोदत्यादिमपि वदन्ति तथा च "परिनिविभ्यः सेव'' इत्यत्र न्यासेपि षीपदेशस्य "सात्पदाद्योः'' इति निषेधे प्राप्ते ऽपरस्य त्वप्राप्ते वचेति दिदं षेपदेशलक्षणपर्युदासे सेक्तृवर्जमिति स्त्यायतिवदनुपादानाद्भाष्यकारस्यानभिमतत्वमिव प्रतीयते अत एव "परिनिविभ्यः'' इत्यत्र हरदत्तः सेवतिर्भूवादिष्वनुदात्तेदित्याह न तु सेवतीति ( पेवते ग्लेवते तेवते म्लेवते म्लेवते शेवते खेवते मेवते ) क्लेवृ इत्यपि क्वचित्पठ्यते 501
