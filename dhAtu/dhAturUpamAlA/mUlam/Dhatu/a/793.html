दीपी
 
दीप्तौ
 ( दीप्यते दिदीपे, दीपिता दीपिष्यते दीप्यताम् अदीप्यत दीप्येत, दीपिषीष्ट ) लिङौ ( अदीपि अदीपिष्ट ) "दीपजन'' इति ते कर्तरि चिण्विकल्पः ( दिदीपिषते देदीप्यते देदीप्ति दीपयति अदीदिपत् अदिदीपत् ) "भ्राजभास' इत्यादिना णौ चङि ह्रस्वविकल्पः (दीपः) उगुपधलक्षणः कः ( दीपिता ) "सूददीपदी क्षश्च'' इति अनुदात्तेल्लक्षणे युच्प्रत्यये प्रतिषिद्धे, ताच्छीलिकस्तृन् (दीप्रः) "नमिकम्पि'' इत्यादिना रः ( दीप्तिः ) "क्तिन्नाबादिभ्यः'' इति क्तिन् "तितुत्र'' इतीण्निषेधः ( दीप्तः, दीप्तवान् ) ईदित्वादिनट्त्वम् 42
