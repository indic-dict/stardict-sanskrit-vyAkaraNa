 ( ध्रस्नाति दध्रास दध्रसिथ ध्रसिता ध्रसिष्यति ध्रस्नातु ध्रसान अध्रस्नात् ध्रस्नीयात् ) आशिषि ( ध्रस्यात् अध्रासीत् अध्रसीत् दिध्रसिषति दाध्रस्यते दाध्रस्ति ध्रासयति अदिध्रसत् ध्रसित्त्वा ध्रस्त्वा ) उदित्त्वादिड्विकल्पः, ( ध्रस्तम् ) "यस्य विभाषा'' इत्यनिट्त्वम् अयं चुरादौ अत्र क्षीरस्वाम्युकारं धात्ववयवमाह तन्मते-( उध्रस्नाति ) इत्यादि, ( उध्रसांचकार औध्रस्नात् ) इत्याद्युदाहार्यम् अनवयवत्वे स्वादिवद् ध्रसु इति ब्रूयादिति तस्याभिप्रायः स एव भूवादौ यदाह--
