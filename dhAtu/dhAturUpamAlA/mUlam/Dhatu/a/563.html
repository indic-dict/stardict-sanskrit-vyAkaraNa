भ्रक्ष
 
अदने
 भ्लक्षेति क्षीरस्वामी भक्षेति मैत्रेयः तत्र "गतिबुद्धि'' [ भक्षेरहिंसार्थस्य प्रतिषेधः ] इति वार्तिकस्य भक्ष्यति पिण्डीं देवदत्तः भक्षयति पिण्डीं देवदत्तेनेति ण्यन्ताद्धेतुमण्णौ प्रयोज्यस्य कर्मत्वाभावं दर्शयता भाष्यकृता भक्षतिरनभिहितः सति हि अणिकर्त्ता स णावितयसामञ्जस्यादस्मादेव हेतुमण्णवुदाहृतं स्यात्, तथा वृत्तितद्व्याख्यानेष्वपि [ भक्षेर्हिंसार्थस्य ] इत्यत्र चुरादिण्न्तादेव भक्षेर्हेतुमण्णावुदाहियते ( भ्लक्षति भ्लक्षिता बिभ्लक्षिषति बाभ्लक्षयते बाभ्लष्टि ) "स्कोः'' इति कलोपः लीटि हौ ( बाभ्लड्ठि ) धित्वष्टुत्वजश्त्वानि लङि तिप्सिपोर्हल्ङ्यादिलोपो ( अबाभ्लट् भ्लक्षयति अबभ्लक्षत म्लक्षयति अबभ्लक्षत् भ्लक्षते ब्भलेक्षे ) इत्यादि योत्र प्लक्ष इति केषां चित् पाठः सोपि प्रक्षरतीति प्लक्षइति प्रपूर्वात् क्षरतेः प्लक्षशब्दस्य भाष्यकारेण व्युत्पादितत्वादनार्ष एव तथा च श्रुतिरपि "तस्य शिरश्छिता मेध्यं प्राक्षा रयात् स प्लक्षोऽभवत्तत्प्लखस्य प्लक्षत्वं, यत् प्लक्षशाखेति' इत्र लत्वं कपिलादित्वाद् द्रष्टव्यम् 877
