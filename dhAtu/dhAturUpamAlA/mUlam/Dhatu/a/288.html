 ( द्रमति दद्राम द्रमितेत्यादि ) लुङि "ह्मयन्त'' इति वृद्धिनिषेधात् ( अद्रमीत् दिद्रमिषति दन्द्रमीति दन्द्रमीति दन्द्रन्ति दन्द्रान्त इत्यादि ) उत्तमे "म्वोश्च'' इति नत्वे णत्वम् ( दन्द्रण्मीत्यादि) ( अदन्द्रन् )इत्यत्र तिप्सिपोर्हल्ङयादिलोपे "मो नो धातोः'' इति नत्वे "पदान्तस्य'' इति णत्वं न भवति ( द्रमयति अदिद्रमत ) अमन्तत्वान्मित्त्वम् ( अद्रमि अद्रामि द्रमन्द्रमम् द्रामन्द्रामम् ) "चिण्ण्मुलोः'' इति दीर्घविकल्पः स्यादिषु चिण्वद्पिक्षे दीर्घविकल्पे णिलोपेच (द्रमिष्ते द्रामिष्त)इत्यादि शुद्धे पुनरिटि णेर्गुणायादेशायोः(द्रमयिष्यते), इत्यादि अण्यन्तात् चिणि "नोदात्तोपदेश'' इति वृद्धिनिषेधात् ( अद्रमि )इत्येवेति एवं णिति कृति च (द्रमःद्रमक) इत्यादि ( दन्द्रमणः)"जुचङ्कम्य'' इत्यादिना तच्छीलादौ यङन्तादस्माद्युचि अल्लोपयलोपौ (द्रमिडः) "ञमत्ताडुः'' इति डः "नेड्विशि कृति'' इति निषेधो बाहुलकान्न्भवति ( हम्मति जहम्म हम्मितेत्यादि जिहम्मिषति जेहम्म्यते जंहम्मीति जंहन्ति जंहन्त ) इत्यादावनुपधात्वान्नोपधादीर्घः लङि हलङयादिसंयोगान्तलोपयोः "मो नो धातोः'' इति नत्वे (अजंहन् हम्मयति अजहम्मत् हम्मित्वा हम्मतिः) ज्ञानं, गत्यर्थो बुद्ध्यर्थः, न हम्मतिः ( अहम्मति ) अज्ञज्ञनम् यद्वा अहमिति मतिः अहंशब्दो मान्तोहङकारवचनोव्ययम् ( मीमति मिमीम मिमीमतुः मीमितेत्यादि मिमीमिषति मेमीम्यते मेमीमीति मेमीन्त इत्यादि लङि अमेमीन् मीमयति अमिमीमत् मीमित्वा मीमितः ) अयं शब्दे चेति क्कचित्पठ्यते 464
