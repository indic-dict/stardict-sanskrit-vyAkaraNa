उदात्तः अथाजन्तानुभयपदिन आह ।
श्रिञ्
 
सेवायाम्
 उदात्तः ( श्रयति शिश्राय शिश्रियतुः शिश्रियुः शिश्रयिथ शिश्राय शिश्रियिव ) अचि इत्यादेशः अन्यत्र वृद्धिगुणौ अत्र "असंयोगाल्लिट् कित्'' इति लिटः कित्त्वात् "श्रयुकः किति'' इति प्राप्तस्येटो निषेधस्य क्रादिनियमेन बाधः ( श्रयिता श्रयिष्ति श्रयतु अश्रयत् ) श्राशिषि ( श्रीयात् ) "अकृत्सार्वधातुकयोः'' इति दीर्घः ( अशिश्रियत् ) "णिश्रि'' इति चङ् लघूपधगुणादन्तरङ्गत्वादियङ् ( श्रयते शिश्रिये इत्यादि शिश्रयिषति शिश्रीषति) "सनीवन्तर्द्ध'' इत्यादिना इड्विकल्यः अनिट्पक्षे "अज्ज्ञन'' इति दीर्घः "इको ज्ञल'' इति कित्त्वादगुणत्वं, ( शेश्रीयते शेश्रयीति शेश्रेति श्राययति अशिंश्रयत ) वृद्ध्यायौ, णौ कृतस्य स्वानिवत्त्_वात् श्रीशब्दस्य द्विर्वचनम् ( श्रायः ) "श्रिणीभूवोनुपसर्गे'' इति घञ् कथं पतनान्ताः समुच्छ्रया इति , "कृत्यल्श्युटो बहुलम्'' इति वचनात्, एवं च उदिश्रायतीति श्तिपा निर्देशात् यङ्लुगन्तात् ( उछ्राय ) इति अजेव ( विश्रयी ) "जिदृक्षिविश्रि'' इति ताच्छीलिक इनिश्H ( श्रित्वा श्रितः) "श्र्युकः किति'' इति इण्निषेधः ( कष्टश्रितः ) "द्वितीया श्रिता'' इति समासः ( श्रेणिः ) "वहिश्रिश्रुयुद्रुग्लाहात्वरिध्यो नित्'' इति निः ( श्रेणीकृताः ) ''श्रेण्यादयःकृतादिभिः'' इति तत्पुरुषः [श्रेण्यादिषु च्व्यर्थवचनम् ] इति तत्र वार्त्तिकम् [ क्किब् वचिप्रच्छ्यायतस्तुकटप्रजुश्रीणां दीर्घोऽसंप्रसारणं च ] इति क्किब्दीर्घो इयङ् स्थानित्त्वात्, नदीत्वाभावात् ( हे श्रीः ) इत्यत्र "अम्बार्थनद्योर्ह्रस्वः'' इति न ह्रस्वः ङिद्वचनेषु "ङिति ह्रस्वश्च'' इति पक्षे नदी त्वादाटि ङेश्चामि ( श्रियै श्रियाः श्रियामित्यादि ) अन्यदेयङि ( श्रिये, श्रियः श्रियि ) षष्ठीबहुवचने च "वामि'' इति नदीत्वे नुतिट ( श्रीणां, श्रियाम् श्रिमन्यं ब्राह्मणकुलम् )भाष्यकारवचनात् श्रीशब्दस्य ह्रस्वः मुमभावश्च ( श्मश्रु ) "श्यनि श्रयतेर्हुन्'' इति श्यन्शब्दे उपपदे हुन्, टिलोपः, ( श्मश्रुणः, ) पामादित्वान्नः 881
