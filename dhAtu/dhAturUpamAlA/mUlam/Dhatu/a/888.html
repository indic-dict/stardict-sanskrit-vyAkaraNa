कृञ्
 
हिंसायाम्
 ( कृणोति कृणुते चकार चक्रतुः चकर्थ चक्र चकार चकर चकृव) क्रादिसूत्रे कृग्रहणेनायमपि गृह्यतइतीण्निषेधः (चक्रे चक्राते चकृषे चकृढ्वे चक्रे चकृवहे ) अगुणवृद्धिविषये अजादौ यणादेशः ( कर्त्ता करिष्यति करिष्यते कृणोतु कृणुताम् अकृणोत् अकृणुत कृणुयात् कृण्वीत ) आशिषि ( क्रियात् ) "रीङ् ऋतः'' "रिङ् शय'' इति रिङादेशः ( कृषीष्ट अकार्षीत् अकार्ष्टाम् अकृत अकृषाताम् ) आत्मनेपदे लिङ्सिचोः "उश्च'' इति कित्त्वान्न गुणः (चिकीर्षति) स्तृञ्वत्सनः कित्त्वे दीर्घः ( चेक्रीयते ) "अकृत्सार्वधातुकयोः'' इति वर्तमाने "च्वौ च'' इति च"रीङ् ऋत'' इति रीङ् ( चरीकर्ति ) इत्यादि ( कारयति अचीकरत ) सर्वत्र कृग्रहणेषु कृञग्रहणेषु च करोतिरेव वृत्तिकारादिभिरङ्गीकृत इत्ययं नोदाह्रियते, क्रादिसूत्रे तु कृग्रहणेनास्यापि ग्रहोऽङ्गीकृत आत्रेयादिभिः कारा बन्धने इति भिदादौ पठ्यते तत्रास्मादङित्यात्रेयः न्यासकारस्तु करोतेः 7
