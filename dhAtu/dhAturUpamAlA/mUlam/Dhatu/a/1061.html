ओविजी
 
भय,चलनयोः
 ( विनक्ति विङ्क्तः विञ्जन्ति विनक्षि विनज्मि विञ्ज्वः विञ्ज्मः विविजतुः विविजिथ विविजिव ) "विज इट्'' इति इडादिप्रत्ययस्य पित्त्वेऽपिङित्त्वान्न गुणः ( विजिता विजिष्यति विनक्तु विङ्क्तात् विङ्ग्धि विनजानि अविनक् अविङ्क्ताम् अविनक् अविनजम् विञ्ज्यात् विञ्ज्ययाताम् विज्यात् विज्यास्ताम् ) लिङौ ( अविजीत् अविजिष्टाम् विविजिषति वेविज्यते वेविजीति वेवेक्ति वेजयति अवीविजत् विजित्वा ) "नक्त्वा सेट्'' इत्यकित्त्वेऽपि "विज ईट्'' इति ङित्त्वान्न गुणः ( विग्नः विग्नवान् ) ईदित्त्वादनिट्त्वम् ओदित्त्वान्निष्ठानत्वम् ( वेगितम् ) इति घञन्तात्तारकादित्वादितजित्युक्तम् ( वेवेक्ति वेविक्ते ) इति श्लौ गतम् उद्विजत इति शे 23
