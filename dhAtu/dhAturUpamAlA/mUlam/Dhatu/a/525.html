उद्गिरणे
 
उद्गिरणमन्तर्गतस्य मुखान्निसारणम्
अत्र सुधाकर उद्गिरण इति पठन्निपानात् तत्र इद्धातोरित्याह (वमति ववाम ववमतुः) वादित्वान्नेत्वाभ्यासलोपौ भागवृत्तौ त्वनयोर्विकल्पेन वेमतुरित्युदाहृतं, तद्भाष्यादिषु चिरन्तनेषु कुत्रापि न दृष्टन् ( वमिता अवमीत् ) "ह्म्यन्त'' इति न वृद्धिः ( विवमिषति वंवम्यते वंवन्ति वंवान्तः ) हेरपित्त्वात् ङित्त्वात् "अनुनासिकस्य'' इति दीर्घेऽनुस्वारे ( वंवांहि अवंवन् वमयति वामयति ) "ग्लास्त्रावनुवमामनुपसर्गाद्वा'' इति मित्वनिषेधः ( वामः ) "नोदात्तोपदेशस्य'' इति वृद्धिनिषेधस्य [ आचकिममिवमीनाम ] इति निषेधः (वान्तः वान्तवान् ) "आदितश्च'' इति चकारस्यानुक्तसमुच्चयार्थतया निष्ठायामनिट्त्वमिति वृतौ भाष्यवार्त्तिकयोरनुक्तमपीदं प्रयोगबाहुल्यादुक्तमिति हरदत्तः, तदेवं काशयादीनामुदित्पाठः प्रत्युक्तः ( वमथुः ) टित्त्वादथुच् 834
