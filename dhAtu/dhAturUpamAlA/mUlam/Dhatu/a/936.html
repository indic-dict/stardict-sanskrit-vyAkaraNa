पूरणे
 ( उभति उवोभ ऊभतुः ) "अभ्यासस्यासवर्णे'' इति गुणविषेय उवङ् अन्यत्र सवर्णदीर्घः ( उम्भति उम्भाञ्चकार ) इत्यादि ( उभौ ) इगुपधात्कः नित्यं द्विवचनान्तः सर्वादित्वात्सर्वनामत्वं, तत्कार्याणि "सर्वनाम्नस्तृतीया''इत्यादीनि यथासंभवं नेयानि उभाववयवावस्य उभयो मणिरित्यत्र "संख्याया अवयवे तयब्'' इति तयपि तस्य "उभादुदात्तो नित्यम्'' इत्ययजादेश उदात्तः ( उभाञ्जिलि उभयाञ्जलि उभाहस्ति उभायाहस्ति उभापाणि उभायापाणि उभाबाहु उभयाबाहु ) इत्यत्रैव निर्द्देशादिचो लोपः, प्रत्ययलक्षणेनेजन्तत्वादव्ययीभावत्वादव्ययत्वमिति न्यासे द्विदण्ड्यादिगणपाठादेवैषां साधुत्वे सिद्धे "द्विदण्ड्यादिभ्यश्च'' इति सूत्रमिजन्तत्वविज्ञानार्थमिति, तस्य च प्रयोजनमव्ययीभावसंज्ञेति हरदत्तादयः तिष्ठद्ग्रप्रभृतिषु इजिति पठ्यते, एवं समुदायनिपातनाच्चार्थविशेषे एवावतिष्ठन्ते, तथा च वृत्तिः द्विदण्डि प्रहरति द्विमुसलि प्रहरति)एवमुभाञ्जलीत्यादीनामपि प्रहारो ऽन्यपदार्थः केनोभ्यते इति (कुम्भः) कर्मणि घञ् ( अयस्कुम्भः ) "अतः कृकमिकंसकुम्भ'' इति ''विसर्जनीयस्य सः'' अत एव कुम्भेतिनिर्द्देशात्कशब्दाकारस्य लोपः पररूपता वा ( कुम्भी ) गौरादित्वात् "जातेरस्त्री'' इति वा ङीष् ( अयस्कुम्भी ) लिङ्गविशिष्टपरिभाषया सत्वम् कुम्भीव नासिका ऽस्य ( कुम्भीनसः ) "अञ् नासिकायाः संज्ञायाम्'' इत्यादिना नासिकाया नसादेशोऽच् प्रत्ययश्च 39-40
