 ( पुणति ) इत्यादि ( मातरि निपुणः ) इगुपधलक्षणः कः ( पितरि निपुणः ) "साधुनिपुणाभ्याम्'' इति तद्युक्तात्सप्तमी अप्रतेरित्यपनीय [ अप्रत्यादियोगइति वक्तव्यम् ] इत्युक्तम् प्रत्यादयश्च प्रतिपर्यनव इति पाणिनिनोक्ताः, तेन तैर्योगे द्वितीयैव, (पितरं प्रति निपुण) इत्यादि ( आचारनिपुणः ) "पूर्वसदृश'' इत्यादिना तृतीयासमासः ( नैपुण्यम् ) ब्राह्मणादित्वात् ष्यञ् षित्त्वान् ङीष् ( नैपुणी ) "हलस्तद्धितस्य'' इति यलोपः ( नैपुणम् ) युवादित्वादण् ( अनिपुणः ) तत्पुरुषो बहुव्रीहिर्वा तस्येदमा ( नैपुणम् अनैपुणम् ) "नञः शुचि'' इत्यादिना पूर्वपदस्य वा वृद्धिरुत्तरपदस्य तु नित्यम् 52
