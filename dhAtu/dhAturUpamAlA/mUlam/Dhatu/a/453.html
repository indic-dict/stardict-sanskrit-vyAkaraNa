रुच
 
दीप्तौ,अभिप्रीतौ च
 (रोचते देवदत्ताय मोदकः) "रुच्यर्थानां प्रीयमाणः'' इति प्रीयमाणस्य संप्रदानत्वाच्चतुर्थी (रुरुचे रोचिता अरुचत् अरोचिष्ट रुरुचिषते रुरोचिषते रुचित्वा रोचित्वा ) भृशं रोचतइत्यर्थे यङ्लुगिष्यत इतिवृत्तिन्यासादौ (रोचयति रोचयते)"अणावकर्मकात्''इति कर्त्रभिप्राये प्राप्तेपरस्मैपदे"न पादम्याङ्यमाङ्यसपरिमुहरुचिनृतिवदवसः'' इति निषिध्यते ( रोचयति रोचनः रोचनम् ) नन्दिवासिदमिरुच्यसीधिवर्द्धिरोचिभ्यो ण्यन्तेभ्यः संज्ञज्ञयामिति नद्यादिपाठे ल्लुयः ( रोचिष्णुः ) "अलंकत्'' इत्यादिनेष्णुच् ( रुच्यः ) "राजसूय'' इत्यादीना कर्त्तरि क्यप् निपात्यते (रोकः) घञ् "चजोः कुधिण्ण्यतोः'' इति निष्ठायामनिटः कुत्वं वदतो वार्त्तिककारस्य "विभाषा श्यावारोकाभ्याम्'' इति निपातनात् कुत्त्वम् (रोचिः) "शुचिरुच्योः संज्ञाम्'' इति सिः ( रुक्मम् ) "युजिरुचिनितां कश्च'' इति मन्प्रत्ययः तकारस्य ककारः ( रोचकः ) "क्कुन् शिल्पिसंज्ञयोः'' इति क्कुन् 732
