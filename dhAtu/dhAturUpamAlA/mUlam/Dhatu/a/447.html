वृहि
 
शब्दार्थे
 वृंहितं करिगज्जितम् वृहिर इति चन्द्रगुप्तौ तत्फलं च (अवृहत् ) अवर्हीदिति पक्षे इछ्सिद्धिरित्याह क्षीरस्वामी वृह वृहि भाषार्थौ वर्हत इति प्राधान्ये गतम् तुहिर् दुहिर् उहिर् अर्दने (तोहति तुतोह तोहिता) लुङि "हरितो वा'' इति पक्षे अङ् ( अतुहत् अतोहीत् ततुहिषति तुतोहिषति तुहित्वा ) "रलो व्युपधाद्'' इति कित्त्वम् ( तुह्मते तोतोढि तोतूढ इत्यादि तोहयति अतूतहत् तुहितम् ) "वेपितुहोर्ह्रस्वश्व'' इतीन् प्रत्ययः, ह्रस्ववचनं वेपत्यर्थम् (देहतीत्यादि तोहतिवत् ) दिहिर्दुहिर्मेहतिरोहती इत्यनिट्कारिकायां मैत्रेयादायो नास्य ग्रहणमिच्दन्ति ( उहति उवोह ऊहतुः ऊहुः ओहितेत्यादि ) उषिवत्प्रक्रिया, लुङि (मा भवानुहत्) 722
