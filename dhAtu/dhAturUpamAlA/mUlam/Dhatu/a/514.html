 टलतीत्यादि चलतिवत् ( ट्वलति टट्वाल -ट्वलतुः ट्वलिते त्यादि अट्वालीत् ट्वालः, ट्वलः ) स्थलति तस्थाल स्थलिता असथालीदित्यादि तिस्थलिषति ) "स्तौतिण्योरेव'' इति न षत्वम् ( स्थलति तस्थालः स्थलिता अस्थालीदित्यादितिस्थलिषति ) "स्तौतिण्योरेव'' इति न षत्वम् ( तास्थल्यते तास्थलन्ति स्थालयति अतिष्ठलत् स्थलः स्थालःस्थलम् स्थली ) "जानपद'' इत्यत्राकृत्रिमायां भूमौङीष् अन्यथा ( स्थला स्थालः स्थलम् स्थली ) "जानपद'' इत्यत्राकृत्रिमायां भूमौ ङीष् अन्यथा ( स्थला स्थाली) गौरादित्वात् ङीष् (विष्ठलम् ) "विकुशमिपिभ्यः स्थलम्'' इति मूर्द्धन्यः ! एवं क्कादिभ्यः ( कपिष्टलम् ) "कपिष्ठलो गोत्रे'' इति र्मूद्धन्यो निपात्यते अन्यत्र कन्यत्र कपीनां स्थलं, (कपिस्थलम् ) 821
