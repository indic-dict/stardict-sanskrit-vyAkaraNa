मर्दने
 ( म्रदते मम्रदे म्रदिता मिम्रदिषते माम्रद्यते माम्रदीति माम्रत्ति म्रदयति अमम्रदत् ) "अत् स्मृदृत्त्वरप्रथम्रद'' इत्यादिनाभ्यासस्यात्वं मित्त्वां पवादः मृदुः "प्रथिम्रदि'' इतृयादिना कुप्रत्ययः सम्प्रसारणं च नैतेन मृद्धातोरभावोनुतव्यः तत्र संप्रसारणस्यार्थं वक्तव्यत्वात्, सत्त्व एव हि मृद्धातोर्मृदेत्यादिनिदैशोपपत्तिः मृदोरपत्यं, ( मार्देयम् ) "शुभ्रादिभ्यश्च'' इति ढक् "ढे लोपोकद्रवाः'' इति उकारलोयः ( मृद्वी ) गुणवचनत्वात् ङीष् ( मृद्विका ) कनि "केणः'' इति ह्रस्वः ( म्रदिमा ) पृथ्वादिभ्य इमनिचि "र ऋतः'' इति रः तदभावे ( मार्दवम्, ) स्वदुमृदुनं इदं (सौवादुमृदवम् ) "द्वारादीनां च'' इति वृद्धिप्रतिषेध ऐजागमश्च मृद क्षोद इति ऋकारोपधः क्र्यादौ 755
