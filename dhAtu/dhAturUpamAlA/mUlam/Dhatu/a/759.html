अदने
 आदान इत्येके अदर्शन इत्यपरे "धात्वादेष्षस्सः'' ( स्नुस्यति सुष्णोस, सुष्णुसतुः सुष्णोसिथ, सुष्णु, सुष्णोस, सुष्णुसिव, ) "आदेशप्रत्यययोः'' इत्यभ्यासात्परस्य षत्वे ष्टुत्वं, "रषाभ्याम्'' इति णत्वं वा ( स्नोसिता स्नोसिष्यति स्नुस्यतु अस्नुस्यत् अस्नुस्यः स्नुस्येत् ) आशिषि ( स्नुस्यात् अस्नौसीत्, अस्नौसिष्टाम्, अस्नौषीः, अस्नोषिष्ट, ) "नेटि'' इति सिचि वृद्धेरभावः ( सुस्नुसिषति, सुस्नोसिषति, स्नुसित्वा स्नोसित्वा ) "रलोव्युपधात्'' इति वा कित्त्वम् "स्तौतिण्योः'' इत्पभ्यासात्परस्य न षत्वम् ( सोष्णुस्यते सोष्णुसीति, सोष्णोस्ति, सोष्णुस्तः स्नोसयति असुष्णुसत् स्नुषा ) इगुपधलक्षणे क टाप् सुषामादित्वात् षत्वम् औणादिक इत्यात्रेयः ष्णुस निरसनपइत्येके मिच्चेत्यात्रेयमैत्रेयौ उदुपधं चाहतुः स्वामिकाश्यपौ तु अदुपधममुमेव पठित्वा ष्णुस अदर्शनइति द्रमिहा इत्युकारोपधं पक्षान्तरमाहतुः तत्र मित्त्वं "जनीजॄष्ष्णसु'' इतिपाठाश्रयेण ( स्नस्यति, सस्नास ) इत्यादि, षोपदेशस्य प्रयोजनं णौ ( स्नसयति, असिष्णसत्, सिष्णसयिषति ) इति 6
