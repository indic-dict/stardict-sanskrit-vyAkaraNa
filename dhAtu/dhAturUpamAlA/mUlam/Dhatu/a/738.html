अनुदात्त आत्मनेभाषः।
ओहाङ्
 
गतौ
 अनुदात्त आत्मनेभाषः ( जिहीते, जिहाते, जिहते, जिहीषे, जिहीध्वे, जिहे, जिहिवहे जहे हाता हास्यते जिहीतां, जिहाताम् अजिहीत्, अजिहाताम्, अजिहत जिहीत, जिहीयाताम् ) आशिषि ( हासीष्ट, हासीयास्ताम् अहास्त, अहासाताम् ) इत्यादि माङ्वत् ( जिहासते, जाहायते ) "घुमास्था'' इतीत्वे जहातीति निर्देशादस्याग्रहः अत एव "जहातेश्च क्विः'' इति हिभावोऽपि नेति हात्वेति भवति ( हापयति अजीहपत् हानः, हानवान् ) "ओदितश्च'' इति निष्ठानत्वम् जिहीते भावानिति ( हायनः संवत्सरः, ) "हश्च व्रीहिकालयोः'' इति ण्युट् एको हायनोऽस्या ( एकहायनी, त्रिहायणी, चतुर्हायणी ) "दामहायनान्ताच्च'' इति सङ्ख्यादेर्दामहायनान्ताच्च बहुव्रीहेः स्त्रियां ङीप्, "त्रिचतुर्भ्यो हायनस्य'' इति णत्वम् असंज्ञात्वाद् न "पूर्वपदासंज्ञायाम्'' इति णत्वप्राप्तिः एते च ङीब्णत्वे वयस्येव स्मर्येते तेनह न भवतः, एकहायनाशाला त्रिहायना चतुर्हायना इति ( हानिः ) "ग्लास्माज्याहाभ्यो निः'' इति निप्रत्ययः भृञादयस्त्रयो गताः 7
