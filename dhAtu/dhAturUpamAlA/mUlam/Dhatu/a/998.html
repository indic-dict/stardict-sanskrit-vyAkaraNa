 मैत्रेयेणैते न पठ्यन्ते आत्रेयादयस्तु पठन्ति ( फ्फुडति पुस्फोड ) खयः शेषः ( चुडति चुचोड व्रुडति वुव्रोड ) इत्यादि अत्र क्रुड भृडनिमज्जनइत्येके इत्यात्रेयः मैत्रयस्तु क्रुडं पठित्वा भृड डत्यप्येके इत्याह ( क्रुडति क्रोडः)घञ् (क्रिडा) अश्वानामुरः, टाबन्तोऽयंस्वभावतो विशेषविषयः कल्याणी क्रीडा यस्याः सा ( कल्याणक्रोडा ) बहुव्रीहावुपसर्जनह्रस्वत्वे "न क्रोडादिबह्वचः'' इति स्वाङ्गलक्षणो ङीष् निषिध्यते क्रोडादिषु टाबन्तस्य पाठाद् भुजा तामात्रवचनस्य क्रोडशब्दस्य बहुव्रीहौ स्वाङ्गलक्षणो ङीष्विकल्प एव भवति (कल्याणक्रोडी, कल्याणक्रोडा ) मयूरीति (भृडति) इत्यादि व्रश्चादय उदात्ता उदात्तेतः 109-110- 111
