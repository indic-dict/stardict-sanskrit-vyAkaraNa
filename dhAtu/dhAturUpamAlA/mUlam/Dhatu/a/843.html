काङ्क्षायाम्
 ( ताम्यति ) इत्यादि शमिवत् ( ताम्बूलम्, ताम्बूरम् ) "तमेर्बुक्च'' इत्यूरीलचौ प्रत्ययौ णिद्वद्भावाद्वृद्धिः, बुगागमश्च ( ताम्रम् ) "शमितम्योर्दीर्घश्च'' इति रकि दीर्घंः ( तमसा ) नदी "अत्यमिचमितकि'' इत्यसच् (तमः,) असुन् ( अवतमसम्, सन्तमसम्, अन्धतमसम् ) "अवसमन्धेभ्यस्तमसः'' इत्यच्समासान्तः ( तमिस्रा ) "ज्योत्स्नातमिस्रा'' इति संज्ञायां मत्वर्थीयो रः, उपधायाश्चेकारः स्त्रीत्वमतन्त्रम्, अन्यत्रापि दृश्यते, तमिस्रं नभ इति वृत्तौ ( तमसाकृतम् ) "ओजस्सहोम्भस्तमसस्तृतीयाया'' इत्यलुक् ( तमी ) इन्नन्तात् डीष् 94
