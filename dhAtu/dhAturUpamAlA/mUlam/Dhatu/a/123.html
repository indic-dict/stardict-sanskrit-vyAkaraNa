 ( ध्रजति दध्राज ध्रजितेत्यादि अध्रजीत् अध्राजीत् ) "अतो हलादे0'' इति वा वृद्धिः ( दिध्रजिषति दाध्रज्यते दाध्रजीति दाध्रक्ति )"चोः कुः'' झलि पदान्ते च "खरि च'' इति चर्त्वम् ( ध्राजयति अदिध्रजत् ध्रञ्जति दध्रञ्ज ध्रञ्जितेत्यादि दिध्रञ्जिषति दाध्रञ्ज्यते दिध्रञ्जिषति दाधृज्यते दाध्रञ्जीति दाध्रङ्क्ति ) लङि तिप्सिपोर्हलङ्यादिलोपे संयोगान्तलोपे च ( अदाध्रन् धर्जति दधर्ज धर्जितेत्यादि दिधर्जिषति दरीघृज्यते ) "रीगृदुपधस्य च'' इति रीक् ( दर्ध्रजीति दरिधृजीति दरीधृजीति दर्धर्क्ति दरिधर्क्ति दरीधर्क्ति )" रुग्रिकौ च लकि'' इति रुग्रिग्रीकः "नभ्यस्तस्याच्''इति पक्षे न गुणः, एवं तसादौ ( दर्धर्क्तः) इत्याद्युदाहार्यम् धर्जयति अदध्रजत् अदीधृजत् उरृद्वा धर्जित्वा) "न क्त्वा सेट्'' इत्यकित्त्वाद्गुणः, धृज्यम्, "ऋदुपधाच्चाक्लृपिचृतेः'' इति क्यप् धर्जः "चजोः कुधिण्ण्यतोर्निष्ठायामनिटः'' इति वार्त्तिकसूत्रेणकुत्वम् ( धृञ्जति दधृञ्ज धृञितेत्यादि दिधृजिषति दरीधृञ्ज्यते ) पूर्ववद्रीक् ( दायधृञ्जीति दाधृङ्क्तीत्यादि धृजिवत् ) लङि तिप्सिपोर्हलङ्यादिलोपसंयोगान्तलोपयोरदाधृन् संयोगान्तलोपस्यासिद्धत्वात् अलघूपधत्वात् प्रत्ययलक्षणेन न गुणः ( धृञ्जयति अदधृञ्जत् ध्वजति ध्वञ्जतीत्यादि ) ध्रजधृजिवत् तरङ्गिणीकाश्यपसंमतास्तु आद्यौ व्रजव्रजी इति दन्त्योष्ट्यादी पठ्येते सर्वत्र तवर्गीयादीएव220
