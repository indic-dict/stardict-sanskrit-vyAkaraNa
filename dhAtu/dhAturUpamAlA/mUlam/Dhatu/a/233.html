 एतदादयस्तोभत्यन्ता उदात्ता अनुदोत्ततः तपिं तिपिं चापीत्यनिट्कारिकासु पाठात्. तिपिरेकोऽनुदात्तः ऋदित्पवर्गान्तात्मनेपदित्वसाम्याच्चेह पाठः क्षीरस्वामिना त्वयं सेडुदाहृतः, तत्काश्यपवृत्तिन्यासपदमञ्जर्यादिविरोधात् तपिं तिपिसिति व्याघ्रभूतिवचनविरोधात् चोपेक्ष्यम् ( तेपते तितेपे तितपिषे तितपिवहे ) क्रादिनियमादिट् ( तेप्ता तेप्त्स्यते तेपताम् अतेपत तेपेत आशिषि तिप्सीष्ट अतिप्त अतिप्साताम् ) झलि सिज्लोपः लिङ्सिचावात्मनेपदेषुइति इक्समीपाद्धलः परयोर्झलाद्योर्लिङसिचोः कित्त्वान्न गुणः आत्मनेपदेष्विति सिच एव विशेषणं नलिङः, परत्वासम्भवात् परस्मैपदिषु झलादित्वासंभवाच्च ( तितिप्सते ) हलन्ताच्चइति इक्समीपाद्धलादेः सनः कित्त्वाणौ चङि ह्रस्वाभावः ( तेपते तितेपे तेपिता तेपिष्यते तेपताम् अतेपत तेपेत तेषिषीष्ट अतीपिष्ट तितेपिषते, तेतिप्यते तेतेपीत तेतेप्ति स्तेपते तिष्टिपे तिष्टिपाते ) "अभ्यासे खयः शेषः, स्तेपितेत्यादि तेपिवत् ( तिस्तेपिषते तिस्तिपिषते, स्तिपित्वा, स्तेपित्वा ) "स्तौतिण्योरेव'' इत्यषत्वम् "रलो व्युपधात्'' इति वा कित्त्वम् ( तेष्टिप्यते तेष्टिपीति तेष्टेप्ति टुष्टिप्तः ) तिपिवत् ष्टपिः अभ्यासे खयः शेषो विशेषः दिपृ देपृ इति च काश्यपः 361
