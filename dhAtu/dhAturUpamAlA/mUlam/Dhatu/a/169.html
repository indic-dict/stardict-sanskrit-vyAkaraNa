ताडने
 ( तडते ) इत्यादि तण्डः अच् ( वतण्डः ) "वष्टिभागुरिरल्लोपमवाप्योरुपसर्गयोः'' इत्युपसर्गस्याल्लोपः वतण्डस्य गोत्रापत्यं, ( वातण्ड्यः वातण्डः ) "गर्गादिभ्यो यञ्'', "शिवादिभ्योण्'' इति यञणौ यदा त्विदमपत्यमाङ्गिरसं भवति तदा"वतण्डाच्च'' इति शिवाद्यणं बाधित्वायमेव अपत्ये तु स्त्री लिङ्गे "लुक् स्त्रियाम्'' इति लुक् "शार्ङरवाद्यो ङीन्" इति ङीना वतण्डी अनाङिरसे तु लुगभावात् वातण्ड्यायनी "प्राचां ष्फस्तद्धिते'' इति यञ्चन्तात् ष्फः, फस्यायनादेशः षित्करणसामर्थ्यात् स्वार्थे ङीष्ष्फप्रत्ययाभावे "यञश्च'' इति ङीषि "यस्येति च'' इत्यल्लोपे आपत्यस्य च तद्धितेनातिआपत्ययकारस्यानकारादौ भसंज्ञानिमित्ते तद्धितईति च परे लोपे वातण्डी अनातीति वचनात् वातड्यायनीत्यत्र यलोपो न भवति ( वितण्डा, तण्डुला ) "लुठितनितण्डिभ्य उलच् इत्युलच् 277
