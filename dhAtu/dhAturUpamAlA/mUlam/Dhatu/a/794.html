आप्यायने
 ( पूर्यते ) ( पूर्यते ) इत्यादि दीपिवत् ( अपूपुरत् ) "णौ चङि'' इति नित्यं ह्रस्वः ( पूरः ), इगुपधलक्षणः कः ( ऊर्ध्वपूरं ) पूर्यते "ऊर्ध्वे शुषिपूरोः'' इति कर्तृवाचिन्यूर्ध्वे ( 1 ) शब्दे इत्याधिकं पुस्तकान्तरे उपपदे णमुल् कषादित्वाद्यथाविध्युनुप्रयोगः ऊर्ध्वेस्सन् पूर्यत इत्यर्थः ( पूर्णः,पूर्णवान्, पूर्तिः)आबादित्वात् क्तिन् (पूर्णिः) इत्यौणादिकः पूर्णिमिमीति इति (पूर्णिमा) मूलविभुजादित्वात् कः ( चर्मपूरं ) स्तृणोति, ( उदरपूरं ) भुंक्ते "चर्मोदरयोः पूरेः'' इत्यनयोः कर्मणोरुपपदयोः णमुल्प्रत्ययः पूरयतेर्धातुसम्बन्धे विधानाद् धात्वन्तरस्य अनुप्रयोगः ( गोष्पदपुरं ) वृष्टो देवः, ( गोष्पद्मं वा ) "वर्षमाणऊलोपश्चास्यान्यान्यतरस्याम्'' इति कर्मण्युपपदे वर्षस्य इयत्तायां गम्यमानायां पूरयतेर्णमुल् ऊकारलोपश्चास्य विकल्पेन ( पूर्णः ) "वा दान्तशान्तपूर्ण'' इति क्ते णिलुगिडभावौ पक्षे निपात्येते अन्यदा "निष्ठायां सेटि'' इति णिलोपे ( पूरितः ) पूरयतीति ( पूरणः ) बाहुलकात्कर्तरि ल्युट् ( पूरुः ) बहुलवचनादुरप्रत्ययः 43
