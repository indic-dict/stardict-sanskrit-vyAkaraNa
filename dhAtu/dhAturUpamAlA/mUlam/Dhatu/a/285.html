 ( स्तनति तस्तान स्तनितेत्यादि तिस्तनिषति तंस्तन्यते तंस्तनीति संस्तन्ति ) ध्वनतिवत्प्रक्रिया ( स्तनयति अतिष्टनत् अभिनिष्टानः ) विसर्जनीयः "अभिनिसः स्तनः शब्दसंज्ञायाम्" इति षत्वम्, अन्यत्राभिनिस्तनति मृदङ्गः अत्र न्यासे सकारादिरयं पठितः तदस्य षोपदेशपर्युदासवाक्ये उपादनान्निर्मूलम् अत एव पुरुषकारे मृर्द्धन्यादिरेवायं दर्शितः ( स्तान्यम् ) ण्यत् ( स्तनः ) अच् ( स्तन्यम् ) "शरीरावयवाद्यत्'' इति भवार्थे यत् ( स्तनिनम् स्तनयतीति देवशब्दार्थस्य कथादिपठितस्य ( वनति ववान ववनतुः वनितेत्यादि वनितम् वनित्वा वतिः ) क्तिनि "तितुत्र'' इतीण्निषेधात् "अनुदात्तोपदेशवनतितनोत्यादीनामननुनासिकलोपो ज्ञलि क्डिति'' इत्यनुनासिकलोपे ( वतिः ) इति भवति अनुदात्तोपदेश यमिरमिमिहमिहनिगमिमन्यतयः ( प्रवत्य ) "वा ल्पपि'' इत्यनुदात्तोपदेशादीनां विधीयमानोयमनुनासिकलोपविकल्पो व्यवस्थितविभाषाविज्ञानात् नान्तेषु नित्यः, अन्यत्र विकल्प इतीह नित्यः ( वन्तिः ) क्तिच् "न क्तिचि दीर्घश्च'' इति दीर्घानुनासिकलोपयोर्निषेधः "अनुदात्तोपदेशबनति'' इतिश्तिपा निर्देशात् ( वंवान्त ) इत्यादावनुनासिकलोपो न भवति वनोतिनिवृत्त्यर्थं तु श्तिपा निर्देशा न भवति तनोत्यादित्वेन तस्यानुनासिकलोपस्येष्टत्वात् (वनिष्ठः) अपानं मांसविशेषश्च ऋतन्यक्षिवनिचत्यादिनेष्ठच् ( वनम् ) अच् वनानां समूहो ( वन्या ) "पाशादिभ्यो यः'' इति सूमूहे यः तदेततस्वभावात् स्त्रीलिङ्गम् ( वन्यम् ) दिगादित्वाद्भवार्थे यत् ( वानेयम् ) पूर्वनगरीतिनद्यादित्वात् ढक् शैषिकः ( वनिः वनी ) "कृदिकारादक्तिनः'' इति वा ङीष् 458
