कृती
 
छेदने
 उदात्त उदात्तेत् ( कृन्तति चकर्त चकृततुः चकर्तिथ चकृत चकृतिव कर्तिता कर्त्स्यति कर्तिष्यति ) "सेसिचि'' इत्यादिना सकाराद्वाविड्विकल्पः (कृन्ततु अकृन्तत् कृन्तेत् कृत्यात् ) लिङौ ( अकर्तीत् अकर्तिष्टाम् चिकृत्सति चिकर्तिषति चरीकृत्यते चरिकर्त्ति) इत्यादि (कर्तयति अचकर्तत् अचीकृतत् ) उरृद्वा (विकर्तनः) नन्द्यादिः ( कृतः कृतवान् कृतिः ) क्तिन् ( कृत्समुदकम् ) "सृव्रश्चिकृत्त्यृषिभ्यः कित्'' इति सप्रत्यय स कित् "तितुत्र'' इतीणिनषेधः ( कृत्स्नम् ) "कृत्यशुभ्यां क्स्नन्'' इति क्स्नन् बाहुलकादिडभावश्च 151
