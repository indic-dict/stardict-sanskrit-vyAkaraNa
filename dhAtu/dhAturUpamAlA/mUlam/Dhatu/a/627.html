ष्वञ्ज
 
परिष्वङ्गे
 स्वजते "दंशसञ्जस्वञ्जां शपि'' इति अनुनासिकलोपः ( सस्वजे, सस्वञ्जे सस्वजिषे सस्वञ्जिषे सस्वजिवहे ) "इन्धिभवतिभ्याम्'' इत्यत्र "श्रन्थिग्रन्थिदम्भिस्वञ्जीनां वा'' इति वक्तव्यमिति लिटः कित्वविकल्पनात् पक्षेऽनुनासिकलोपः तथा "सदिस्वञ्ज्योः'' इत्यत्र वृत्तिव्याख्यानेषु स्वञ्जेः संयोगान्तादपि लिटो विभाषा कित्त्वमिच्छन्तीति ( स्वङ्क्ता स्वङ्क्ष्यते स्वजताम् अस्वजत स्वङ्क्षीष्ट अस्वङ्क्त सिस्वङ्क्षते ) "स्तौतिण्योरेव'' इत्यषत्वम् ( सास्वज्यते सास्वङिक्त सास्वञ्जीति सास्वक्तः सास्वजति )लोटि हौ ( सास्वक्धि ) लङि तिप्सिपोर्हल्ङ्_यादिलोपे संयोगान्तलोपे ( असास्वन् स्वञ्जयति असस्वञ्जत्, प्रतिष्वजते ) "उपसर्गात् सुनोति'' इत्यादिना षत्वं, (प्रत्यष्वङ्क्त) "प्राक् सितादड्व्यवायेऽपि''इति षत्वं (परिष्वजते निष्वजते विष्वजते ) "परिनिविभ्यः'' इत्यादिना षत्वम् "उपसर्गात् सुनोति'' इत्यादिनैव सिद्धे स्तुस्वञ्ज्योरुपादानं "सिवादीनां वाङ्व्यवायेऽपि'' इति परिनिविभ्यः परत्वे अङ्व्यवाये सिवादिदित्वादनयोः षत्वविकल्पोऽयं यथा स्यादिति तेन पर्यष्वजत पर्यस्वजत इत्याद्युभयं भवति प्रतिसस्वजे इत्यादौ "स्थादिष्वभ्यासेन'' इत्यभ्यासात् परस्य प्राप्तं षत्वं "सदिस्वञ्ज्योः परस्य लिटि'' इति निवार्यते 957
