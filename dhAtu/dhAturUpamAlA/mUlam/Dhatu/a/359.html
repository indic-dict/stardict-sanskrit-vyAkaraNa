बन्धने
 ( मवति ममाव मेवतुः मविता अमवोत् अमावीत् ) "अतो हलादेः'' इति वा वृद्धिः ( मिमविषति मामव्यते मामवीति सामोति मामूतः मामवति मामोषि मामोमि मामावः मामूमः ) "ज्वरत्वरस्त्रिव्यविमवामुपधायाश्च'' ज्वरादिनामुपधावकारयोः क्कौ ज्ञलादावनुनासिकादौ च प्रत्यये परे ऊडित्यूडादेशे पिति गुणः मामवतीत्यत्र परत्वात् नित्यत्वात् अद्भावे ऽज्ञलादित्वं पर्यवसन्नमिति ऊण्न भवति मामावेतृयत्र वलि लोपे "अतो दीर्घो यञि'' इति दीर्घः "ज्वरत्वर'' इत्यत्र क्क चित् वृत्तिकारैः क्कौ ज्ञलादौ किङतीति न पठ्यते तत्र किङ्द्ग्रहणं संपातायातम् अवतेस्तुन्योरिति दर्शनात्, तथा अनुनासिकादिग्रहणं चावश्यमत्रानुवर्त्यम् अवतेर्मनिन्प्रत्यये तस्य च टिलोपे ओमिति दर्शनात् तथा च "न धातुलोप'' इत्यत्र कैयटे आस्त्रेमाणमिति उपादाय छन्दसत्वाद्वकारस्य ऊण् न भवतीत्युक्तम् ( मामोतु मामौतात् मामौहि मामवानि ) नित्यत्वादाटि अज्ञलादित्वं पर्यवसन्न्मित्यूडभावः लङि तिप्सिपोरूठि ( अमामोत् अमामोः ) मिपि नित्यत्वादम्भावे ( अवामवम् मावयति अमीमवत् ) वकारान्तानामूढ्भाविनां यङ्लुङ् नेति यदुक्तं तत् "च्छ्वोः''इति ऊढ्विषयं द्रष्टव्यम् क्किपि ( मूः, मुवौ ) 591
