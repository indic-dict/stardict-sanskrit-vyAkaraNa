अनुदात्ता उदात्तेतः।
रुह
 
बीजजन्मनि,प्रादुर्भावे च
 अनुदात्ता उदात्तेतः ( रोहति रुरोह रुरोहिथ रुरुहिव )क्रादिनियमाट्टि ( रोढा ) गुणढत्वधत्वष्टुत्यढलोपाः ( रोक्ष्यति ) "षढौः कः सि'' इति कत्वं ( रोहतु अरोहत् रोहेत् ) आशिषि (रुह्मात् अरुक्षत् ) "शल इगुपधादनिटः क्सः'' ( रुरुखति रोरुह्मते रोरोढि रोरूढः ) "ढलोपे'' इति दीर्घः ( रोरोक्षि ) लोटि सोर्हित्वे धित्वष्टुत्वलोपदीर्घेषु (रोरूढि ) लङि ( आरोरोट् अरोरूसम् रोहयति अरूरुहत् रोपयति अरूरुपत् ) "रुहः पोन्यतरस्याम्'' इति पक्षे हकारस्य पकारः, ( आरूढो वृक्षम् आरूढो वृक्षः) "गत्यर्थाकर्मकः'' इत्यादिना कर्त्तृकर्मणोः क्तः ( रोहितिः ) "हूसृरुहिपुषिभ्यइतिः'' इतीतिः ( रोहितः लोहितः ) "रुहेरितश्च लो वा''इति इतः पक्षे रेफस्य लत्वम् अलोहितो लोहितो भवति,(लोहितायते लोहितायति) लोहितादिडाज्भ्यः क्यष्च्वयर्थविषादच्व्यन्तात् भवत्यर्थे क्यष "वा क्यषः''ति पक्षे तङ् ( लोहित एव लोहितको मणिः) "लोहितान्मणौ''इति मणौवर्तमानात् लोहितशब्दात् स्वार्थे कन् ( लोहितकं मुखं कोपेन) "लोहितान्मणौ''इति मणौ वर्तमानात् लोहितशब्दात् स्वार्थे कन् (लोहितकं मुखं कोपेन) "वर्णे चानित्ये'' इति अनित्यवर्णे वर्त्तमानात् लोहितशब्दात् स्वार्थे कन्, स्त्रियां [लांहिताल्लिङ्गिबाधानं वा) इत्युक्तत्वात् "वर्णादनुदात्तात्'' इति पक्षे कनोन ङीब्विधिरिति (लोहितिकेति भवति) अन्यदा तु ङीम्नकारयोर्लिङ्गविशिष्टपरिभाषया कनि हस्वे च (लोहिनिकेति) लोहितेन रक्तो (लोहितकः पटः) "रक्तः'' इति लाक्षादिना रक्तेर्थे वर्त्तमानात् लोहितशब्दात्स्वार्थे कन्, अत्रापि स्त्रियां "लोहिताल्लिङ्गबाधनं वा'' इत्यस्यानुवर्तनात् (लोहितिका पटी लोहिनिका पटीत्युभसयं भवति रोहिषो मृगभेदः) "रुहेर्वृद्धिश्च'' इति टिषचि वृद्धिः स्त्रियां (रौहिषी) 844
