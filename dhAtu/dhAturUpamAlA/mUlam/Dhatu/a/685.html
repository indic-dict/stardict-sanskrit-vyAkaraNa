 नौतीत्यादि यौतिवत् (नुनावयिषति अनूनवत्) इत्यत्र पुयण्जपरत्वाभावादभ्यासस्य नेत्वं, ( नुनूषति, ) "सनि ग्रहगुहोश्च'' इत्यनिट्त्वे "इको झल्'' इति सनः कित्त्वे "अज्झनगमां सनि'' इति दीर्घः (प्रणौति) "उपसर्गादसमासेऽपि'' इति णत्वम् (प्रणुनाव) इत्यादौ "उभौ साभ्यासस्य'' इति साभ्यासस्यानितेर्णत्वविधानाज्ज्ञापकात्ततःपूर्वमेव द्विर्वचनमिति परस्य णत्वं न भवति, व्यवधानात् न च णत्वस्य द्विर्वचने पूर्वत्रासिद्धत्वेन णत्वं न भविष्यतीति वाच्यम् [पूर्वत्रासिद्धीयमद्विर्वचने] इत्युक्तत्वादित्यात्रेयः हरदत्तस्तु "उभौ साभ्यासस्य'' इत्यनेन [पूर्वत्रासिद्धीयमद्विर्वचन] इत्यस्यानित्यत्वं ज्ञाप्यत इति णत्वस्यासिद्धत्वाद् नुशब्दस्य द्विर्वचने परस्य व्यवधानादणत्वमाह ( आनुते सृगालः ) [ आङि नुपृच्छयोरुपसङ्ख्यानम् ] इति तङ्, आङ्पूर्वोऽयं स्वभावादुत्कण्ठापूर्वके शब्दने वर्तते, तदुक्तं न्यासे, अयं धातुरुत्कण्ठापूर्वके शब्दने आङोत्तरः प्रवर्त्ततइति कैयटपदमञ्जर्योरुदाहरणस्य सोत्कण्ठं शब्दं करोतीत्यर्थं उक्तः ( आनुवाते, आनुवते, आनुषे, आनुवे, आनुवहे, आनुनुवे, आनुनुविषे, आनुनुविध्वे, आनुनुविढ्वे, आनुनुवहे, आनविता, आनविष्यते, आनुताम्, आनुष्व, आनवै, आनवै, आनुत, आनुथाः, आनुवि आनुवीत आनुवीथाः आनुवीय ) आशिषि ( आनविषीष्ट आनविष्ट आनविष्ठाः, आनविषि आनुनूषते ) यङ्लुक्यप्ययन्तङ् भवति आनोनुते ) दीर्घान्तोऽयं तुदादौ 25
