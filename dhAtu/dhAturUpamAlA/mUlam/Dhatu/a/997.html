सञ्चलने
 स्फुरतिस्फुलत्योरित्यत्र न्यासे स्फुर स्फुल चलने इत्यनुपसृष्टं पठ्यते, अत्र केवलं शब्दो भिन्नोऽर्थस्त्वेक एव आत्रेयस्तु स्फुर स्फुरणे, स्फुल संचलने इतिपपाठ ( स्फुरति निस्फुरति निष्फुरति विस्फुरति विष्फुरति) "स्फुतिस्फुलत्योर्निर्निविभ्यः'' इति वा षत्वम् निर्निसोः साधारणमुक्तमुदाहरणम् ( पुस्फोर स्फुरितास्फुरिष्यति स्फुरतु अस्फुरत् स्फुरेत्, स्फुर्यात् ) "हलि च'' इति दीर्घः ( अपुस्फुरीत् पुस्फुरिषति ) सेट्त्वात् "हलि च'' इति न दीर्घः कुटादित्वात् "रलो व्युपधात्'' इत्यत्र न भवति ( पोस्फूर्यते पोस्फोर्ति ) दीर्घस्यासिद्धत्वाद्गुणः ( पोस्फूर्तः स्फोरयति ) "चिस्फुरोर्णौ'' इति वाऽऽत्वम् ( अपुस्फुरत्, ) आत्पपक्षे णौकृतमभ्यासे स्थानिवदित्यभ्यासस्यौकारान्तत्वे ह्रस्व इत्युकारः "ओः पुयण्जि'' इति इत्वं न भवति, सकारेण पवर्गपरत्वस्य व्यवधानात् एवं ण्यन्तात्सन्यपि पुस्फोरयिषति पुस्फारयिषतीति भवतः, मैत्रयस्तु स्फुर स्फुरणे इति पुथक् पठित्वा स्फर इत्येके, अदुपधः, (स्फुरति पस्फार) इत्याह (स्फुलति) इत्यादि स्फुरतिवत् णावत्वं त्वस्य न भवति तथा "हलि च'' इति दीर्घत्वमपि न भवति, अरेफान्तत्वात् 107-108
