खुर
 
च्छेदने
 खण्डने चेति मैत्रेयः ( खुरति खुरः ) इगुपधलक्षणः कः ( कल्याणखुरा ) "स्वाङ्गाच्चोपसर्जनात्'' इति ङीषो "न क्रोडादिबह्वचः'' इति निषेधः ( खुरणः ) [ खुरखराभ्यां नस् वक्तव्यः ] इति नासिकाया नस्भावः "अञ् नासिकायाः संज्ञायाम्'' इत्येव सिद्धे वार्तिकमचो निवृत्यर्थं, पक्षेऽच् प्रत्ययोऽषीष्यतइति वृत्तिः तेन ( खुरणसः) इत्यपिभवति 61
