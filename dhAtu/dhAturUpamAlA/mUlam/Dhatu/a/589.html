अभ्यासे
 ( मनति ) "पाघ्रा'' इत्यादिना मनोदशः शिति अन्यत्र ग्लैवत्मन्नावित्यादि ( आम्रायः ) घञि युक् ( आम्रातः ) आम्रातमनेनाम्राती व्याकरणे "इष्टादिभ्यश्च'' इति प्रथमान्तादनेनेत्यर्थे इनिः, सप्मीविधाने [ क्तस्येन्विषयस्य कर्मण्युपसंख्यानम् ] इति कर्मणि सप्तमी नीचभावभ्यस्यतीति ( निम्रम् ) सुपीति योगविभागात् "आतोऽनुपसर्गे''इति वा कः नेनींचभाववचनत्वेन मनत्यर्थस्थाविशेषणान्न तं प्रतीहोपसर्गत्वं, दिने आम्रायत इति ( द्युम्रम् ) घुशब्दो दिनचनम्, दृश्यते च धारयेरत्र ( द्यून् ) सुतरामाम्रायत इति (सुम्री) सुखम् घञर्थे कः 912
