 ( सनोति सनुते ) इत्यादि तनोतिवत् "धात्वादेः षः सः'' णत्वन्तु निमित्ताद्यपायादपैति विशेषस्तु "जनसनखनां सन्झलोः'' इति झलादौ सनि किङ्ति च नित्यमात्वम् "ये विभाषा'' इति यकारादौ क्डिति वात्वं च ( सिषासति ) "सनीवन्त'' इत्यादिनेड्विकल्पः इट्पक्षे-( सिसनिषति ) अत्र षण्परत्वात् "स्तौतिण्योरेव'' इति नयिमात् प्रकृतेस्सस्य न षत्वम् ( सात्वा ) उदित्त्वादिड्विकल्पः इटि ( सनित्वा सातः सातवान् ) "यस्य विभाषा'' इत्यनिट्त्वं, तादावनुनासिकलोपं बाधित्वा "जनसनखन'' इत्यात्वं भवति ( असात असाथाः ) इत्यत्र "तनादिभ्यस्तथासोः'' इति सिचो लुकि तथासोर्ङित्त्वमाश्रित्य आत्वं भवति ( सायते सन्यते संसन्यते सासायते) इति यकि यङि "ये विभाषा''इत्यात्वं वा भवति ( सन्यात् )इत्यत्र तु "ये विभाषा''इत्यात्वंये इत्यत्वतो निर्द्देशशन्न भवतीति धातुपारायणे ( सातिः, सतिः सन्तिः ) "सनः क्तिचि लोपश्चास्यान्यतरस्याम्'' इत्याकारः, पक्षे तदभावे वा नलोपश्च ( सातिः ) "ऊतियूति'' इत्यादिना क्तिनि निपात्यते 2
