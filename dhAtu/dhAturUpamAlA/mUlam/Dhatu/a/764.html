 ( त्रस्यति, त्रसति, ) "वा भ्राश'' इति श्यन्विषये पक्षे शप् ( तत्रास, तत्रसतुः, त्रेसतुः, तत्रसिथ, ) "वा जॄभ्रमुत्रसाम्'' इति किति लिट थलि च सेटि एत्वाभ्यासलोपविकल्पः ( त्रसिता त्रसिष्यति त्रस्यतु, त्रसतु, अत्रस्यत्, अत्रसत् त्रस्येत्, त्रसेत् ) आशिषि ( त्रस्यात् अत्रासीत्, अत्रसीत् तित्रसिषति तत्रास्यते तात्रस्ति तात्रसीतित्रासयति अतित्रसत् त्रस्नुः) "त्रसिगृधि'' इत्यादिना क्नुः ( त्रस्तः, त्रस्तवान् ) ईदित्त्वादनिट्त्वम् ( तरङ्गापत्रस्तः ) "अपेतापोढ'' इति पञ्चमीसमासः ( त्रासः ) घञ् रत्नदोषो भयं च त्रस धारणे चुरादौ 10
