पाको विक्लित्तिः
तथा "शृतं पाक' इत्यत्र वृत्त्यादावुक्तम् ( शायति शशौ शाता श्रायति शश्रौ, श्रातेत्यादि ) गायतिवत् ( श्रपयतीति ) घटादित्वात्पाके मित्वम अन्यत्र ( श्रापयति ) स्वदेयतीत्यर्थः ( शृतं क्षीरम् ) "शृतं पाके'' इति क्ते शृभावो निपात्यते तत्र पाको विक्लित्तिः, तत्फलको व्यापारश्च तत्र तत्फलके व्यापारे णिचमन्तरेण वृत्तेरभावात् श्रपेरप्ययं शृभाव इति ( शृतं क्षीरं देवदत्तेनेति भवति ) तथा च तत्र वार्त्तिकं, "श्राश्रप्योः शृभावो निपात्यते'इति यत्तु श्रयेर्णिचि श्रपिशब्दः, न तस्यायमादेशः, पाठ इति वचनात्, अत्र पाचनार्थः पाकस्तु गुणीभूतः अयं शृभावो व्यवस्थितविभाषाविज्ञानात् क्षीरहविषोर्नित्यं भवति अन्यत्र तु न भवति ( श्राणा यवागूः श्रपिता यवागूरिति ) "संयोगादेः'' इति निष्ठानत्वम् श्राणास्मै नियमेन नियमेन दीय्रते ( श्राणिकः ) श्राणामांसौदनाट्टिठन्इति नियमेन दीयमानोपाधिकार्थात्तदस्मै दीयते चतुर्थ्यर्थे टिठन् टित्त्वात् स्त्रियां ( श्राणिकी श्राता ) इत्यादादिकस्य, ( श्रापयतीति ) ( चौरादिकस्य ) 903
