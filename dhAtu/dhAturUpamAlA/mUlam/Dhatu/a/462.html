वर्त्तने
 ( वर्त्तते ववृते वर्तिता वर्त्तिष्यते वर्त्स्यति वर्त्तताम् अवर्त्तत वर्त्तेत वर्त्तिषीष्ट अवृतत् अवर्त्तिष्ट अवर्त्तिष्यत अवर्त्स्यत् विवर्त्तिषते विवृत्सति ) वृझ्यः स्यसनोःवा परस्मैपदम् वृज्ञ्य इति बहुवचननिर्देश आद्यर्थावगमार्थ इति वृतुवृधुसृधुस्यदिकृपीनामपि स्यसनोर्वा परस्मैपदं द्रष्टव्यम् न वृज्ञ्यतुर्भ्यःइति सकादेरार्धधातुकस्य परस्मैपदरस्य इढभावः, ज्ञलादेः सनः "हलन्ताच्च'' इति कित्त्वादगुणत्म् "न वृज्ञ्यश्चतुभ्यः'' इति अत्र भाप्यवार्तिकयोः [ सिद्धु तुवृतादीनामात्मनेपदेन समानपदस्थस्येङ्वचनात् ] इत्युक्तत्वात् कृत्परेपि सादौ परस्मैपदे लुकि चेटः प्रतिषेधो भवति (विवृत्सिता विवृत्सितुम् विवृत्सितव्यम् ) विवृत्सेति समान्येन वलादाविति सिद्धे आरभ्यमाणछिमात्मनेपदेन समानपदस्येड्विधाने ऽर्थादन्यनिवृत्त्यर्थं, विवृत्सिषीयत इति आत्मनेपदेन समानपदस्थत्वेप्यन्तरङ्गः प्रकृत्याश्रायः प्रतिषेधो भवति न तु बहिरङ्ग आत्मनेपदाश्रयो विधिः (वरीवृत्तृयते) "रीगृत्यतः इति रीक (अवर्वः) इत्यादि नेयम् (वर्त्तयति अववर्त्तत् अवीवृतत्) वर्त्स्यन् बर्त्तिष्यमाणो वा (वर्त्तिष्णुः) "अलंकृञ्''इत्यादिना इष्णुच् (हस्तवर्त्त वर्त्तयति) हस्ते वर्त्तिग्रहोःइति हस्तवाचिनि करणे उपपदे ण्यन्तादस्मात् ग्रहेश्च णमुल् हस्तहत्यर्थग्रहणमिति (पाणिवतृर्तम्) इत्याद्यपि भवति कषादित्वा थाविध्युनप्रयोगः वृत्तः अयं यदान्तभांवितण्यर्थस्तदा सकर्मकः, यथा "तेन निर्वृत्तम्'' इति निर्वर्त्तितमिति ह्मत्रार्थः, यद्यैवं "णेरध्ययने वृत्तम्'' इति निष्ठायामिडभावनिपातनं किमर्थं यस्मादन्तर्भावितण्यर्थादस्मात्वृत्तमध्ययनमिति भविष्यति एवं तर्हि ण्यन्तादप्यध्ययने वृत्तमित्येव यथा स्यान्न तु वर्त्त्तिमित्येवमर्थं निपातनम् अत्र वृत्तिकारो ऽमुं पक्षमुक्त्वा वर्त्तितो गुणो देवदत्तेनेति चेष्टमित्याह अत्र हरदत्तः तेषां नार्थः सूत्रेणेति वृत्तं तदिति ( वर्त्म ) "उणादयो बहुलम्'' "भूतेपि दृश्यन्ते'' इति मनिन् (वर्त्मनिः) रथामार्गः "अदेर्मुद्'' "वृत्तेश्च''इति मुदागमो निप्रत्ययश्च अदिवृतिभ्यां मुट् चेति वक्तवृये पृथग्योगकरणातृपक्षे मुण्नभवति ( वर्त्तः ) इत्यपि भवति ( वर्त्तनो ) कृदिकारत्वादि पूर्ववत् वर्धिष्णुः ) "अलंकृञ्'' इत्यादिनेष्णुच् ( वर्द्धनः ) नन्द्यादिपाठात् ल्युः (वृद्धः) अतिशयेन वृद्धौ (वर्षिष्ठः वर्षीयान् ) प्रियस्थिरःइति इष्ठन्नीयसुनोर्यथासंख्याद् वृद्धस्य वर्षादेशः ( ज्येष्ठः ज्यायान् ) "ज्य च'' इति ज्योदशः अकारोच्चारणसामर्थ्याट्टिलोपो न भवतीति प्रागेवोक्तम् ( वृद्धमाचष्टे वर्षयति ज्यापयति ) णाविष्ठवदिति वर्षत्यादेशौ वृद्धिः "तितुत्र'' इति नेट् वृद्धत्यर्थं द्रव्यमपि, तत् प्रयच्छतीति ( वार्द्धुषिकः ) तत्प्रयच्छति गर्ह्मम्इति द्वितीयान्ताद् गर्ह्मोपाधिकात्प्रयच्छतीत्यर्थे ठगिति ठक् वृद्धेर्वृधुष्भावः वृधुषिः प्रकृतृयन्तरं वेति वृत्तिः वृद्धेर्गर्ह्मत्वमशास्त्रीयत्वेन अयं भाषार्थश्चुरादौ 747
