"प्रातिपदिकाद्धात्वर्थे बहुलमिष्ठवच्च'' प्रातिपदिकाद्धात्वर्थे णिच्, तस्मिंश्चेष्ठनीव कार्य्यं भवति इदं च "तत्कारोति तदाचष्टे'' तेनातिक्रामतीनां वक्ष्यमाणत्वात् तद्विषयम् ( पटुमाचष्टे पटयति लघुमाचष्टे लघयति अपीपटत्., अलीलघत् ) अत्र परत्वाद्वृद्धौ"णाविष्ठवत्'' इति टिलोप इत्यग्लोऽप्ययं नेति "सन्वत्'' इतीत्वम् "दीर्घो लघोः'' इति दीर्घश्च भवति अत्र केचित् "नाग्लोपि'' इत्यत्रागितिप्रत्याहारग्रहणसामर्थ्याद्वृद्धेः प्रागेव टिलाइत्यग्लोप्यङ्गमितीत्वदीर्घत्वे नेच्छन्ति ( अपपटत् अललवत् ) इति तदसत्"हलिकल्योरदन्तत्वनिपातनं सन्वद्भावप्रतिषेधार्थम्'' इति वार्त्तिकविरीरोधात् यदि हि वृद्धेः प्राक् टिलोपः स्याद् अग्लोप्येवाङ्गमिति किमदन्तत्वनिपातनक्लेशेन प्रत्याहारग्रहणस्य तु प्रयोजनं ( स्वामिनमाख्यत् असस्वामत्, गोमिनमाख्यत् अजुगोमत्, वीरुधमाख्यत् अविवीरत्, प्रावृषमाख्यत् अपप्रावत् यादृशमाख्यत्, अययादद् ) इति स्यात् एषामेषूदाहरणेष्वनभिधानान् णिज् नेति मतं तेषामपि स्रग्विणमाख्यद् इत्यादौ "विन्मतोर्लुक्'' इतीष्ठवद्भावेन विनो लुकि ह्रस्वाद्यभावः प्रयोजनमेवासस्रजदिति अत्र बहुलग्रहणात् स्रजयतीत्यत्र वृद्धिर्न्न भवति, तथा कुमारीमाचष्टे कुमारयतीत्यत्राप्रातिपदिकत्वेऽपि णिजिति केचित् अन्ये तु "प्रकृत्यकाच्'' इति प्रकृतिभावादेव स्रजयतीत्यत्र वृद्धिर्न्न भविष्यति तथा यथाकथञ्चित्प्रातिपदिकग्रहणे लिङ्गविशिष्टपरिभाषया "ङ्याप्प्रातिपदिकात्'' इति सूत्रे भाष्ये स्थापनात् कुमारयतीत्यादौ णिजपि सिद्ध इति बहुलग्रहणं विकल्पमात्रफलमाहुः "तत्करोति तदाचष्टे'' तदिति द्वितीयान्तोपलक्षणम् तत्करोत्याचष्ट इत्यत्र धात्वर्थमात्रं णिजर्थो न कर्तापि, तथा चैवोदाहृतम् "तेनातिक्रामति'' अत्रापि तेनेति तृतीयान्तोपलक्षणम् अश्वेनातिक्रामति ( अश्वयति ) हस्तिनातिक्रामति ( हस्तयति ) "धातुरूपञ्च'' णिच्प्रकृतिर्धातुरूपं प्रतिपद्यते अनेन [ "आख्यानात्कृतस्तदाचष्टेकृल्लुक् प्रकृतिप्रत्यापत्तिः प्रकृतिवच्च कारकम्'' इत्येतत्सूच्यते कंसवधमाचष्टे कंसं घातयतीत्युदाहरणम् "कर्तृकरणाद्धात्वर्थे'' अत्र कर्तृग्रहणं करणस्य विशेषणम् कर्तुः करणाद्धात्वर्थे णिजिति सूत्रार्थः करिभिरवबध्नाति ( अवकरयति ) असिना हन्ति ( असयति ) परशुना छिनत्ति ( परशयति ) दात्रेण लुनाति ( दात्रयति )
