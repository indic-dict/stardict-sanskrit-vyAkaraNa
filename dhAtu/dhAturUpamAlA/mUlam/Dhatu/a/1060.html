सङ्कोचने
 ( तनक्ति तङ्क्तः तञ्जन्ति तनक्षि तनच्मि ततञ्च ततञ्चतूः ततञ्चिथ ततङ्कथ ततञ्चिव ) ऊदित्त्वात् स्वरत्यादिना इड्विकल्पः क्रादिनियमः प्रतिषेधविषय इत्युक्तम् (तङ्ता तञ्चिता तङ्क्ष्यति तञ्चिष्यति तनक्तु तङ्क्तात् तङ्ग्धि तनचानि अतनक् अतङ्क्ताम् अतञ्चन् अतनक् अतनचम् अतञ्च्व तञ्च्यात् ) आशिषि (तच्यात् अतञ्चीत् अताङ्क्षीत् तितञ्चिषति तितङ्क्षति तातच्यते तताङ्क्ति तञ्चयति अततञ्चत् आतञ्चनम्, ) ल्युट् ( आतङ्कः ) घञ् ( तञ्चित्वा, ) इट्पक्षे "न क्त्वा सेट्'' इत्यकित्त्वान्नलोपाभावः ( तक्तः) "यस्य विभाषा'' इत्यनिट्त्वम् अयं प्रथमान्तोऽपि तृतीयान्तमध्ये ऊदित्प्रसङ्गादेकवर्गीयत्वाच्च पठितः आत्रेयादयस्तु तृतीयान्तमेवपठन्ति ( तक्रम् ) "स्फायितञ्चि'' इत्यादिना रक् बाहुलकाच्चकारस्यककारः 22
