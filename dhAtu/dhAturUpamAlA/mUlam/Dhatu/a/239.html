त्रपूष्
 
लज्जायाम्
 त्रपते त्रेपे "तॄफलभजत्रपश्च'' इति किति लिट एत्वाभ्यासलोपौ त्रपेस्थलि नास्ति ( त्रपिता त्रप्ता त्रपिष्यते त्रप्स्यते ) उदित्त्वादिड्विकल्पः ( त्रपताम् अत्रपत त्रपेत आशषि त्रपिषीष्ट त्रप्सीष्ट अत्रपि अत्रप्त ) झलि सिचो लोपः ( तित्रपिषते तित्रप्सते तात्रप्यते तात्रपीति तात्रप्ति त्रपयति अतित्रपत्) घटादित्वान्मित्त्वम् ( त्रपित्वा त्रप्त्वा त्रप्तः त्रप्तवान् ) "यस्य विभाषा'' इति निष्ठायामिण्निषेधः त्राप्यम् "आसुयुवपिरपिलपित्रपिचमश्च'' इति भावे ण्यत् "पोरदुपधात्'' इति यतोऽपवादः ( त्रपा ) "षिद्भिदादिभ्योङ्'' इति स्त्रियामङ् अपत्रविष्णुः "अलङ्कृञ्निराकृञ्प्रजनोत्पतोन्मदरुच्यपत्रपवृतुवृधुसहचर इष्णुच्'' इति तच्छीलादाविष्णुच् त्रपु "शस्वृस्निहित्रप्यसिहनिक्लिदिबन्धिमनिभ्यश्च'' इत्युप्रत्ययः त्रपुणो विकारस्त्रापुषम् त्रपुजतुनोः षुक्इति षुगागमोऽण्प्रत्ययश्च 371
