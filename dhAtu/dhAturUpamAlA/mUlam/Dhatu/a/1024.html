गतौ
 उदात्तेत् (विच्छायति) "छे च'' इति तुक् "गुपूधूपविच्छि'' इत्यायः सार्वधातुके नित्यम् (विच्छायांचकार विविच्छ विविच्छतुः विविच्छिथ विविच्छिव) "आयादय आर्द्धधातुके वा'' इत्यार्द्धग्धातुके भाविन्यायोत्पत्तविकल्पः निवृत्तिविकल्पे प्रत्ययलक्षणेनायान्तत्वादाम्प्रत्ययः स्यात्, (विच्छाञ्चकार) इति ( विच्छिता विच्छायिता विच्छिष्यति विच्छायिष्यति विच्छायतु अविच्छायत् विच्छायेत् ) आशिषि ( विच्छाय्यात् विच्छ्यात् अविच्छायीत् अविच्छायिष्टाम् अविच्छीत् अविच्छिष्टाम् विविच्छायिषति विविच्छिषति वेविच्छ्यते ) यङ्लुको विवक्षायान्त्वार्धधातुकविषयत्वस्यैवापहारादायोत्पत्तिविकल्पो नेति प्रागेवोपपारितत्वाद्यङ्लुङ्नास्ति ( विच्छयति अविविच्छत् अविविच्छायत् विश्नः "यजयाच'' इति नङ् "च्छ्वोः शूड्'' इति शः, "नेड्वशि कृति'' इतीण्निषेधः, ( विच्छित्वा विच्छायित्वा विच्छितः विच्छियितः ) अयं भासार्थे चुरादौ 139
