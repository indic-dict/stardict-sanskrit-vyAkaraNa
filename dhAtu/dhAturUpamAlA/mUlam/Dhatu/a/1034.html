लिप
 
उपदेहे
 ( लिम्पति लिलेप लिलेपिथ लिलिपिव लेप्ता लेप्सति लिम्पतु अलिम्पत् लिम्पेत् लिप्यात् अलिपत् ) "लिपि सिचि ह्वश्च'' इत्यङ् परस्मैपदे, अन्यत्र "आत्मनेपदेष्वन्यतरस्याम्'' इति पक्षे सिच् ( अलिपत अलिपेताम् अलिपन्त अलिप्त अलिप्साताम् अलिप्सन्त ) इत्युभयं भवति, ( लिम्पते लिलिप्सति लिलिप्सते लेलिप्यते लेलेप्ति लेपयति अलीलिपत् लिम्पः ) "अनुपसर्गाल्लिम्प'' इति शः ( निलिम्पा देवाः ) "नौ लिम्पेः'' इति शः ( लिप्तवासितम् ) इत्यत्र राजदन्तादिपाठात्पूर्वकालस्यापि परनिपातः ( लिपिः लिबिः) "इगुपधात्किच्च'' इति इत्प्रत्ययः "दिवाविभा'' इत्यादौ लिपिलिबीति निर्द्देशात्पकारस्य पक्षे बकारः 149
