 ( ईट्टे ) ष्टुत्वचर्त्वे ( ईडाते ईडिषे, ईडिध्वे ईडे, ईड्वहे, ) ईडजनोर्ध्वे चअत्र ईश इति वर्त्तते से इति च, ईशीडिजनीभ्यः सेध्वेशब्दयोः सार्व्वधातुकयोरितीट् ( ईडांचक्रे ईडिता, ईडितासे ईडिताहे ईडिष्यते, ईट्टाम् ईडाताम् ईडिष्व ईडिध्वम् ईडै ) "सवाभ्यां वामौ'' इति वामयोः कृतयोरप्येकदेशविकृतस्यानन्यत्वात्सेध्वेशब्दावेवेतीट् ( ऐट्ट ऐडाताम् ऐट्टाः ऐड्ध्वं, ) विकृतिग्रहणे प्रकृतिग्रहणाभावान्नेट् (ऐडि ऐड्वहि ईडीत ईडीथाः ईडिध्वम्, इडिय) आशिषि ईडिषीष्ट ईडिषीष्ठाः ईडिषीध्वम्, ईडिषीय ऐडिष्ट ऐडिध्वम् ऐडिषि, ) कर्म्मणि ( ईड्यते ) इत्यादि (ईडिडिषते ईडयति माभवानीडिडत् ईडा ) "गुरोश्च हल'' इत्यकारः "ईडाया वा'' इति निर्द्देशाद्ध्रस्वः ( ईड्यः ) स्तुत्यः, ण्यत् "ईडवन्दवृशंसदुहां ण्यतः'' इत्याद्युदात्तत्त्वम्, (इडेन्यः) औणादिक एन्यप्रत्ययो बाहुलकात् अयं चुरादौ च 9
