स्थानिवत्त्वात् द्विर्वचनम् ( गन्ता गमिष्यति ) "गमेरिट् परस्मैपदेषु'' इति सकारादेरिट् ( गच्छतु अगच्छत् गच्छेत् आशिषि गम्यात् अगमत् ) लृदित्वात् अङ् "गमहन'' इत्यत्रानङिति पर्युदासात् नोपधालोपः ( जिगमिषति ) जिगमिष त्वत्र हेर्लुका लुप्तत्वात् परस्मैपदाभावे भाष्यवार्त्तिकयोः सिद्धं तु गमेरात्मनेपदेन समानपदत्थस्येटः प्रतिषेधात् इति न्यासान्तरेण आत्मनेपदेन समानपदस्तस्य गमेः सकारादेरिण्निषेधात् इडभाव इति एवं जिगमिषितेत्यादौ कृत्यपीट् भवति इदं भाष्यवार्त्तिकं चेति आत्मनेपदेऽपीण्निषेधे सिद्ध एवारभ्यमाणमर्थादन्यत्रेटं बोधयति अत एव भाष्यवर्त्तिकविरोधात् "गमेरिट्'' इत्यत्र परस्मैपदग्रहणं गम्युपलक्षणार्थम् परस्मैपदेषु यो निर्दिष्ट इति पुरुष ( 1 ) ( 1 ) परिशेषकार इति 2 पु पाठः कारदर्शनमुपेक्ष्यं, वृत्तावयं पक्षः प्रतिक्षेपार्थमेव उपक्षिप्तः यदाह तत्र हरदत्तः न पुनरयं पक्षः स्थापितो भाष्यवार्त्तिकविरोधादिति कः पुनरत्र फलभेदः उच्यते सम्पूर्वस्याकर्मकस्य गमेः "समो गमृच्छि'' इत्यात्मनेपदित्वात् परस्मैपदेष्वदर्शनात् संजिगमिषिते इत्यादौ इण् न स्यात्, तथा जिगमिषां चक्र इत्यत्र भावकर्मविषयस्य गमेः परस्मैपदेष्वदर्शनात् इण्न स्यात् अथ मतं परस्मैपदेषु दृष्ट एव गमिः उपसर्गादिवशेनात्रात्मनेपदेऽ( 2 ) ( 2 ) पीड्भवतीति 2 पु पा पीति इद्भविष्यतीति तर्ह्यस्य गमेः सर्वत्रेट्. स्यात्,
