ष्टुभु
 
स्तम्भे
 ( स्तोभते तुष्टुभे स्तोभितेत्यादि तुस्तुभिषते तुस्तोभिषते स्तुभित्वा स्तोभित्वा ) "रलो व्युपधात्'' इति वा कित्त्वम् उदित्त्वात्पक्षे क्त्वायामिडभावे "निष्ठा'' इति निष्ठायां "यस्य विभाष'' इति (स्तुब्धम्), सनि "स्तौतिण्योरेव'' इत्यषत्वं, ( निष्टोभिता निष्टोब्धा ) "उपसर्गात्सुनोति'' इत्यादिना "प्राक् सितादड्व्यायेपि'' इति नियम एवावतिष्ठते ( अनुष्टुप् ) पूर्ववत् षत्वं, ( त्रिष्टुप् ) "सुषामादिषु च'' इति षत्वम् अनुष्टुबेवानुष्टुभम् एवं (त्रैष्टुभं), [छन्दसः प्रत्ययविधाने नपुंसकेस्वार्थउपसंख्यानम् } इति स्वार्थे उत्सादिभ्यः प्राग्दीव्यतीयोण्, स्वार्थेपि प्राग्दीव्यतीयः तिप्यादय उदात्ता अनुदात्तेतः तिपिवर्जम् 391
