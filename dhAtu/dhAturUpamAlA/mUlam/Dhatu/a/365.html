भिक्ष
 
भिक्षायाम्,अलाभे,लाभे च
 ( भिक्षते ) इत्स्यादि पूर्ववत् अयं द्विकर्मकः, तत्राप्रधाने दुहादीनामिति वचनात् लकृत्यखलर्था अप्रधाने भवन्ति ( पौवो गां भिक्ष्यते भिक्षितव्यः भिक्षितः सुभिक्ष इत्यादि ) षष्ठी तु द्वितयावदुभयत्र भवति (भिक्षिता गौः पौवस्येति) अप्रधाने तूभयथा गोणिकापुत्र इति भाष्य उक्तत्वात् पौरवमिति द्वितीयापि भवति ( भिक्षितवयो राजा देवदत्तेन ) इत्यत्रानभिहिते कर्मणि कर्त्तरि च "कर्तृकर्म्मणोः'' इति षष्ठ्या उभयप्राप्तौ कृति प्रतिषेध इत्युक्तत्वात् न भ्वति सर्वमेतन्नाथतावुपपादितं तत एवावगन्तव्यम् ( भिक्षुः ) "सनाशंसभिक्षउः'' इति ताच्छीलिक उप्रत्ययः ( भक्षाकः ) "जल्पभिक्षः'' इति षाकन् षित्त्वात् स्त्रियां ( भिक्षाकी ) 598
