गतौ
 ( ऋणाति ऋणीतः ऋणासि ऋणामि अराञ्चकार ) इत्यादि, "इजादेश्च गुरुमत'' इत्याम् ( अरिता अरीता अरिष्यति अरीष्यति ऋणातु ऋणीताम् ऋणीहि ऋणानि आर्णात् आर्णीताम् आर्णाः आर्णाम् आर्णीव ऋणीयात् ऋणीयाताम् ) आशिषि ( ईर्यात् ईर्यास्ताम् इत्त्वरपरत्वादि ( आरीत् आरिष्टाम् ) "सिचि च परस्मैपदेषु'' इतीटो न दीर्घः ( अरिरिषति आरिरीषति ईर्षिषति ) "इट् सनि वा'' इतीड्विकल्पः, इटि गुणः, पूर्ववद्वादीर्घश्च, अनिटि सनः कित्त्वादित्त्वादि, "नन्द्राः'' इति रेफवर्जितस्य द्विर्वचनम् ( आरयति माभवानरिरत् ईर्त्वा समीर्णः समीर्णवान् ) "उदोष्ठ्यपूर्वस्य'' इत्यत्र प्रत्यासत्त्याऽङ्गावयव एवोष्ठ्यो गृह्यतइत्युत्त्वमत्र न भवति ऋच्छतीति शपि इयर्त्तीति श्लौ 26
