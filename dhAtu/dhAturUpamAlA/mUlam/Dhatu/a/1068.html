क्षिणोतिवल्लिट्प्रक्रिया ( अर्णिता अर्णिष्यति अर्णिष्यते अर्णोतु अर्णुडि ) गुणे कृते संयोगपूर्वत्वादुकारस्य "उतश्च प्रत्ययात्'' इति हेर्लुग्न भवति ( आर्णोत् आर्णुताम् ऋण्यात् अर्णुयात् ) सर्वत्र विकरणापेक्षो गुणः आशिषि ( ऋण्यात् अर्णिषीष्ट आर्णीत् अर्णिष्टाम् आर्णिष्ट आर्त आर्णिष्ट आर्णिष्ठाः आर्थाः,) तनादित्वात्सिज्लुक्यनुनासिकलोपः ( अर्णिनिषति ) गुणे कृते "न न्द्राः'' इति रेफवर्जितस्य द्विर्वचनम् अत्रात्रेयो नान्तोपदेशत्वेऽपि द्विर्वचने कर्तव्ये णत्वस्य नासिद्धत्वम्, [पूर्वत्रासिद्धीयमद्विर्वचने] इति न्यायात् तेन प्रकृत्यभ्यासयोर्द्वयोरपि णकारावित्याह पारायणे तु
