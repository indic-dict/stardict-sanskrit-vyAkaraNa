चायृ
 
पूजा,निशामनयोः
 ( चायति चचाय चायिता चिचायिषति चेकीयते ) "चायः की'' इति यङ्यङ्लुकोः ककीभावः दीर्घोच्चारणं यङ्लुगर्थमिति स्थितम् ( चेकयीति चेकेति चेकीतः चाययति अचचायत् चायते इत्यादि अचायिष्ट चायित्वा अपचितः ) अपचितश्चइति निष्ठायामनिट्त्वं चिभावश्च पक्षे निपात्यते, अन्यदा ( अपचायितम् ) [ क्तिनि नित्यं चिभावो वक्तव्यः ] ( अपचितिः चनः ) अन्नम् "चायतेर्ह्रस्वश्च'' इत्यसुनि नुगामे ह्रस्वश्च, वलिलोपः 865
