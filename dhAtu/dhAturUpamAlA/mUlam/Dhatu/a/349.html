पूरणे
 ( पूर्वति ) "उपधायां च'' इति दीर्घः क्किपि राल्लोपे ( पूः ) पिपर्त्तौ च व्युत्पादयिष्यते पुरि भवादि ( पौरेयम् ) "नद्यादिभ्योढक्'' इति शैषिको ढक् वृत्तौ पूर्वनगरीति पीठित्वा केचित्पूर्वनगरीति पठन्ति विच्छिद्य च प्रत्ययं कु_वन्ति, उभयमपि प्रमाणमित्युक्तम् (द्विपुरी) "ऋक्पूरब्धू'' इत्यकारः समासान्तः, स चाकृत एव समास इति जम्भतावुक्तं, तेनाकारान्तोत्तरपदद्विगत्वे ङीप् पूर्ववदत्रापि स्वतिकिंनञः परत्वे समासान्तो न भवन्ति पूर्वः अच् "सर्वादीनि सर्वनामानि'असंज्ञोपसर्जनानि सर्वादीनि सर्वनामसंज्ञानीति सर्वनामत्वे "सर्वनाम्रः स्मै'' इति ङेः स्मैभावः ( पूर्वस्मै ) "पूर्वादिभ्योनवभ्यो वा'' इति ( पूर्वस्मात् पूर्वात् पूर्वस्मिन् पूर्वे ) इति पूर्वस्य स्मात्स्मिनौ, "आमि सर्वनान्नः सुट्'' इति (पूर्व्वेषाम्) इत्यामः सुडागमः, "बहुवचने ज्ञल्येत्'' इत्येत्वे षत्वम् "सि च'' "पूर्वपरावरदक्षिणोत्तरापराधरिण व्यवस्यायामसंज्ञायाम्'' इति जसः कार्यं प्रति सर्वनामसंज्ञाविकल्पनात् सर्वनामत्वे "जसः शी'' इति शीभावे ( पूर्वे पूर्वाः ) जसः काय प्रतीत्युक्तत्वात् "अव्ययसर्वनान्नमकच् प्राक् टेः'' इत्यकचि संज्ञायां विकल्पनाभावात् ( पूर्वके पूर्वका ) इति भवति व्यवस्था नाम स्वाभिधेयापेक्षो ऽवधिनियमः मासेन पूर्वाय ( मासपूर्वाय ) इत्यत्र"पूर्वसदृ''इति तृतीयासमासत्वे "तृतीयासमास'' इति सर्वनामत्वाभावःअत्र समासइति प्रकृते पुनः समासवचनाद्वक्येपि नेयं संज्ञा स्त्रियां तु "सर्वनाम्नः स्याड् ह्रस्वश्च'' इति ङिद्वचनेषु स्याडागमे ह्रस्वत्वे ( पूर्वस्यै, पूर्वस्याः, पूर्वस्यामिति ) शीभावाद्यनदान्तत्वान्न भवति पूर्वस्मिन्न्हनि, (पूर्वस्यै, पूर्वस्याः, पूर्वस्यामिति ) शीभावाद्यनदन्ततृवान्न भवति पूर्वस्मिन्नहनि, ( पूर्वेद्युः )सद्यःपरुत्परार्यैषमःपरेद्यव्यद्यपूर्वेद्युरन्येद्युरन्यतरेद्युरितरेद्युरुभयेद्युत्तरेद्युरुः'' इति द्युप्रत्यये यस्येति लोपो निपात्यते "तद्धितश्चा सर्वविभक्तिः'' इति अव्ययश्चायम् (पुरो वसति पुरत आगतः, पुरो रमणीयम् ) पूर्वाधरावराणामसिपुरधवश्चैषाम्इति सप्तमीपञ्चमीप्रथमान्येभ्यः पूर्वादिभ्यो ऽसिप्रत्ययः स्वार्थे तत्संनियोगेनैषां यथासंख्यं पुरादय आदेशा इत्यपि प्रत्यये पूर्वस्य पुरादेशः (पुरस्ताद्वसति पुरस्तादागतः पुरस्ताद्रमणीयम् ) अस्तातिच अस्तातौ परतः पूर्वादिभ्यः पूवर्वत् पुराद्यादेशाः (पूर्वतः पूर्वत्र ) किंसर्वनामबहुभ्यो ऽद्वयादिभ्यः'' इत्यधिकृत्य"पञ्चम्यासिल्'' "सप्तम्यास्त्रल्'' इति स्वार्थे तसिलंत्रलौ, पुरो भवं (पौरस्त्यम् ) "दउक्षिणापश्चात्पुरसस्त्यक्'' इति शैषिकस्त्यक्, (पुरस्कृत्य) "पुरोव्ययम्''इति क्रियायोगे गतित्वात् कुगतिसमासे क्तवो ल्यप् "नमस्पुरसोर्गत्योः'' इति कुष्वोः परतो विसर्जनीयस्य सत्वम्, गतिसंज्ञायामव्ययवचनात् पुरः कृत्वेत्यत्रानव्ययत्वेन गतित्वाभावात् सत्वसमासयोरभावः पूर्व कृतमनेन (पूवीं) पूर्वादिनिःइति प्रथमान्तात्नपूर्वशब्दादानेनेत्यर्थे इनिः, पूर्वं कृतमनेनेति "सुप्सुपा'' इति समासः (कृतपूर्वी कटम् ) "सपूर्वाच्च'' इतीनिः सपूर्वाद्विद्यमानपूर्वादित्यर्थः (पूर्वपुरुषः पूर्वपूरुषः) "पूर्वापरप्रथमचरमजघन्यसमानमध्यमध्यमवीराश्च'' इति समानाधिकरणविशेषणसमासः, (पूर्वः कायस्य पूर्वकायः) पूर्वापराधरोत्तरमेकदेशिनैकाधिकरणेइति एदेशिना ऽवयविना षष्ठ्यन्तेन पूर्वादीनि सामर्थ्यादवयवचनानि समस्यन्त इति त्पुरुषः एकाधिकरण इत्युक्तत्वादवयविन्येकद्रव्ये सत्ययं समासः तेन पूर्वः छात्राणामित्यत्र समासो न भवति ( पूर्वापरमम् पूर्वापरे, पूर्वापरौ ) "विभाषा वृक्षङ्गतृणधान्यव्यञ्जनपशुशकुन्यश्ववडवपूर्वापराधरोत्तराणाम्'' इति द्वन्द्वस्य विभाषयैकवद्भावः "नपुंसकम्'' इति विहितैकवद्भावो नपुंसकलिङ्गः ( पूर्वपूवं पुष्यति ) पूर्वतरं पुष्तीत्यर्थः [ पूर्वपरयोरर्थातिशयविवक्षायां द्वे भवतः ] इति द्वित्वम् ( पर्वति पपर्व पर्विता पिपर्विषति पापर्व्यते पापर्ति ) लङ्तिप्सिपोः (अपापः) क्किपि "राल्लोपः'', (पः, परौ परः, पर्वतः) "भृमृदृशियजिपर्वी'' इत्यादिनातच्, पर्वते भवो मनुष्यः ( पर्वतीयः ) "पर्वताच्च'' इति शषिकः छः भवादावमनुष्ये "विभाष ऽमनुष्ये'' इत्यणि ( पार्वतम् ) इत्यपि भवति तस्य जन्यं रघोर्घोरं पार्वतीयैर्गणैरभूदिति प्रयोगः "तस्येदम्''
