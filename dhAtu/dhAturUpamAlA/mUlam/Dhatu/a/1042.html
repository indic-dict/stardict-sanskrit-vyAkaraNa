विरेचने
 ( रिणक्ति रिङक्ते रिणक्षि रिड्क्थः रिङ्क्थ रिणच्मि रिञ्च्वः रिरेच रिरिचतुः, रिरेचिथ रिरिचिव रेक्त रेक्ष्यति रिणक्तु रिङ्क्तात् रिङ्क्ताम् रिञ्चन्तु रिङ्ग्धि रिङ्क्तात् रिणचानि अरिणक् रिञ्च्यात् रिञ्च्याताम्) आशिषि ( रिच्यात् ) लुङ् ( अरिचत् अरैक्षीत् रिङ्क्ते रिरिचे रेक्तारेक्ष्यतेरिङ्क्तात् रिङ्द्व रिणचै अरिड्क्त अरिङ्क्थाः अरिञ्चि रिञ्चीत रिक्षीष्ट अरिक्त अरिक्षाताम् रिरिक्षति रिरिक्षते रेरिच्यते रेरिचीति रेरेक्ति रेचयति अरीरिचत् रेकः ) घञि "चजोः कुघिण्ण्यतोः'' इति कुत्वम् ( रिक्थम् ) "पातृतुदिविचिरिचिसिचिभ्यस्थक्'' ( विरिञ्चः विरञ्चिः विरिचिनः, ) एते पृषोदरादयः रिच विरेनसम्पर्चनयोरिति चुरादौ 4
