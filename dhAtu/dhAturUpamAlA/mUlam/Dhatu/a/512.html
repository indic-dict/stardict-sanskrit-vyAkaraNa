कल्पने
 ( चलति चवाल चेलतुः चेलुः ) चलितेत्यादि पूर्ववत् कम्पने घटादिपाठात् चलयति अन्यत्र ( चालयतिचलः चालः ) ज्वलतीति णविकल्पः ( चकाचलः ) इत्यर्थः [ चरिचलिपतिवदीनां द्वित्वमच्याक्वाभ्यासस्य ] इति द्विर्वचनम् अभ्यासस्यागामश्च ( चाचलिः ) सहिवहिचलिपतिभ्यो यङ्तेभ्यः किकिनाविति तयोरन्यतरः "अतो लोपः'' "यस्य हलः'' इति अल्लोपयलोपौ ( चलनः ) "चलनशब्दार्थादकर्मकाद्युच्'' इति युच् चल विसलन इति तुदादौ, चल गताविति चुरादौ 817
