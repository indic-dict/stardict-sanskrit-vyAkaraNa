क्रुश
 
आह्वाने, रोदने च
 सदादयस्त्रयोनुदात्ता उदात्तेतः तथा च व्याघ्रभूतिः
तुदिं नुदिं चिद्यति विन्त इत्यपि प्रतीहि दान्तान् दशपञ्च चानिटः दिशिं दृशिं दंशिमथो स्पृशिं मृशिम् रुशिं शिशिं क्रोशमिमष्टमं विशिम् ।लिशिं च शान्तानिटः पुराणगाः पठन्ति पाटेषु दशैव नेतरान्
( क्रोशति चुक्रोश चुक्रूषतुः चुक्रोशिथ चुक्रुशिव चुकुशिम ) क्रादिनियमादिट् ( क्रोष्टा क्रोक्ष्यति क्रेशेत् ) आशिषि (क्रुश्यात् अक्रुक्षत् ) "शल इगुपध्णाद्'' इति क्सः ( चुक्रुद्वक्षति ) "हलन्ताच्च'' इति कित्त्वं, ( चोक्रुक्षते चोक्रोष्ब्टि हौ चोक्रुदिड धित्वषत्वढत्वजश्त्वानि लङि तिप्सिपोः ( अचोक्रोट् क्रोशयति अचुक्रुशत् आक्रोशकः ) देविक्रुशोश्चोपसर्गेइति तच्छीलादौ वुञ् (हे क्रोष्टो ) "सितनिगमि''इत्यादिना तुन् अस्य तुज्वत् क्रोष्टुःइति असम्बुद्धौ सर्वनामस्थाने तृजन्तरूपातिदेशात् क्रोष्टा क्रोष्टारावित्यादि शसि (क्रोष्टुनेत्यादि ) आमि [ नुमचिरतृज्वद्भावेभ्यो नुट् पूर्वविप्रतिषेधेन ] इति नुमि अनजादित्वात् तृज्वद्भावाभावे ( क्रोष्टूनामिति ) भवति ( क्रोष्ट्री ) "स्त्रियां च'' इति तृज्वद्भावः "मतिबुद्धि'' इत्यत्र चकारस्यानुक्तमुच्चयार्थत्वात् ( अक्रुष्टः ) वर्त्तमानेक्तः ( क्रोशः क्रुशः) णाभावे अगुपधत्वात्कः 841
