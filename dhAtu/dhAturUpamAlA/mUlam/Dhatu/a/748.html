भर्त्सन,दीप्त्योः
 "कपिर्बभस्ति ते जनम्' ( बब्धः ) "घसिभसोर्हलि च'' इत्युपधालोपे "झषस्तथोर्धो धः'' "झलां जश् झशि'' इति जश्त्वम् ( बप्सति ) उपधालोपे "खरि च'' इति चर्त्वम् "न पदान्त''इत्यादिना जश्त्वचर्त्वयोरुपधालोपस्य न स्थानिवत्त्वम् अन्यदपि यथादर्शनम् ( भस्म ) "अन्येभ्योऽपि दृश्यते'' इति भाषायामपि मनिन् ( भस्रा ) "हुयामा'' इत्यादिना त्रन् स्वभावात् स्त्रियाम् ( भस्त्रिका, भस्त्रका, अभस्त्रिका, अभस्त्रका, परमभस्त्रिका, परमभस्त्रका) "भस्त्रैषा'' इत्यादिना केबलस्य नञ्पूर्वस्य अन्यपूर्वस्य चास्यककारात्पूर्वस्यात्सथानिकस्याकारस्य वेत्वम् भस्त्रया हरति ( भस्त्रिकः भस्त्रिकी ) "भस्त्रादिभ्यष्ठन्'' इति हरत्यर्थे ष्ठन् षित्त्वाङीष् स्त्रियाम् न बभस्तीति ( नभः ) क्विबित्यात्रेयः 18
