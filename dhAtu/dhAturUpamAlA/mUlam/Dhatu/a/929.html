विमोहनमाकुलीकरणम्
( लुभति लुलोभ लोब्धा ) "तीषसह'' इतीड्विकल्पः ( लोभिष्यति लुभतु अलुभत् लुभेत् लुभ्यात् अलोभीत् लुलुभिषति लुलोभिषति लुभित्वा लोभित्वा ) "रलो व्युपधात्'' इति कित्वविकल्पः ( लोलुभ्यते लोलोब्धि लोभयति अलूलुभत् विलुभिताः ) केशाः "लुभो विमोहन'' इति क्त्वानिष्ठयोर्नित्यमिट् गार्ध्ये तु लुभ्यतीति गतम् 25
