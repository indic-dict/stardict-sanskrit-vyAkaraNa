त्विष
 
दीप्तौ
 ( त्वेषति तित्वेष तित्विषतुः तित्वेषिथ तित्विषिव ) क्रादिनियमादिट् ( त्वेष्टा त्वेक्ष्यति त्वेषतु अत्वेषत् आशिषि त्विष्यात् अत्विक्षत् अत्विक्षाताम् ) "शल इगुपधादनिटः क्सः'' ( त्वेषते तित्विषे त्वेष्टासे त्वेक्ष्यते त्वेषताम् अत्वेषत त्वेषेत त्विक्षीष्ट अत्विक्षत अत्विक्षाताम्, अत्विक्षन्त ) "क्सस्याचि'' इत्यलोपः ( तित्विक्षति तित्विक्षते तेष्वेष्टि तेत्विष्टः ( त्वेषयति ) अतित्विषत् त्वष्टा ) "त्विषेर्दवतायामकारश्चोपधाया'' इति तृन् उपधाया अत्वं, त्वष्टारौ " "ऋतोऽङि'' इति गुणे "अप्तृन्'' इत्यादिना दीर्घोऽसंबुद्धौ सर्वनामस्थाने तृन्नन्तत्वादेव सिद्धे नप्त्रादीनां ग्रहणं व्युप्तत्तिपक्ष नियमार्थं संज्ञाशब्दानामेषामेवेति वृत्तौ त्वष्टुरिदं (त्वाष्ट्रम्) अचामादेरचो वृद्धिरन्त्यलक्षणां वृद्धिं बाधतइति वृत्तौ, ( त्विट् ) संपदादित्वात् क्विप् अवपूर्वोऽयं दाननिरसनयोरिति मैत्रेयादायः 981
