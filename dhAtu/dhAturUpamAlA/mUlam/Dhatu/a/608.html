 ( च्यवते चुच्युवे चुच्युविषे चुच्युविध्वे चुच्युवे ) "अचि श्नुधातु'' इति उवङ् ( च्योता च्योष्यते अच्यवत च्यवेत च्योषीष्ट अच्योष्ट चुच्यूषते ) "इको झल्'' इतिकित्त्वे "अज्झन'' इति दीर्घः ( चोच्यूयते चीच्योति च्यावयति अचुच्यवत् अचिच्यवत् ) "स्रवतिशृणोति'' इत्यादिना अभ्यासस्येत्वम् ( च्यवनम् ) चलनार्थत्वात् युच् ( च्योत्रम् ) बाहुलकात्त्रण् ( ज्यवत ) इत्यादि च्यवतिवत् अत्र नन्दी जु इत्युकारं पठति, तत् जु इति सौत्रो धतुरिति वृत्तिन्यासपदमञ्जर्यादिष्वभिधानादनार्षमिव प्रतीयते ( प्रवते पुप्रुवे प्लवते पप्लुवे ) इत्यादि गवतिवत् अत्र मैत्रैयः क्लुङ् इत्येके ( क्लवते ) इत्यादि (क्लब इति विक्लब इति च) बवयो रभेदात् द्रष्टव्यम् क्रुङ् इति क्वचित्पठ्यते 939
