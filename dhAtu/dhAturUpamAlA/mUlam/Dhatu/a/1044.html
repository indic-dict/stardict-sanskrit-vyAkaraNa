क्षुदिर्
 
सम्पेषणे
 ( क्षुणत्ति क्षुन्तः क्षुन्दिन्त क्षुणत्सि क्षुणद्मि क्षुन्दः क्षुन्ते क्षुन्दाते क्षुन्त्से क्षुन्ध्वे क्षुन्द्वहे चुक्षोद चुक्षुदतुः चुक्षोदिथ चुक्षुदिव चुक्षुदे चुक्षुदिषे चुक्षुदिवहे क्षोत्ता क्षोत्स्यति क्षोत्स्यति क्षोत्स्यते क्षुणत्तु क्षुन्तात् क्षन्ताम् क्षुन्दन्तु क्षन्धि क्षुणदानि क्षुन्ताम् क्षुन्दाताम् क्षुन्दताम् क्षुन्त्स्व क्षुणदै क्षुणदावहै अक्षुणत् अक्षुन्ताम् अक्षुन्दन् अक्षुणत् अक्षुणद् अक्षुणः अक्षुणदम् अक्षुन्द्व अक्षुन्त अक्षुन्दाताम् अक्षुन्दत अक्षुन्थाः अक्षुन्दि अक्षुन्द्वहि क्षुन्द्यात् क्षुन्द्याताम् क्षुन्दीत क्षुन्दीयाताम् ) आशिषि ( क्षुद्यात् क्षुद्यास्ताम् क्षुत्सीष्ट क्षुत्सीयास्ताम् अक्षुदत् अक्षुदताम् अक्षुदन् अक्षौत्सीत् अक्षौत्ताम् अक्षुत्त अक्षुत्साताम् चुक्षुत्सति चुक्षुत्सते चोक्षुद्यते क्षोदयति अचुक्षुदत् क्षुदः ) "स्फायितञ्चि'' इत्यादिना रक् क्षुद्राभिः कृतं ( क्षौद्रम् मधु ) "क्षुद्राभ्रमर'' इति तेन कृतमिति विषयेऽञ् ( क्षोदिष्टः क्षोदीयान् क्षोदिमा ) "स्थूलदूर'' इत्यादिना यणादेर्लोप इष्टेमेयस्सु पूर्वस्य च गुणः, पृथ्वादित्वादिमनिच्, ( क्षोदयति ) णाविष्ठवदिति यणादेर्लोपो गुणश्च 6
