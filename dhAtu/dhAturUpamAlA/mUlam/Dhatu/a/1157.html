लुण्ठ
 
स्तेये
 ( लुण्ठयति अलुकुण्ठत् ) लुण्ठतीति लुठि स्तेये इत्यस्य शपि ननु लुण्ठतेरिदित्पाठेनापि लुण्ठयति लुण्ठतीति रूपद्वयं भविष्यति, किं भ्वादीदित्पाठेन सत्यम् सिध्यतीदं रूपद्वयं, लुण्ठितेति ताच्छीलिकस्तृ ( 1 ) ( 1 ) लुण्ठीति ताच्छीइको णिनिर्न स्यादिति पुस्तकान्तरे पाठः न्न स्यात् "जल्पभिक्षकुट्टलुण्ठ' इति षाकना बाधात् लुठि पाठे तु प्रतिपदोक्तत्त्वादस्यैव तत्र ग्रहणमिति भौवादिकात्तृनि लुण्ठितेति भवतीत्युभयत्र पाठः कर्तव्यः अत एव न्यासे कुट्ट छेदे, लुण्ठ स्तेये चौरादिकावेतावित्युक्तम् ( लुण्ठाकः ) "जल्पभिक्ष'' इत्यादिना षाकन्, षित्त्वान्ङीष् ( लुण्ठाकी ) 29
