षृभु
 
षृम्भु
 
हिंसार्थौ
 अत्र मैत्रय एवं पठित्वा षिभु षिम्भु इत्येकइति ऋकारस्य स्थाने इकारमप्याह तरङ्गिण्यां तु ऋकारवन्तौ दन्त्यादौ पठित्वा षोपदेशपर्युदासे सृग्रहणेनानर्थकस्यापि ग्रहणमित्युक्तम् ( सर्भति ससर्भ सर्भितेत्यादि सिषर्भिषति ) मैत्रेयपाठे"स्तौतिण्योरेव'' इत्यषत्वम् ( सरीष्टभ्यते सरीष्टभीति सरिष्टभीति सर्ष्टभीति सरीष्टब्धि सरिष्टब्धि सर्ष्टब्धि सर्भयति अससर्भत्)असीसृभतुरृद्वा(सिषर्भिषति) तरङ्गिण्यां मते तु षत्वं न भवति(सर्भित्वासृप्त्वा) ऊदित्त्वादिड्विकल्पः ( सृब्धम् ) "यस्य विभाषा'' इत्यनिट्त्वम् ( सृभ्यम् ) ऋदुपधत्वात् क्यप् ( सृम्भति ससृम्भ सृभ्यात् ससृम्भतुः ) नलोपः ङ्कितोः इकारवत्पाठपक्षे (सेभति सिम्भतीत्यादि) 428
