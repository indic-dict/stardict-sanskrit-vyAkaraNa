स्तुतौ
 ( ऋचति ) इत्यादि ऋतिवत् ( अर्च्यम्, ) ण्यति "यजयाचऋचप्रवचर्चश्च'' इति कुत्वनिषेधः अत एव कुत्वनिषेधाज्ज्ञापकादृदुपधलक्षणः क्यब्न भवति ( ऋक ) पूर्ववत्क्विप् ऋचो व्याख्यानम् ( आर्चिकम् ) "द्व्यजृद्ब्राह्मण'' इत्यादिना भवव्याख्यानयोष्ठक् ऋचोऽर्द्धम् ( अर्द्धर्चम् ) "अर्द्धं नपुंसकम्'' इत्येकदेशिनार्द्धशब्दस्य षष्ठीसमासः "ऋक्पूरब्धूः'' इत्यादिनाऽकारः समासान्तः "अर्द्धर्चाः पुंसि च'' इति पुंनपुंसकयोरयम् अनृचो माणवको ज्ञेयो बह्वृचश्चरणाख्यायामिति विशेषे अकारप्रत्ययस्य स्मरणाद् (अन्वृकं साम, बह्वृकं सूक्तम् ) इत्यादौ शेषलक्षणः कबेव भवति ऋक् च साम च (ऋक्सामे) "अचतुर'' इत्यादिना द्वन्द्वे ऽच् समासान्तः "नस्तद्धित'' इति साम्नष्टि लोपः 22
