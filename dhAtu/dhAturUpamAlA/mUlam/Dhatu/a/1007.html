 ( रियति ) लघूपधगुणादन्तरङ्गत्वादियङादेशः ( रिराय रिर्यतुः रिर्युः) "एरनेकाच'' इति यण् ( रिरयिथ रिर्यिव रिरेथ रिर्यिम ) क्रादिनियमादिट् थलि "अचस्तास्वत्'' इतीण्निषेधः भारद्वाजनियमाच्च विकल्पः रिर्यतुरित्यादौ यणो बहिरङ्गलक्षणस्यासिद्धत्वाद्रेफस्य हल्परत्वाभावात् "उपधायां च'' इति न दीर्घः ( रेता रेष्यति रियतु अरियत् रियेत् ) आशिषि ( रीयात् ) "अकृत्सार्वधातुकयोः'' इति दीर्घः ( अरैषीत् अरैष्टाम् अरैषुः रिरीषति ) सनः कित्त्वे "अज्झनगमाम्'' इति दीर्घः ( रेरीयते रेरयीति रेरेति राययति अरीरयत् ) ( पियति ) इत्यादि रियतिवत् 120 121
