कुत्सितगतिः पलायनं स्वापश्चेति
स्वामी तत्र प्रायो निपूर्वोऽयंस्वापवचनः ( द्राति, निद्राति, प्रणिद्राति, ) इत्यादि यातिवत् ( निद्रालुः, तन्द्रालुः )"स्पृहिगृहिपतिदनिद्रातन्द्रा'' इति निपूर्वात्तत्पूर्वाच्चालुच् अस्मादेव निर्देशात्तदो दकारस्य च नकारः ( निद्रः ) "आतश्चोपसर्गे'' इति कः ( निद्रा ) स्त्रियाम् "आतश्चोपसर्गे'' इत्यङ् ( तन्द्रा ) "घञर्थे कविधानम्'' इति कप्रत्यये टाप् ( तन्द्रिः) "अच इः'' इतीप्रत्ययः, बाहुलकात्कित्त्वादाल्लोपः ( तन्द्री ) "कृदिकारादक्तिन'' इति वा ङीष् सर्वत्रात तदो दकारस्य नाकारः, "निद्रातन्द्रा'' इति निपातनस्य सामान्यापेक्षत्वाद् द्रष्टव्यः (द्राणः, द्राणवान् ) "संयोगादेरात'' इति निष्ठानत्वम् ( निद्रित ) इति निद्राशब्दात् तारकादित्वादितचि द्रायतीति स्वप्ने शपि 44
