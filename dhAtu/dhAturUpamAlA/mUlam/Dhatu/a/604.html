एतदादयः स्वरन्त्यन्ता अनुदात्ता आत्मनेपदिनः ।
ष्मिङ्
 
ईषद्धसने
 एतदादयः स्वरन्त्यन्ता अनुदात्ता आत्मनेपदिनः (स्मयते सिष्मिये सिष्मियिढ्वे सिष्मियिध्वेसिष्मिये सिष्मियिवहे) "अचि श्नुधातु'' इत्यादिना इयङ् ( स्मेता स्मेष्यतेस्मयताम् अस्मयत स्मयेत स्मेषीष्ट अस्मेष्ट सिष्मयिष्यते) "स्तौतिण्योरेव'' इत्यषत्वम् "स्मिपूङ्रञ्ज्वशां सनि'' इतीट्, ( सेष्मीयते सेष्मेति ) ( मुण्डो विस्मापयते असिष्मपत् ) "नित्यं स्मयतेः'' इति हेत्वेककारके स्मयत्यर्थे आत्वं, तत्र पुक्, "भीस्म्योर्हेतुभये'' इति हेत्वेककारयोर्नित्यं तङ् अत्र भयग्रहणं चित्तविकारोपलक्षणं, तेन बिभेतेर्हेतुभये स्मयतेर्हेतुस्मय इत्यर्थो भवति यदा तु हेतुत एव स्मयो न भवति तदात्वं न, नापि तङ्नियम इति ( मुण्डो रूपेण विस्माययतीति भवति ) ( स्मेरं ) मुखम् "नमिकम्पि'' इति रः स्मित्वा स्मितम् ) अयमनादरे चुरादिः 929
