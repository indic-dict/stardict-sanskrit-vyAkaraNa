ओविजी
 
भय,चलनयोः
 प्रायेणायमुत्पूर्वः उद्विजते उद्विविजे उद्विविजिषे उद्विजिता उद्विजिष्यते ) "विज इट्'' इति इडादेरञ्णित प्रत्ययस्य ङित्त्वान्न गुणः (उद्विजताम् उदविजत उद्विजेत उद्विजिषीष्ट उदविजिष्ट उद्विविजिषते उद्विजित्वा) क्त्विसनो "रलोव्युपधात्'' इति कित्त्वस्य पाक्षिकत्वेऽप्यस्य "विज इड्'' इति ङित्त्वमिति न गुणः ( उद्वेविज्यते उद्वेविजीति उद्वेवेक्ति उद्वेजयति उदवीविजत् उद्विग्नः उद्विग्नवान् ) ईदित्त्वादनिट्त्वम् चोः कुत्वम् ओदित्त्वान्निष्ठानत्वम् ( वेगितम् ) इति घञन्ताद्वेगशब्दादितचि ( उद्वेजिताम् ) इति ण्यन्तस्य निष्ठायामिटि ( वेवेक्ति, वेविक्ते ) इति जुहोत्यादौ पृथग्भावे 9
