ष्टुञ्
 
स्तुतौ
 अनुदात्त उभयतोभाषः (स्तुते स्तुवाते, स्तुवते स्तुषे स्तुध्वे, स्तुवे स्तुवहे तुष्टुवं तुष्टुवाते, तुष्टुविरे तुष्टुषे तुष्टुध्वे तुष्टुवहे) अभ्यासे खयः शेषः क्रादिषु पाठादिडभावः (स्तोता स्तोष्यते स्तुताम् स्तुवाताम् स्तुष्व स्तुध्वम् स्तवै अस्तुत अस्तुवाताम्, अस्तुथाः, अस्तुवि, स्तुवीत स्तुवीयाताम् स्तुवीथाः स्तुवीय ) आशिषि (स्तोषीष्ट) इत्यादि (अस्तोष्ट, अस्तोषाताम् अस्तोष्ठाः अस्तोढ्वम् अस्तोषि) हलादौ सार्वधातुके शिब्विषये यदा "तुरुस्तुशम्यम'' इतीट् तदा (स्तुवीते स्तुवीषे स्तुवीध्वे स्तुवीवहे, स्तुवीताम् स्तुवीष्व स्तुवीध्वम्, अस्तुवीत अस्तुवीथाः, अस्तुवीध्वम् अस्तुवीवहि ) इति लिङि सलोपे ऽहलादित्वान्नेट् प्रसङ्गाः अन्यत्र सार्वधातुके विकरणेन व्यवधानान्नैवास्तीटः प्राप्तः ( स्तौति स्तुवीति स्तुतः,स्तुवीतः, स्तौषि स्तवीषि स्तौमि स्तवीमि स्तुवः स्तुवीवः ) तिप्सिपोरिट्पक्षे गुणः अन्यदा वृद्धिः अन्यत्रेटि सर्वत्रोवङ्, ( तुष्टाव तुष्टोथ, तुष्टव ) थलि भारद्वाजनियमं बाधित्वा पुरस्तात्प्रतिषेधारम्भसामर्थ्यात् क्रादिप्रतिषेध एव भवति ( स्तोता स्तोष्यति स्तौतु स्तवीतु स्तुतात् स्तुवीतात् स्तुतां, स्तुवीतां, स्तुवन्तु स्तुहि स्तुवीहि ) हेरपित्त्वान् ङित्त्वान्न गुणो नापि वृद्धिः, ( स्तवानि अस्तौत् अस्तवीत् अस्तुताम् अस्तवीतामस्तुवन् अस्तौः अस्तवीः अस्तवम् अस्तुव अस्तुवीव
