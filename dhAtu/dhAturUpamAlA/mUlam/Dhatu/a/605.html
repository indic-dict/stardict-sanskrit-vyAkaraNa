अव्यक्ते_शब्दे
 (गवते जुगुवे जुगुविषेजुगुविवहे गोता गोष्यते आशिषि गोषीष्ट अगोष्ट अगोष्यत जुगूषते )"अज्झन'' इति दीर्घः ( जोगूयते ) "अकृत्सार्वधातुकयोदीर्घः'' ( जोगोति गावयति अजूगवत् ) णौ कृतस्य स्थानिवत्त्वाद् गुशब्दस्य द्विर्वचनं, ( गुत्त्वा गुतः ) "दुग्वोर्दीर्घश्च'' इति अत्र निरनुबन्धस्य तौदादिकस्य गु पुरीषोत्सर्गइत्यस्य ग्रहणमिति नात्र निष्ठानत्वदीर्घत्वे भवतः ( गोत्रम् ) "गुपचि'' इत्यादिना त्रप्रत्ययः (ब्राह्मणिगोत्रा) "कुत्सितानि कुत्सनैः'' इति समासः वृत्तिविषये गोत्रशब्दः कुत्सनवचन इति चेलतावुक्तम् "घरूप'' इत्यादिना पूर्वपदस्य ह्रस्वः (गवाकः) क्रमुकवचनः, पिनाकादिपाठादाकप्रत्ययः ( गवलं ) माहिषं शृङ्गम् बाहुलकादलप्रत्ययः अनन्तरं धृञ् धारण इति पठ्यते वर्धमानोऽमुं न सहते यदाह केचित् धृञ् धारणइति पठन्ति सोनार्षः ( धरति धरते ) इति प्रयोगदर्शनात् तत्साहसमात्रमुपादीयते च "अनुपर्गात् लिम्प'' इत्यत्र न्यासपदमञ्जर्योर्धृञ् धारणइति दैवेऽवधारणार्थः ( धरति धरते ) इत्युभयमिति अस्माभिस्तु मैत्रेयाद्यनुसारेण करणे प्रत्यये हरतेरनन्तरं पठित्वायमुदाहृतः 930
