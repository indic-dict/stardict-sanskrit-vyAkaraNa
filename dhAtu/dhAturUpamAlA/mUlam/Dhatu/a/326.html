फल
 
निष्पत्तौ
 (फलति) इत्यादि ञिफलावत् (फलितम् ) इति निष्ठायामिडेव फलतीति फलं (संफल) पाककण'' इति ङीषं बाधित्वा संभस्त्रनिजशणपिण्डेभ्यः फलात्'' इत्यजादिपाठाट्टाप् मस्त्रफलादायोप्येवम् "ङ्यापोः संज्ञाच्छन्दसोर्बहुलम्'' इति ह्रस्वः अल्पं, (फलिका) ततोन्यत्फलम् फलं च तत्फलिका च (फलाफलिका) [ कृतापकृतादीनामुपसंख्यानम् ] इति कर्मधारयः "अन्येषामपि दृश्यते'' इति दीर्घः पूर्वपदस्य एकमेव फलमवस्थाभेदादल्पत्वमहतृवाभ्यां युज्यते ( त्रिफला ) अजा दिदर्शनाट्टाप् ( फलिनः ) [फलबहांभ्यामिनज्वकृतव्यः ] इति मत्वर्थे इनच् ( फलक ) संज्ञायाम् कन् निष्पाद्यते कृषिरनेनेति फालः "हलश्च'' इति करणे घञ् 524
