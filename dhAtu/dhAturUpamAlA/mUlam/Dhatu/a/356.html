प्रीणनार्थाः
 अत्र के चिदाद्यमिकारादिं पठित्वा पुनः पाठोर्थोदादिति, धातूनामनेकार्थत्वे तु न प्रयोजनमस्ति तस्मात् मैत्रेयोक्त एव हिवीति पाठो ज्यायान् (हिन्वतीत्यादि जिहिन्व हिन्वितेत्यादि पिन्वतीवत् एवं दिन्वतीत्यादि) धिनोतेः धिन्विकृण्व्योर चइति शब्विषये उप्रत्ययः, अकारश्चान्तादेशः तस्यातो लोपः, तस्य स्थानिवत्त्वात् उप्रत्ययमपेक्ष्य लघृपधगुणो न भवति, तस्य पित्सु गुणः ( धिनोति धिनुतः धिन्वन्ति ) अन्तादेशे यणादेशः (धिनोषि धिनुवः धिन्वः) लोपश्चास्यान्यतरस्यां म्वोःअसंयोगपूर्वो य उकारस्याभावः ( धिनोतु धिनु ) उतश्च प्रत्ययादसंयोगपूर्वात्योसमुकारो ऽसंयोगपूर्वः तदन्तातृप्रत्ययात् परस्य हेर्लुगिति लुक् ( धिनवानि ) नित्यत्वादुकारलोपात् पूर्वमाट् ( अधिनोत् अधिनुताम् अधिनोः अधिनवम् अधिन्व अधिनुव धिनुयात् ) अन्यत्र लकारे पूर्ववत् (धिन्वयति इत्यादि जिन्वतीत्यादि देवने गतम् 586
