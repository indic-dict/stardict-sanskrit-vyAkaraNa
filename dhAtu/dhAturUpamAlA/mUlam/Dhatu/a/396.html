गृहू
 
ग्रहणे
 ( गर्हते जगृहे जगृहिषे जघृक्षे जगृहिध्वे जघृढ्वे गर्हिता गर्ढा गर्हिष्यते घर्क्ष्यते गर्हताम् अगर्हत गर्हिषीष्ट घृक्षीष्ट ) "लिङ्सिचावात्मनेपदेषु'' इत्यनिट्पक्षे कित्त्वान् न गुणः ( अगर्हिष्ट अघृक्षत अगर्हिषाताम् अगर्हिषत अघृक्षाताम् अगर्हिष्ठाः अघृक्षथाः अगर्हिढ्वम् अगर्हिध्वम् अघृक्षध्वम् अगर्हिषि अघृक्षि अगर्हिष्वहि अघृक्षावहि ) अनिट्पक्षे "शल इगुपधादनिटः क्सः'' "क्सस्याचि'' इत्यल्लोपः यञादौ "अतो दीर्घो यञि'' इति दीर्घः, ढत्वादि तु पूर्ववत् ( जिगर्हिषते जिघृक्षते ) इडभावे "हलन्ताच्च'' इति सनः कित्त्वम् ( जरीगृह्मते ) "रीगृत्वतः'' इति रीक् ( जर्गृहीति जर्गर्ढि जर्गृढः ) रुग्रिकोरप्युदाहार्यम् ( गर्हित्वा गृढ्वा गृहम् ) इगुपधलक्षणः कः अत्र मैत्रेयः "गेहे कः'' इति गृह्णातेः कविधानं तस्यापि गृहमिति यथा स्यादिति अयं क्कचित्कोषे ग्रिह इदुपधः पठ्यते तदनार्षः, यदाह देवः ण्यन्तस्य ग्रहणे ग्रहेर्ग्रहयते तत्राऽनदन्ताद्गृहेर्भूवादेः शपि गर्हते श्नि प्रत्यये गृह्णाति अगृहीत् चेति अत्र पुरुषकारे गृह इत्यनेनैव पूर्वस्माददन्तत्वमात्रविशेषः गृहीत्वा विशिष्टमिति दर्शयिनां गृहपाठाशङ्का निरवकाशतां नीता ग्लह इति स्वामिकाश्यपसम्मताकारादयः पठन्ति तदपि ग्राह्यम् तथा च "अक्षेषु ग्लहः'' इत्यत्र वृत्तिः ग्लहिः प्रकृत्यन्तरमिति ग्लहत इत्यादि पूर्ववत् (ग्लहोऽक्षस्य ) अक्षेषु ग्लहः'' इत्यप् अक्षादन्यत्र (ग्लाह) इति घञ् एषां तु निपातनं ग्रहेरेतदिति तेषामनक्षे ग्रह इति प्रत्युदाहार्यम् लत्वाभावो विशेषः 642
