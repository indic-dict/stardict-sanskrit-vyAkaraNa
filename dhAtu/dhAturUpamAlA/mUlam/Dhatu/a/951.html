द्रुण
 
हिंसा,गति,कौटिल्येषु
 (द्रणति द्रुणः) खर्जुरः इगुपधात् कः ( द्रुणी ) कच्छपी गौरादित्वान् ङीष् ( द्रोणः ) पचाद्यच् ( द्रोणी ) पूर्ववन् डीष् द्रोणस्य गोत्रमपत्यं ( द्रौणिः द्रौणायनः ) "द्रोणपर्वतजीवन्तादन्यतरस्याम्'' इति फगिञौ द्रोणं पचतीति ( द्रौणी, द्रौणकी ) "तत्पचतीति च द्रोणादण् च'' इत्यण्ठञौ द्विद्रोणेन धान्यं क्रीणाति "प्रकृत्यादिभ्य उपसंख्यानम्'' इति तृतीया अत्रापि द्वितीया प्राप्नोतीति कैयटे द्वयोर्द्रोणयोः समाहारो ( द्विद्रोणम् ) पात्रादित्वात् स्त्रीत्वाभावः 56
