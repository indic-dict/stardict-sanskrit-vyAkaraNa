प्रयत्ने
 ( यतते येते येतिष ) एत्वाभ्यासलोपौ ( यतिता यतिष्यते यतताम् अयतत यतेत् आशिषि यतीषीष्ट अयतिष्ट अयतिष्यत् ) भावे ( यत्यते ) इत्यादि ( यियतिषते यायत्यते यायतीति यायत्तीत्यादि ) लङि तिप्सिपोर्हल्ङयादिलोपे जश्त्वे वा चर्त्त्वम् सिपि तु "दश्च''इति वा रुत्वमपि ( यातयति अयीयतत् यत्यम् [ तकिशसियतिजनीनामुपसंख्यानम् ] इति भावे यत् ( आयत्तः आयत्तवान्, ) ईदित्वात् "श्वीदितो निष्ठायाम्'' इतीडभावः यत्नः "यजयाचयतविच्छप्रच्छरक्षो नङ्' इति भावादो नङ् ( यतिः ) "इन् सर्वधातुभ्यः'' इतीन् यत् निकारोपस्कारयोरिति चुरादौ 30
