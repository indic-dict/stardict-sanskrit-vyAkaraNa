पुष
 
पुष्टौ
 (पुष्यति पुपोष, पुपोषिथ, ) क्र्यादिनियमान्नित्यमिट् भारद्वाजनियमस्तुउपदेशेऽजन्तानामत्वतां चेदिह न भवति ( पोष्टा पोक्ष्यति पुष्यतु अपुष्यत्, पुष्येत्, पुष्यात्, ) लिङौ (अपुषत्) "पुषादि'' इत्यादिनाङ्परस्मैपदे तङि तु (व्यत्यपुक्षत) इति क्सो भवति कर्मणि तु ( अपोषि, अपुक्षाताम्, अपुक्षत ) इति चिण्क्सौ भवतः (पुपुक्षति पोपुक्ष्यते पोपोष्टि पोषयति, अपूपुषत्) पुष्यन्त्यस्मिन्नारब्धानि कार्याणीति (पुष्यः,) "पुष्यसिद्धौ नक्षत्रे'' इति क्यप् निपात्यते पुष्येण युक्तः ( पौषाः, ) "नक्षत्रेण युक्तः कालः'' इत्यण् (पौषमहः,) "तिष्युपुष्ययोर्नक्षत्राणि'' इति यलोपः स्वपोषां पुष्यतीत्यादि पोषतिवन्नेयम् "स्वे पुषः'' इति णमुल् ( पुषति, पोषति, ) इति द्वयं शपि पुष्णातीति क्र्यादौ पोषयतीति चुरादौ 74
