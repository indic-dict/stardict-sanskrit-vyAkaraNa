ञिष्विदा
 
गात्रप्रस्रवणे
 ( स्वेदते सिष्वदे स्वेदिता अस्विदत् अस्वेदिष्ट सिस्विदिषते ) "स्तौतिण्योरेव'' इत्यषत्वम् ( सेष्विद्यते सेष्वेयत्ति असिस्विदत् ) ण्यन्तात्सनि "सः स्विदिस्वदिसहीनां च'' इति अभ्यासात्परस्य षत्त्वम् सिस्वेदयिषतीति निष्ठायां मिदिवत् ( स्विन्नः ) इत्यादि अत्र स्वामी ञिक्ष्विदेति क्ष्वकारं पठित्वा ञिष्विदेति नान्दीत्याह तत्र ( क्ष्वेदते चिक्ष्विदे अचिक्ष्विदत् अक्ष्वेदिष्यतेत्यादि ) क्ष्वेदतीत्यव्यक्ते शब्दे भविष्यति 1 स्विद्यतीति स्नेहनसेकयोर्दिवादौ स्विद्यतीति गात्रप्रक्षरणे 731
