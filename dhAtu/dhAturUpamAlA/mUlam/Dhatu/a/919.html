छेदने
 (वृश्चति) "ग्रहिज्या''दिना सम्प्रसारणे पूर्वरूपत्वे "स्तोःश्चुना श्चुः'' इति सकारस्य शकारः ( वव्रश्च वव्रश्चतुः वव्रश्चिथ वव्रष्ठ वव्रश्चिव ) संयोगान्तत्वेन लिटः कित्त्वाभावात् "ग्रहिज्या''दिना सम्प्रसारणन्न भवति अभ्यासस्य "लिट्यभ्यास'' इति सम्प्रसारणे "स्तोः श्चुना श्चुः'' इति सकारस्य शकारः "न सम्प्रसारणे सम्प्रसारणम्'' इति पूर्वस्य निषेधाद्रेफस्यैव सम्प्रसारणप्रसङ्गः, उरदत्त्वस्याङ्गत्वादङ्गक्षिप्तप्रत्ययेन परनिमित्तत्वात् "अचःपरस्मिन्'' इति स्थानिवत्त्वेन सम्प्रसारणपरत्वातूदित्त्वात्सर्वत्रेड्विकल्पः क्रादिनियमः प्रतिषिद्धविषय इति थल्यपि विकल्प एव भवति तत्रेडभावे झल्परत्वात् "स्कोः'' इति सलोपे व्रश्चादिना षत्वे ष्टुत्वम् ( व्रष्टा, व्रश्चिता व्रक्ष्यति व्रश्चिष्यति ) अनिटि सलोपषत्वयोः "षढोः कः सि'' इति कत्वं, ( वृश्चतु अवृश्चत् वृश्चेत् ) आशिषि ( वृश्च्यात् अव्रश्चीत् अव्रश्चिष्टाम् अव्रश्चिषुः अव्रश्चीः अव्रश्चिष्टम् अव्रश्चिष्ट अव्रश्चिषम् ) "नेटि'' इति न वृद्धिः अनिटि तु ( अव्राक्षीत् ) पूर्ववत्सलोपषत्वयोः कुत्वम् ( अव्राष्टाम् ) "झलो झलि'' इति सिचो लोपे प्रत्ययलक्षणेन"षढोः कः सि'' इति न भवति तस्य वर्णाश्रयत्वाद्वर्णाश्रये च प्रत्ययलक्षणाभावात् ( विव्रश्चिषति विव्रक्षति वरीवृश्च्यते ) "रीगृत्वतः'' इति रीक् ( वाव्रष्टि वाव्रष्टः )व्रश्चतीति श्तिपा
