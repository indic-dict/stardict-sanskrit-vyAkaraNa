 ( चकास्ति, चकास्तः, चकासति, चकास्सि, चकास्मि, चकास्वः चकासांचकार ) "कास्यनेकाच'' इत्याम् ( चकासिता चकासिष्यति चकास्तु, चकास्तां, चकासतु, चकाधि, ) "धि च'' इति सलोपः केचित्तु "धि च'' इति सिचो लोपमिच्छन्तः चकाद्धि इति सकारस्य दत्वमुदाहरन्ति तत् "यथा तु भाष्यं तथा चकाधीत्येव भवितव्यम्' इति वदाता वृत्तिकारेणैव प्रत्युक्तम् ( अचकात् ) "तिप्यनस्तेः'' इति सकारस्य दत्वम् ( अचकास्ताम्, अचकासुः ) "सिजभ्यस्त'' इति जुस् ( अचकात्, अचकाः ) "सिपि धातो रुर्वा'' इति रुत्वदत्वे ( अचकासम् चकास्यात्, चकास्याताम् ) आशिषि ( चकास्यात्, चकास्यास्ताम् अचकासीत्, अचकासिष्टाम्, अचकासीः, अचकासिषम् चिचकासिषति चकासयति अचीचकासत्,) ऋदित्त्वात् "णौ चङि'' इति न ह्रस्वः 64
