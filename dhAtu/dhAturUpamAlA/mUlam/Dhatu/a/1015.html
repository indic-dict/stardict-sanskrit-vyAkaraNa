ज्ञीप्सायाम्
 ( पृच्छति ) "छे च'' इति तुकि "स्तोः श्चुना श्चुः'' इति श्चुत्वम् "ग्रहिज्या'' आदिना सम्प्रसारणं ङिति (पप्रच्छ पप्रच्छतुः पप्रष्ठ पप्रच्छिव) क्रादिनियमादिट् थलि तु "उपदेशेऽत्वत'' इति निषेधाद्भारद्वाजनियमाच्च विकल्पः अनिटि व्रश्चादिना षत्वे "ष्टुना ष्टुः'' इति थकारस्य ठकारः अन्तरङ्गत्वात्पूर्वन्तुकि संयोगात्परत्वाल्लिटः कित्त्वाभावान्न सम्प्रसारणम् ( प्रष्टा प्रक्ष्यति ) "षढोः कः सि'' इति कत्वं, ( पृच्छतु अपृच्छत् पृच्छेत् ) आशिषि ( पृच्छ्यात् अप्राक्षीत् अप्राष्टाम् ) "झलो झलि'' इति सिचो लोपः "वदव्रज'' इति वृद्धिः (पिपृच्छिषति ) "किरश्च पञ्चभ्य'' इतीट्, "रुदविद'' इत्यादिना सनः कित्त्वात्सम्प्रसारणम्, ( परीपृच्छ्यते ) "रीगृत्वत'' इति रीक् ( पाप्रव्छीति पाप्रष्टि पाप्रष्ट ) इत्यादि, पृच्छतीति श्तिपा निर्देशान्न सम्पसारणम् ( प्रच्छयति, अपप्रच्छत्, सम्पृच्छते ) "समो गम्यृच्छि'' इत्यकर्मकात्सम्पूर्वात्तङ् ( सम्पप्रच्छे, सम्पप्रच्छिषे सम्प्रष्टा सम्प्रक्ष्यते सम्पृच्छताम् समपृच्छत सम्पृच्छेत सम्प्रक्षीष्ट समप्रष्ट समप्रक्षाताम् समप्रक्षत ) इति, झलि सिचो लोपे ष्टुत्वम् अन्यत्र कत्वम् ( आपृच्छते गुरुम् ) [ आङि नुप्रच्छयोरुपसंख्यानम् ] इति तङ् ( प्रश्नः ) "यजयाच'' इति नङ् "च्छ्वोश्शूड्'' इति सतुक्कस्य च्छस्य शकारः, "शात्'' इति श्चुत्वनिषेधः "प्रश्ने चासन्नकाले'' इति निर्देशान्न सम्प्रसारणम् पृच्छतीति ( प्राट् ) शब्दं पृच्छतीति ( शब्दप्राट् "क्विब्वचिप्रच्छि'' इत्यादिना निरुपपदात्सोपपदाच्च पृच्छेः क्विप्, दीर्घोऽसम्प्रसारणं च "च्छ्वोश्शूड् इति शकारः "व्रश्च'' आदिना तस्य पदान्ते षत्वं, जश्त्वचर्त्वे च (पृच्छा) भिदादेराकृतिगणत्वादङित्यात्रेयः ( पृष्ट्वा पृष्टः पृष्टवान् ) अयं द्विकर्मकः अत्र "अप्रधाने दुहादीनाम्'इति लादयोऽप्रधाने भवन्ति (माणवकः पृच्छ्यते पन्थानम् ) ( प्रष्टव्यः पृष्टः पृष्टवान् ) इति अन्यदपि द्विकर्मकसाधारणं नाथत्यादौ प्रपञ्चितं तत एवावधार्यम् 129
