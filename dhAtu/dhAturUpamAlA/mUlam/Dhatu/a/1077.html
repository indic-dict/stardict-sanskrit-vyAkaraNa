हिंसायाम्
 ( मीनाति मीनीतः मीनासि मीनामि मीनीवः मीनीते मीनाते मीनीषे मीने मीनीवहे ममौ मिम्यतुः ममिथ ममाथ मिम्यथुः मिम्य ममो मिम्यिव) "मीनातिमिनोति''इत्येज्विषये आत्वं ल्यपि च अन्यत्र "एरनेकाच''इति यण् (मिम्ये मिम्यिषे मिम्ये मिम्यिवहे माता मास्यति मास्यते मीनातु मीनीतम् अमीनात् अमीनीत मीनीयात् मीनीत ) आशिषि ( मीयात् ) यासुटः कित्त्वादनेज्विषयत्वान्नात्वम् ( मासीष्ट अमासीत् अमासिष्टाम् अमास्त अमासाताम्) आत्वे "यमरमनमाताम्'' इति सगिटौ सनि ( मित्सति मित्सते ) "सनिमीमा''' इत्यच इस् "सः स्यार्द्धधातुके'' इति तत्वम् "अत्र लोप'' इत्यभ्यासलोपः ( मेमीयते मेमेति ) "मीनाति'' इति स्तिपा निर्देशान्नात्वम् (मापयति) आत्वे पुक् ( अमीमपत् प्रमीणाति ) "हिनुमीना'' इत्युपसर्गस्थान्निमित्तात्पस्यमीनाशब्दनकारस्य णत्वम् (प्रमीणीतः प्रमीणन्ति) इत्यादौ परनिमित्तकस्येत्वस्याल्लोपस्य च पूर्वविधौ स्थानिवत्त्वादेकदेशविकृतस्यानन्यत्वाद्वा णत्वम् ( मीत्वा मीतः मीतवान् प्रमाय ) "मीनाति'' इति ल्यप्यात्वम् ( ईषत्प्रमयः मयः ) ( निमिमीलियां खलचोः प्रतिषेधः ] इत्यात्वनिषेधः 4
