 ( ऋषति प्रार्षति ) "उपसर्गादृति'' इति वृद्धिः ( आनर्ष आनृषतुः आनृषुः आनर्षिथ आनृषिव ) अभ्यासस्योरदत्वे हलादिः शेषे "अत आदेः'' इति दीर्घः "तस्मान्नुड् द्विहल'' इति नुट् ( अर्षिता अर्षिष्यति ऋषतु आर्षत् ) "आटश्च'' इति वृद्धिः ( ऋषेत् ऋष्यात् ) मा भवानर्षीत् "नेटि'' इति न वृद्धिः ( अर्षिषिषति ) परत्वाद्गुणे कृते द्वितीयस्यैकाचो द्विर्वचनं "नन्द्राः'' इति रेफस्य न भवति (अर्षयति आर्षिषत् अर्षित्वा ) "न क्वा सेड्'' इत्यकित्त्वम् ( ऋष्टः ऋष्टवान् ) ईदित्त्वादनिट्त्वम् ( ऋषभः ) "ऋषिवृषिभ्यां किद्'' इत्यभच् प्रत्ययः ( आर्षभ्यो ) वत्सः "ऋषभोपानहोर्ञ्यः'' इति विकृतिवाचिन ऋषभशब्दाद्विकृत्यर्थायां प्रकृतौ ञ्यः अत्र ऋषभावस्था विकृतिः वत्सावस्था प्रकृतिः ( ऋषभतरः ) भारवहने मन्दशक्तितरनड्वान् "वत्सोक्ष'' इत्यादिना ष्टरच् अत्र तनुत्वेन भारवहनाशक्तिरुच्यते (ऋक्षः) "स्नुव्रश्चिकृत्यृषिभ्यः कित्'' इति स उदात्त उदात्तेत् 7
