मुचादयोऽनुदात्ताः स्वरितेतः।
मुचॢ
 
मोक्षणे
 ( मुञ्चति ) "शे मुचादीनाम्'' इति शे नुम् ( मुमोच मुमुचतुः मुमोचिथ मुमुचिव मोक्ता मोक्ष्यति मुञ्चतु अमुञ्चत् मुञ्चेत् मुच्यात् अमुचत् अमोक्ष्यत् मुञ्चते मुमुचे मोक्ता मोक्ष्यते मुञ्चताम् अमुञ्चत मुञ्चेत मुक्षीष्ट अमुक्त अमुक्षाताम् ) झलि सिचो लोपः "चोः कुः'' ( मुमुक्षति मुमुक्षते वत्सं) कर्मकर्तरि मोक्षते वत्सः स्वयमेवेत्यपि भवति "मुचो ऽकर्मकस्य गुणो वा'' इति सन्नन्तस्य पक्षे गुणे "अत्र लोपोऽभ्यासस्य'' इत्यभ्यासलोपः, यक्चिणोः "भूषाकर्मकिरादिसनाम्'' इति निषेधः ( मोमुच्यते मोमोक्ति मोचयति अमूमुचत् मोक्षः ) सन्नन्तात् घञि पूर्ववद्गुणाभ्यासलोपौ निर्मुच्यत इति ( निर्मोकः ) घञि "चजोः कुघिण्ण्यतोः'' इति कुत्वम् "अमोच्यमश्वं यदि मन्यसे' इत्यत्रावश्यके ण्यः "ण्य आवश्यक'' इति कुत्वनिषेधः ( नखमुचानि धनूंषि, ) मूलविभुजादित्वात्कर्तरि कः मुष्टेर्बहिर्भूतानीत्यर्थंः ( जलमुक् जलमुचौ ) क्विप् ( मुक्ता,) "क्तिच्क्तौ च संज्ञायाम्'' इति संज्ञायां क्तः मुक्तेव ( मौक्तिकम् ) "विनयादिभ्यश्च'' इति स्वार्थे ठक् ( नमुचिः ) "इगुपधात्किच्च'' इति इन् "नभ्राड्'' इत्यादिना नञः प्रकृतिभावः ( चक्रमुक्तः ) "अपेतापोडमुक्त'' इति पञ्चमीसमासः प्रमोचनेयं चुरादिः मुञ्चते,मुञ्चति इति कल्कने शपि गतम् 176
