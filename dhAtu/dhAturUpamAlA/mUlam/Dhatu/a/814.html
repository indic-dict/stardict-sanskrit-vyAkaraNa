युध
 
सम्प्रहारे
 ( युध्यते ) इत्यादि बुधिवत् ते तु अयुद्ध इति सिजेव ( राजयुध्वा, ) राजानं योधितवानित्यर्थः "राजनि युधि कृञः'' इति राजनि कर्मण्युपपदे भूते क्वनिप् इहान्तर्भावितण्यर्थत्वेन सकर्मकत्वम् ( सहयुध्वा, ) "सहे च ''इति भूते क्वनिप् ( राजयुध्वा ब्राह्मणी, सहयुध्वा ब्राह्मणी ) [ वनो न हश ] इति ङीब्रेफयोर्निषेधः ( योधः, यौधः, ) प्रज्ञादिपाठात्स्वार्थे वाण् अत एव पाठादिगुपधलक्षणं कं बधित्वाच् आदाय युध्यतेऽनेनेति ( आयुधम्, ) [घञर्थे कविधानम्] इति कः आयुधेन जीवति ( आयुधीयः, आयुधिकः, ) "आयुधाच्छ च'' इति जीवतीत्यर्थे ठक्छौ ( युयुधानः,) बाहुलकादानचि द्वित्वम् ( युध्मः ) आहवो योद्धा च, "इषियुधि'' इत्यादिना मक् (दुर्योधनः, सुयोधनः,) "भाषायां शासियुधि'' इति खलर्थे युच् युध्यतीति युद्धमिच्छतीति क्यचि 65
