स्फुटिर्
 
विशरणे
 अयं पाठो मैत्रेयादीनाम्( 1 ) ( 1 ) देवेत्यधिकं क्कचित् पु0 ( स्फोटति पुस्फोट स्फोटिता पुस्फुटिषति पुस्फोटिषति स्फुटित्वा स्फोटित्वा ) इत्यादि इकाररेफयोः "उपदेशेऽजनुनासिक इत्" "हलन्त्यम्'' इतीत्त्वं, तेन लुङि "इरितो वा'' इति परस्मैपदे विधीयमानो ऽड्विकल्पो भवति ( अस्फुटत् अस्फोटीदिति ) "इदितो नुम् धातोः'' इति नुम् अन्तेदितो विधानादस्य न भवति स्फुटेति चन्द्रः अस्याङ् नास्ति स्वामिकाश्यपौ तु स्फट स्फुठि स्फटिरिति त्रीन् धातून् पठतः ( स्फण्ठति स्फोटतीत्यादि स्फुटा ) अच् टाप् स्फुट विकाशनइत्यात्मनेपदिषु टान्तः 325
