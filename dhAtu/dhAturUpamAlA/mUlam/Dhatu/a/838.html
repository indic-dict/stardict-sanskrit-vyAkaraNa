द्रुह
 
जिघांसायाम्
 ( देवदत्ताया द्रुह्यति ) "क्रुधद्रुह'' इति दोषविषयस्सम्प्रदानम् ( देवदत्तमभिद्रह्यति ) "क्रुधद्रुहोरुपसृष्टयोः कर्म'' इत्युपसृष्टत्वे कर्मत्वम् ( दुद्रोह, दुद्रुहथुः, दुद्रोहिथ, ) अनिट्पक्षे "वा द्रहमुहष्णुहष्णिहाम्'' इति झलि पदान्ते च हकारस्य वा धत्वे "झषस्तथोः" इति थकारस्य धत्वे "झलां जश् झशि'' इति घकारस्य गत्वे, ( दुद्रोग्ध ) धत्वाभावे "हो ढः'' इति ढत्वे धकारस्य धत्वष्टुत्वयोः ढो ढे लोपे, ढलेपदीर्घात्परत्वाद्गुणे, ( दुद्रोढ, दुद्रुहथुः, दुद्रहिव, दुद्रह्व, द्रोहिता, द्रोग्धा, ) घत्वादि पूर्ववत् ( ढत्वादि ( द्रोहिष्यति ध्रोक्ष्यति ) कृते घत्वे सकारपरत्वेन "एकाच'' इति भष्भावे "खरि च'' इति चर्त्व ककारः ढत्वपक्षेऽपि कृते ढत्वेपदाभावे च "षढोः कस्सि'' इति कत्वम्, इदमेव रूपम् ( द्रुह्यतु अद्रुह्यत् द्रह्येत्, द्रह्यात् अद्रुहत् दुद्रोहिषति, दुद्रुहिषति, ) "रलो व्युपधात्'' इति कित्त्वविकल्पः अनिटि (दुध्रुक्षति,) घत्वढ्वयोर्भषभावादिः पूर्ववत् (दोद्रुह्यते दोद्रोग्धि, दोध्रोढि, दोध्रोक्षि द्रोहयति अदुद्रुहत् द्रोही ) सम्पृचादिना घिनुण् (द्रुहित्वा, द्रोहित्वा, द्रुग्ध्वा, द्रुढ्वा द्रुग्धः, द्रुढः, द्रुढवान् ) घत्वादि पूर्ववत् ढत्वे धत्वष्टुत्वढलोपदीर्घाः ( मित्रध्रुक् मित्रध्रुट् ) "सत्सूद्विषद्रुह'' इति क्विपि पदान्तत्वे "वा द्रुह'' इति वा घत्वम् अन्यदा ढत्वम् भष्भावाद्युभयत्रापि भवति 89
