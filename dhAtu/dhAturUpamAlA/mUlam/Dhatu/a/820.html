अल्पीभावे
 ( लिश्यते लिलिशे लेष्टा लेक्ष्यते ) ब्रश्चादिना षत्वे कत्वम् ( लिश्यताम् अलिश्यत लिश्येत, लिक्षीष्ट अलिक्षत अलिक्षाताम् ) "शल इगुपधात्'' इति क्सः "क्सस्याचि'' इत्यकारलोपः ( लिलिक्षते लेलिश्यतेलेलेष्टि लेशयति अलीलिशत् लिक्षा, ) बाहुलकात् क्सप्रत्ययः अयं गत्यर्थस्तुदादौ पदादयोऽनुदात्ता अनुदात्तेतो ण्यतिवर्जम् 71
