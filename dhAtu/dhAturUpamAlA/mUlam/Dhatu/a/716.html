शुद्धौ
 ऊदित्त्वमिङ्विकल्पार्थम् अस्यानुदात्तत्वमनार्षम् उक्तं च "सृजिं मृजिं विद्ध्यनिट्स्वरान्' इति कारिकायां वृत्तौ मृजेरूदित्त्वादिटो विकल्पेन भाव्यम् आगमश्च न दृश्यते तदिहास्य पाठे प्रयोजनं चिन्त्यम् के चित्त्वस्य स्थाने विजिं पठन्ति इति (मार्ष्टि, मृष्टः, मृजन्ति, मार्जन्ति मार्क्षि, मार्ज्मि, मृज्वः) व्रश्चादिना षत्वे तवर्गस्य ष्टुत्वम् सकारे तु "षढोः कस्सि'' इति कत्वे सकारस्य षत्वम् क्ङितोऽन्यत्र "मृजेर्वृद्धिः'' इत्यचो वृद्धिः अजादौ क्डिति "मृजेरजादौ सङ्क्रमे विभाषा वृद्धिः'' इतीष्ट्या वा वृद्धिः इङ्क्रमो नाम गुणवृद्धिप्रतिषेधविषयः अत्र हरदत्तः "अचीत्युच्यमानेऽपि "यस्मिन् विधिस्तदादौ' इत्यजादाविति सिद्धे आदिग्रहणं मुख्याजादिपरिग्रहार्थंम् तेन व्यपदेशिवद्भावेनाजादौ यूयं ममृज त्वया ममार्ज इत्यादौ न भवति' इति, आत्रेयस्तु "अत्रापि भवति' इति तस्यादिग्रहणं यस्मिन्विधिः' इति सिद्धस्यैवानुवाद इति मतम् यदाह "मृजेति भिदादिपाठादङ् गणपाठादेव वृद्ध्यभावश्च'' इति तथा च "इको गुणवृद्धी'' इत्यत्र भाष्ये "मृजेर्वृद्धिरचस्ततो ऽचि, क्ङित्यजादौ क्ङिति मृजेर्वृद्धिर्भवति ; ( परिमृजन्ति परिमार्जन्ति, ) किमर्थमिदं, नियमार्थम्, अजादावेव क्ङिति नान्यत्र क्व माभूद् मृष्टः, मृष्टवानिति ततो वा चि क्ङिति मृजेर्वृद्धिर्भवति'' इति ( ममार्ज, ममार्जजतुः, ममृजतुः ममार्जुः, ममृजतुः ममार्जिथ, ममार्ष्ठ, ममार्जिव, ममृजिव, ममृज्व ) ऊदित्वादिड्विकल्पः पूर्ववदजादौ वृद्धिविकल्पः ( मार्जिता, मार्ष्टा मार्जिष्यति, मार्क्ष्यति ) अनुदात्तोपदेशत्वस्यानार्षत्वाद अमागमो नेत्युक्तम् ( मार्ष्टु, मृष्टात्, मृष्टां, मृजन्तु, मार्जन्तु, मृढ्ढि, मार्जानि, मार्जाव, मार्जाम ) "आडुत्तमस्य पिच्च''इति पित्वात्सङ्क्रमत्वाभावाद् नित्या वृद्धिः (अमार्टं, अमृष्टाम्, अमृजन्, अमार्जन्, अमार्ट्, अमार्जम्, अमृज्व,) तिप्सिपोर्हल्ङ्यादिलोपे षत्वजश्त्वचर्त्वेषु च "रात्सस्य'' इति नियमात् न संयोगान्तलोपः ( मृज्यात्, मृज्यातां, मृज्याः मृज्यां, ) मृजेर्वृद्धिगुणविषये' इति सा न भवति आशिषि, ( मृज्यात्, मृज्यास्ताम् ) इत्यादि (अमार्जीत् "इट ईटि'' इति सिचो लोपः (अमार्जिष्टाम्, अमार्जिषुः, आमार्जीः अमार्जिषम्) अनिट्पक्षे, (अमार्क्षीत्, अमार्ष्टाम्, अमार्ष्टाम्, अमार्क्षुः, अमार्क्षीः, अमार्क्षम्) झलादौ "झलो झलि'' इति सिज्लोपः अन्यत्र प्रकृतेः पूर्ववत् षत्वकत्वयोः, "आदेशप्रत्यययोः'' इति षत्वम् एवं ( व्यतिमृष्टे, व्यतिमृजाते, व्यतिमृजते ) इत्यादावात्मनेपदेऽपि यथायोगंवृद्धिस्तदभावश्च ग्राह्यः मिमृक्षति, मिमार्जिषति मरीमृज्यते मरीमार्जीति मरीमार्ष्टि) एवं रुग्रीकोरपि ( मार्जयति अमीमृजत् अममार्जत् ) "उरृत्'' (मार्जित्वा, मृष्ट्वा)इटपक्षे "न क्त्वा सेट्''इति कित्त्वनिषेधान्नित्यं वृद्धिः ( मृष्टः मृष्टवान् ) "यस्य विभाषा'' इतीण्णिषेधः (सिक्तसम्मृष्टम्)राजदन्तादित्वात्पूर्वकालस्य परनिपातः (मृज्यः) "मृजेर्विभाषा'' इति वा क्यप् तदभावे ण्यति कुत्वे वृद्धौ च ( मार्ग्यः ) "उपसर्गस्य घञ्यमनुष्ये बहुलम्'' इत्युपसर्गस्य दीर्घः (कंसपरिमृड्) इत्यादौ क्विबन्तस्य सुबपेक्षया मृजेर्वृद्धिर्न भवति, "घातोस्स्वरूपग्रहणे तत्प्रत्यये कार्यविज्ञानम्' इति कंसपरिमृजमिच्छतीति विगृह्य
