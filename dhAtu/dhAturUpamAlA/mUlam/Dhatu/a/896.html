शक्तौ
 ( शक्नोति शक्नुवन्ति शक्नुहि अशक्नोत् शक्नुयात् ) शेषं शक्यतिवत् लुङि लृदित्त्वादङ्, ( अशकत् ) इति विद्यासु शिक्षते "शिक्षेर्जिज्ञासायाम्'' इति तङ्, विद्याविषये ज्ञाने शक्तो भवितुमिच्छतीत्यर्थः ( शक्तो ) घटः, कर्त्तुं (शकित) इति वा सौनागाः कर्मणि निष्ठायां शकेरिटमिच्छन्ति विकल्पेनेतीड्विकल्पः "विभाषाभावादिकर्मणोः'' इत्यत्र हरदत्तः यद्यपि शकिः केवलोऽकर्मकसतथापि तुमुनन्तवाच्यक्रियाविषयत्वे सकर्मको भवति, तथा च कर्मणि लादयोऽपि दृश्यन्ते अयं योगः शक्योवक्तुमितीति ( शक्यम् ) "शकिसहोश्च'' इति यत् (शकलम्) "शकिशम्योर्नित्'' इति कलप्रत्ययः ( शकः ) "स्फायितञ्चि'' इति रः ( शक्लः प्रियंवदः ) "शक्यादिभ्यः क्ल'' इत्यूहितः क्लः 15
