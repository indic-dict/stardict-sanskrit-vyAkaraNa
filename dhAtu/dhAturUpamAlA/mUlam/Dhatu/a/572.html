 एतदादयो जयत्यन्ता अनुदात्ता उदात्तेत्तः ( धयति दधौ दधतुः दथुः दधाथ दधिथ दधिव ) आदेच उपदेशे शितिइति एजन्तो यो धतुरुपदेशे तस्यात्वमिति सर्वत्राकारे "आत औ णल" इति औत्वे वृद्धिः, अन्यत्र क्ङित्यजादावार्द्धधातुके "आतो लोपः'' इत्याल्लोप डटि च तस्य "द्विर्वचनेचि''इति स्थानिवत्त्वाद्धाशब्दस्य द्विर्वचनं, क्रादिनियमादिट्. थलि भारद्वाजनियमादिड्विकल्पः ( धाता धास्यति धयतु अधयत् धयेत् आशिषि धेयात् ) "एर्लिङि'' घुमास्थागापाजहातिसामित्येकारः दाधा ध्वदाप्इति दाण् दाने, देङ् रक्षणे, डुदाञ् दाने, दो अवखण्डने, धेट् पाने, बुधाञ् धारणादौ एते धुसंज्ञकाः, दाप् शोधने दैप् लवन इत्येतौ वर्जयित्वा प्रकृतिवदनुकारणं भवतीत्यात्वेनैजन्तानुकरणग्रहणार्थं दैपः पित्त्वमर्थवद्भवति ( अदधत् अदधताम् ) विभाषा धेट्श्वयोःइति कर्तृवाचिनि लुङि च्लेर्विभाषा चङि "आतो लोप इटि च'' इतृयाल्लोपस्य स्थानिवत्त्वात् अधुरित्यादि) आतःइति झेर्जुस् सिज्लुगन्ताद्यदि भवति आत एवेति नियमार्थमिदं सूत्रमित्युक्तम्, यदा कर्मव्यतिहारे तङ तदा(व्यतिधयते व्यतिदधे) इत्यादि आदेव उपदेशेइत्यत्र शितीति कर्मधारये नञः तत्पुरुषे श एव कित्त्वे धातोश्चेत्वे "हस्वादङ्गात्'' इति ज्ञल्परत्वेन सिचो लोपेन ( व्यत्यधि व्यत्यधिषत ) सिचः कित्त्वात् गुणो न भवति इत्वविधानं तु गुणविधानार्थमेव स्यात् "स्याध्वोरिच'' इति एकारस्याविधानमपि हस्वादिति
