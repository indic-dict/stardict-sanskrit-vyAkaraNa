कासृ
 
शब्दकुत्सायाम्
 ( कासते कासां चक्रे ) "कास्प्रत्यया"इत्याम् ( कासिता कासिष्यते कासताम् अकासत कासेत कासिषीष्ट अकासिष्ट चिकाषते चाकास्यते ) लोटि हेर्धिभावे "धिच'' इति सलोपे चकाधि न च "धिच इत्यत्र भाष्ये सिजपेक्ष्यते लङि तिपिहल्ङ्यादिलोपे प्रत्ययलक्षणेनतिप्स्वरत्वात् तिष्यनस्तेःइति अस्तिवर्जितस्य सान्तस्य पदस्य तिपि परे दकार इति दकारे तस्य चर्त्वे ( अचकात् ) सिपि तुसिपि धातोरुर्वाइति सान्तस्य अस्तिवर्जितस्य धातोः पदस्य पक्षे रुत्वे (अचकाः) इत्यपि भवति ( कासयति अचकासत ) ऋदित्त्वान्नोपधाह्रस्वः (कासः) "हलश्च'' इति संज्ञायां घञ् (कासारः) औणादिको बहुलाकादारः (कासूः शक्तिः कासूतरी ) "कासूगोणीभ्यां ष्टरच्'' इति अल्पत्वे द्योत्ये ष्टरच् षित्त्वात् ङीष् अस्मादेवच निर्देशादूकारप्रत्ययःकसतीति ज्वरादौ गत्यर्थः कस इति गतिशासनयोरादिकस्य तत्रधान्ये निदित्यादिना ऊः ( कसूः ) इत्याहुः अन्ये तु तालव्यान्त पठन्ति (कशूरिति) 614
