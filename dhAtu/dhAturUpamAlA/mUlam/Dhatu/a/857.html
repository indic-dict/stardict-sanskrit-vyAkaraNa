 ओष्ट्यादिः ( बिस्यति ) इत्यादि ( बिसम् ) इगुपधलक्षणः कः ( बिस्तः ) बाहुलकादौणादिकस्तन्, अगुणत्वं च "तितुत्र'' इतीरण्णिषेधः द्वाभ्यां बिस्ताभ्यां क्रीता ( द्विबिस्ता, द्वैबिस्तीकी ) "बिस्ताच्च'' [ बहुपूर्वाच्च वक्तव्यम् ] इति द्वित्रिबहुपूर्वाद्विस्तान्ताद् द्विगोःपयस्यार्हीयस्य ठञो विभाषा लुक् "द्विगोः'' इति ङीपः"अपरिमाणबिस्त'' इति निषेधः अत एव बिस्तेतिनिर्देशाद्वा निष्ठायामनिट्त्वम् लुगभावे तु "परिमाणान्तस्य'' इत्युत्तरपदवृद्धिः "टिड्ढाणञ्'' इत्यादिना ङीष् एवं त्रिबहुपूर्वादपि द्रष्टव्यम् 109
