धारणे
 ( मलते मेल मलिता ) मल्लतइत्यादि पूर्ववत् ( माला ) "अकर्त्तरि च्च कारके'' इति कर्मणि घञ् "लस्य'' इत्यत्र भाष्यकैयटयोर्माङो ऽथं लप्रत्ययउक्तः ( माली व्रीह्यादित्वादिनिः ( मालभारी ) "इष्टकेषीकामालानां चिततूलभारीपु'' इति यथासंख्यान्मालायो भारिण्युत्तरपदे ह्रस्वः के चित्तु तत्र हारीति पठन्ति ( माल्यम् ) ऋहलोर्ण्यत्'' ( मलयः ) वलिमलीति कयन् शरीरं धार्यतइति ( मलम् ) अच् मृतेर्वा व्युत्पादयिष्यते (मलिनः मलीमसः) "ज्योत्स्नातमिस्त्राशृङ्गिणोर्जस्विन्नृर्जस्वलगोमिन्मलिनमलीमसाः'' इतीनजीमसजन्तौ मत्वर्थे निपातितौ केन मल्यतइति (कमलम्) घञर्थे कविधानमिति कः, "कर्तृकरणे कृता'' इति समासः कमेर्वा कलप्रत्यये व्युत्पादनीयः ( आमल्ली ) "कुन् शिल्पिसंज्ञायोरपूर्वस्यापि'' इति कुनि गौरादित्वान्ङीष् ( मल्लः ) अच्प्रत्ययः (मल्लिका) "संज्ञायाम्''इति स्त्रियां ण्वुल् मल्लकः क्कन् अलं मल्लते धारयतीति ( मलमल्लं कौपीनम् ) 490
