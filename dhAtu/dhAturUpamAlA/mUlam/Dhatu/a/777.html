 एतदादयोऽनुदात्ता आत्मनेपदिनः पीङ्पर्यन्तः डीङ् तु सेट् (दीयते, दीयेते दिदीये, दिदीयाते, दिदीयिरे, दिदीयिषे,दिदीयिढ्वे, दिदीये, दिदीयिवहे ) क्रादिनियमादिट् "दीङो युडचि क्ङिति'' इत्यजादावाधधातुके क्ङिति युडागमः स च "एरनेकाच'' इति यणि कर्तव्ये "असिद्धवदत्र'' इति नासिद्धः, [ वुग्युटावुवङ्यणोः सिद्धौ, वक्तव्यौ] इत्युक्तत्वादत्र ध्वमि "विभाषेटः'' इति मूर्धन्यविकल्पो न भवति कृते युटि प्रतत्ययभक्तेनानेनइणन्तादङ्गात्परत्वमिटो विहितमिति "इणष्षीध्वम्'' इति नित्यमेव भवतीत्येके अपरे तु अद्गादिति नानुवर्तते इति युट एव यणः परत्वान्मूर्धन्यविकल्पमाहुः उक्तं चैतत्पक्षद्वयं वृत्तौ ( दाता दास्यते ) "मिनातिमिनोतिदीङां ल्यपि च'' इति एज्विषये ल्यपि च आत्वम् (दीयताम्, दीयेतां , दीयस्व, दीयै अदीयत, अदीयेताम्, अदयिथाः, अदीये दीयेत, दीयेथाः, दीयेय ) आशिषि (दासीष्ट) इत्यादि एज्विषयत्वादात्वम् ( अदास्त, अदासाताम्, अदास्थाः, अदासि ) अत्रात्वे कृते दारूपत्वेऽपि [ दीडः प्रतिषेधः स्थाध्वोरित्वे ] इति इत्वकित्त्वयोर्घुसंज्ञा निषिध्यते अत एव विषयविषेशे घुत्वनिषेधादन्यत्र कृतात्वस्य घुसंज्ञायां प्रणिदातेत्यादि भवतीत्येके अन्ये तु यावतामुपदेशस्थानामनुकरणं दाधारूपं भवति, तेषामेव घुत्वम् दोदेङ्धेटामप्यनुकरणवशेन दारूपमेव, दीङस्तु नेति नैवैतस्य क्वापि घुसंज्ञाप्रसङ्ग इति उक्तमिदं पक्षद्वयं कैयटपदमञ्जरीवाक्यमञ्जर्यादिष्वपि ( उपदिदीषते ) "इको झल्'' इति सनः कित्त्वादेज्विषयत्वाभावान्नत्वम् कातन्त्रे (दिदासते) इत्यात्वमुक्तम् अत्र शाकटायनः, आत्वप्रकरणे "अम्यपगुरो वा दीङस्सनि'' इत्यात्वविकल्पमाह अमिति णमुलुच्यते स्वामी तु ( उपदिदीषते दपदिदासते ) इत्यपीष्यते इति काश्यपस्तु आत्वपक्षे दिदासते इत्येकइत्युक्त्वा संग्रह इत्वव्यतिरिक्तस्य घुकार्यस्योक्तत्वाद् इस्भावे उपदित्सतइत्याह इदमात्वं सूत्रवार्तिकभाष्येषु न दृश्यते ( देदीयते, देदेति, दिदीतः, देद्यति दापयति ) एज्विषयत्वाभावादात्वे पुक् ) दीत्वा, दीनः, दीनवान् ) "स्वादय ओदित'' इति निष्ठानत्वम् ( उपदाय ) "मीनातिमिनोतिदीङां ल्यपि च'' इत्यात्वम् 26
