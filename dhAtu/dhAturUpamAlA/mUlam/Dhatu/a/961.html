हिंसार्थाः
 ( तृहति ततर्ह स्तृहति तस्तर्ह ) "शर्पूर्वाः खयः'' ( तर्हिता तर्ढा स्तर्हिता स्तर्ढा ) इत्यादि वृहतिवत् ( तरीष्टृह्यते तरीष्टर्ढि ) "आदेशप्रत्यययोः''इति प्रकृतेः षत्वम् (तिस्तर्हिषति तिस्तृक्षति) इत्यत्र तु "स्तौतिण्योरेव'' इति नियमान्न षत्वम् ( तृहति ) शस्य ङित्त्वात् "अनिदिताम्'' इति नलोपः (ततृंह ततृहतुः ततृंहिथ ततृण्ढ ) धत्वढत्वष्टुत्वडलोपेषु नकारस्य "नश्चापदान्तस्य'' इत्यनुस्वारे परसवर्णो णकारः (तृंहिता तृण्ढा तृंहिष्यति तृङ्क्ष्यति तृहतु अतृहत् तृहेत् तृह्यात् )यासुटः कित्त्वान्नलोपः (अतृंहीत् अतृंहिष्टाम्) अनिट्पक्षेऽप्यनिगुपधत्वात् क्सो नास्तीति सिचि ढत्वकत्वयोः सिचः षत्वं, नकारस्यानुस्वारपरसवर्णौ, वृद्धिः, (अतार्ङ्क्षीत् अतार्ण्ढाम् ) इत्यादि ( तितृंहिषति तितृक्षति ) अनिट्पक्षे "हलन्ताद्'' इत्यत्र हल्ग्रहणस्य जातिवाचकत्वात् सनः कित्त्वान्नलोपः (तरीतृह्यते तरीतृंहीति तरीतृण्ढि ) इत्यादि तृंहयति ) अततृंहत् तृंहित्वा तृण्ड्ढ्वा ) इटि "न क्त्वा सेङ्'' इत्यकित्त्वम् ( तृढः तृढवान् ) अत्र सानुषङ्गं मूर्धन्यादिं चतुर्थमपि ष्टृह इति धातुं केचित्पठन्ति आत्रेयमैत्रेयादिभिरयं न पठ्यते तृह हिंसार्थो रूधादौ 67-68-69
