व्याजीकरणे
 ( विचति ) "ग्रहिज्यादिना'' किति सम्प्रसारणम्, ( विव्याच विविचतुः विव्यचिथ विविचथुः विविच विव्याच विव्यच विविचिव ) अपितो लिटः कित्त्वातृसम्प्रसारणे द्विर्वचनं, पिति तु "लिट्यभ्यासस्योभयेषाम्'' इत्यभ्यासस्य "व्यचेः कुटादित्वमनसि'' इत्यञ्णित्प्रत्यये विधीयमानं ङित्त्वमनसीतिपर्युदासेन तत्सदृशे कृतीति हरदत्तादयः तेन थल्युत्तमे णलि च णित्त्वाभावपक्षे नास्ति ङित्त्वं नापि कित्त्वं, पिल्लिट्त्वादिति न सम्प्रसारणम्. अन्ये तु प्रसज्यप्रतिषेधमाश्रयन्ते, तेनात्रापि ङित्त्वमस्तीति सम्प्रसारणं मन्यन्ते प्रसज्यप्रतिषेधाश्रयणे प्रमाणं मृग्यं, वाक्यभेदप्रसङ्गात् ( व्यचिता व्यचिष्यति विचतु अविचत् विचेत् ) आशिषि ( विच्यात् अव्याचीत् अव्याचिष्टाम् अव्यचीत् अव्यचिष्टराम्) "अतो हलादेः'' इति वा वृद्धिः ( विव्यचिषति ) तासिस्यसिचां सनश्चाकृत्वान्न ङित्त्वमिति न सम्प्रसारणम्, प्रसज्यप्रतिषेधवादिनामत्रापि ङित्त्वात्सम्प्रसारणेन भाव्यम् ( वेविच्यते वाव्यचीति वाव्यक्ति ) श्तिपा निर्द्देशान्न सम्प्रसारणम् ( व्याचयति अविव्यचत् विचित्वा ) "न क्त्वा सेड्'' इत्यकित्त्वेऽपि क्त्वः कृत्त्वाद्व्यचेः कुटादित्वमिति ङित्वेन सम्प्रसारणं गुणाभावश्च (विचितः उरुव्यचाः) गतिकारकपूर्वस्यापीत्यसुन् अनसीतिपर्युदासान्न ङित्त्वम् 13
