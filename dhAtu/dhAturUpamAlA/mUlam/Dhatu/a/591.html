कौटिल्ये
 ( ह्वरति जह्वारा जह्वरतुः जह्वर्थ जह्वरिव ) क्रादिनियमादिट् थलि तु "अवस्तास्वत्'' इति निषिध्यते "ऋतश्च संयोगादेर्गुणः'' इति गुणः अयं वृद्ध्यापूर्वविप्रतिषेधेन स्वविषये बाध्यते ( ह्वर्त्ता ह्वरिष्यति )"ऋद्वनोः स्ये'' इति इट् ( ह्वरतुअह्वरत् आशिषि ह्वर्यात् ) गुणोत्तिसंयोगाद्योःइति अर्त्तेऋकारान्तस्य संयोगादेर्यकियकारावार्द्धधातुके लिङि च णुण इति गुणः ( अह्वर्ष्टाम् यकि ह्वर्यते ( लिङीव गुणः (ह्वष्टांम् यकि ह्वर्यते ( लिङीव गुणःयदाशीलिर्ङि च गुण इति गुणः ( अह्वर्ष्टम् यकि ह्वर्यते ( लिङीव गुणः यदाशीर्लिङि यगात्नेपदे च तदा "लिङ्सिचोरात्मनेपदे'' इत्यधिकृत्य "ऋतश्च संयोगादेः''इति पक्षे इटि ( व्यतिह्वरिषीष्ट व्यातृयह्वर्तेत्यादि)"लिङ्सिचावात्मनेपदेषु'' इति अधिकृत्य "उश्च''इति कित्त्वविधानात् न गुणः ज्ञलि सिज्लोपः ( जुहूर्षति ) "अज्ज्ञनगमां सनि'' इति दीर्घे "उदोष्ठ्यपूर्वस्य'' इत्युत्वे रत्वे च "हलि च''इति दीर्घः ( जर्ह्वर्त्ति जर्ह्वर्त इत्यादि ) लोटि हेर्डित्त्वात् ( जर्ह्वहि ) उत्तमे ( जर्ह्वराणि ) लङि ( अजर्ह्वः ) ( हृतम् ) 914
