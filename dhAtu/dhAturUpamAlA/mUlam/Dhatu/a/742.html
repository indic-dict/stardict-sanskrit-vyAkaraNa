णिजिर्
 
शौच,पोषणयोः
 एतदादयो विष्लृपर्यन्तास्त्रयोऽनुदात्तास्स्वरितेतः ( नेनेक्ति, नेनिक्तः, नेनिजति, नेनेक्षि, नेनेज्मि, नेनिज्वः, ) "निजां त्रयाणां गुणश्श्लौ'' इत्यभ्यासस्य गुणः झलि पदान्ते च "चोः कुः'' इति कुत्वे "खरि च'' इति चर्त्वम् (निनेज, निनिजतुः, निनेजिथ, निनिज, निनिजिव ) क्रादिनियमादिट्, ( नेक्ता नेक्ष्यति, नेनेक्तुनेनिक्तात्, नेनिक्ताम्, नेनेग्धि, ) हेरपित्त्वान्ङित्त्वान्न गुणः ( नेनिजानि, "नाभ्यस्तस्य'' इति गुणनिषेधः ( अनेनेक्, अनेनिक्ताम्, अनेनितुः, अनेनेक्, अनेनिजम् ) अन्तरङ्गतवान्मिपोमि "नाभ्यस्तस्याचि पिति'' इति गुणनिषेधः (नेनिज्यात्, नेनिज्यातां, नेनिज्याः, नेनिज्याम् ) आशिषि ( निज्यात् अनिजत्, अनिजताम्, अनिजद्, अनिजः, ) "इरितो वा'' इति च्लेर्वाङ् परस्मैपदे अन्यदा ( अनैक्षीत्, अनैक्ताम्, अनैक्षुः,अनैक्षी, अनैक्षम्, अनैक्ष्व, ) "वदव्रज'' इति वृद्धिः झलि सिज्लोपः ( नेनिक्ते, नेनिजाते, नेनिजते, नेनिक्षे, नेनिजे, नेनिज्वहे, निनिजे, निनिजिवहे नेक्ता नेक्ष्यते नेनिक्तां, नेनिक्ष्व, नेनिजै अनेनिक्त, अनेनिक्थाः, अनेनिजि नेनिजीत नेनिजीयाताम् ) आशिषि (निक्षीष्ट अनिक्त, अनिक्षातां) "लिङ्सिचावात्मनेपदेषु'' इति लिङ्सिचोः कित्त्वान्न गुणः ( निनिक्षति, निनिक्षते ) "हलन्ताच्च'' इति सनः कित्त्वम् ( नेनिज्यते, नेनिजीति, नेनेक्ति नेजयति, अनीनिजत् निक्त्वा, निक्तः, निक्तवान् निजः ) इगुपधलक्षणः कः ( निर्णेजलः ) ण्वुक् ( प्रणेनेक्ति ) "उपसर्गादसमासेऽपि'' इति णत्वम् "उभौ साभ्यासस्य'' इति लिङ्गात् कृते द्विर्वचने प्रवर्तते 11
