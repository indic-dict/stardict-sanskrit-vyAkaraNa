वरणे
 ( वृणाति वृणीते ववार ववरे वरिता वरीता, वरिष्यति वरीष्यति वरिष्यते वरीष्यते वृणातु वृणीतामवृणात् अवृणीत वृणीयात् वृणीत ) आशिषि ( वूर्यात् ) "उदोष्ठ्यपूर्वस्य'' इत्यगुणवृद्धिविषये उत्वे रपरत्वे च "हलि च'' इति दीर्घः वरिषीष्टा वूर्षीष्ट ) "लिङ्सिचोरात्मनेपदेषु'' इतीडभावपक्षे "उश्च'' इति कित्त्वादुत्वादि इट्पक्षे पूर्ववत् "न लिङि'' इति अदीर्घत्वम् ( अवारीत् अवारिष्टाम् ) पूर्ववत् "वॄतः'' इति दीर्घस्य "सिचि च परस्मैपदेषु'' इति निषेधः ( अवरिष्ट अवरीष्ट अवूर्ष्ट ) "लिङसिचोः'' इतीङ्विकल्पे "वॄत'' इति दीर्घः अनिटि "उश्च'' इति कित्त्वादुत्वादि ( वुवूर्षति विवरिषति विवरीषति ) "इट् सनि वा'' इतीड्विकल्पे "वॄत'' इति दीर्घविकल्पः अनिटि "इको झल्'' इति सनः कित्त्वे उत्वादि (वोवूर्यते वावर्ति ) "ऋतश्च'' इति तपरनिर्देशान्न रुगादीत्युक्तम् ( वारयति अवीवरत् वूर्णः वूर्णवान् वूर्णिः ) 14
