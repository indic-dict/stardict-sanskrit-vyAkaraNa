 जप मानसे च ( जल्पति जजल्प जल्पितेत्यादि जल्पाकः ) "जल्पभिक्षकुट्टलुण्टवृङः षाकन् इति तच्छीलादौ षाकन् स्त्रियां जल्पाकी जल्पति पुत्रं देवदत्तः जल्पयति पुत्रं देवदत्तं यज्ञदत्तः [ "जल्पत्यादीनामुपसंख्यानम् ]इति प्रयोज्यस्याशब्दकारकत्वेपि कर्मत्वम्( जपति जजाप जेपतुः जपिता अजपीत् अजापीत् ) "अतो हलादेः'' इति वा वृद्धिः ( जिजपिषति जञ्जप्यते ) "लुपसद'' इत्यादिना भावगर्हायां यङ् "जपजभ'' इत्यादिनाभ्यासस्य नुक्यनुस्वारे परसवर्णविकल्पो जभतिवत् ( जञ्जपीतिजञ्जप्ति जापयति अजीजपत् जप्यम् ) "पोरदुपधात्'' इति यत् (कर्णेजपः) स्तम्बकर्णयोरमिजपोःइति यथासंख्यं जपेः कर्णे सुबन्ते उपपदे च [हस्तिसूचकयोः] इति वचनात् सूचकविषयः, "तत्पुरुषे कृति बहुलम्'' इति सप्तम्या अलुक् ( जञ्जपूकः) "यजजप'' इति यङन्तात्तच्छीलादावुकप्रत्यये "अतो लोपः'' यस्य हलः'' इति अतो लोपयलोपौ ( जपः )"व्यधजपोरनुपसर्गे'' इति अप् उपसृष्टात्तु घञि ( उपजापः ) अनयोः कर्मव्यतिहारे न तङस्ति [ प्रतिषेधे हसादीनामुपसंख्यानम् ] इत्यूक्तत्वात् 395
