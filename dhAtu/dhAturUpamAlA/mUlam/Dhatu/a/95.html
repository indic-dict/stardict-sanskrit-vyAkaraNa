दीप्ति,बन्धनयोः
 ( कञ्चते चकञ्चे कञ्चिता काञ्चत ) इत्यादि( कञ्चुकः ) बाहुलकादुकः प्रत्ययः ( काञ्चनम् ) अनुदात्तेत्त्वाद्युच् काञ्चनस्य विकारः ( काञ्चनम् ) "प्राणिरजतादिभ्योञ्'' इति रजतादित्वादञ् "नित्यं वृद्धशरादिभ्यः' इति मयटोपवादः ( काञ्चिः काञ्ची ) इन्नन्तात् कृदिकारादक्तिनः'' इति वा ङीष् ( काञ्चिकम् ) संज्ञायां कल्कने कन् दीप्तौ चेति वक्तव्ये बन्धनग्रहणं प्रपञ्चार्थम् 168
