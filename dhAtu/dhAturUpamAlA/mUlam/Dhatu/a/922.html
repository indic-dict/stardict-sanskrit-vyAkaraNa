ऋच्छ
 
गति,इन्द्रियप्रलय,मूर्तिभावेषु
 ऋच्छति ) "छे च'' इति तुक्, तुकि यद्वक्तव्यं तत्सर्वं भूवादावृच्छतौ द्रष्टव्यम् ( आनर्च्छ आनर्च्छतुः आनर्च्छुः आनर्च्छिथ आनर्च्छिव ) "ऋच्छत्यॄताम्'' इति लिटि गुणः, "तस्मान्नुड् द्विहलः" इत्यत्र द्विहल्ग्रहणमनेकहलुपलक्षणमिति नुडागम इति हरदत्तादयः ( ऋच्छिता ऋच्छिष्यति ऋच्छतु आर्च्छत् ऋच्छेत् ) आशिषि (ऋच्छयति मा भवानृचिच्छीत् , आर्च्छिष्टाम्, आर्च्छिषम्, ऋचिच्छिषति) अभ्यासे खयः शेषः (ऋच्छयति मा भवान् ऋचिच्छत्) "समो गम्यृच्छि'' इति सम्पूर्वादस्मादकर्मकात्तङ्, ( समृच्छते समानर्च्छे समृच्छिता समृच्छिष्यते समृच्छताम् समार्च्छत समृव्छेत ) आशिषि ( समृच्छिषीष्ट समार्च्छिष्ट ऋच्छित्वा ऋच्छितः ऋच्छितवान् ) ऋच्छां चकारेति प्रयोगो "गुरोश्च हल' इत्यकारप्रत्ययान्तादृच्छशब्दात्कर्मणि द्वितीयायां न त्वामि यतोऽनृच्छ इत्याम्निषिध्यते 16
