षिधु
 
संराद्धौ
 केचिदूदितं पठन्ति तदसत्, अनुदात्तोपदेशवैयर्थ्यात् ( सिध्यति सिषेध सेद्धा सेत्स्यति सिध्यतु असिध्यत् सिध्येत्, सिध्यात्, ) लिङौ ( असिध्यत् सिषित्सति सेषिध्यते सेषेद्धि ) ( अन्नं साधयति, ) "सिध्यतेरपारलौकिके'' इति परलोकप्रयोजनव्यतिरिक्तार्थवृत्तेः सिद्धेः णावेचः आत्वम् पारलौकिके तु तपस्सेधयति तापसम् अत्र सिद्धिः परलोकार्थे ज्ञानविशेषे वर्तते तापस्सिध्यति ज्ञानविशेषमासादयति, तं तपः प्रयुक्ते, स च ज्ञानविशेष उत्पन्नः परलोके जन्मान्तरफलमभ्युदयलक्षणमुपसंहरन् परलोकप्रयोजनो भवति अन्नं साधयति, ब्राह्मणेभ्यो दास्यामीत्यत्र तु षिधेर्निष्पत्तिरर्थः तस्याः प्रयोजनमन्नम्, तस्य यद्दानं तत्पारलौकिकम्, न पुनर्निष्पत्तिरेवेति साक्षात्परलोकप्रयोजने सिद्ध्यर्थे चरितार्थत्वाद् निषेधस्येह न प्रवृत्तिः ( सिद्धः ) ( साङ्काश्यसिद्धः, ) "सिद्धशुष्क'' इति सप्तमीसमासः "तत्पुरुषे कृति बहुलम्'' इत्यलुको "नेन्सिद्धबध्नातिषु च'' इतीवन्ते सिद्धशब्दे बध्नातौ च निषेधः ( सिधित्वा, सेधित्वा, सिद्धा, ) उदित्त्वादिड्विकल्पः, इटि "रलो व्युपधात्'' इति कित्वविकल्पः राधादयोऽनुदात्ता उदात्तेतः अतः परं विभाषेटः परस्मैपदिनो रधादीनष्टाह 84
