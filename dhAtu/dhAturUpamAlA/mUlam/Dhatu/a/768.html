क्षिपिस्त्वनिट् ।
क्षिप
 
प्रेरणे
 ( क्षिप्याति चिक्षेप, क्षिप्ता क्षिप्स्यति क्षिप्यतु अक्षिप्यत् क्षिप्येत्) आशिषि ( क्षिप्यात् अक्षैप्सीत्, अक्षैप्ताम् चिक्षिप्सते, चेक्षिप्यते चेक्षेप्ति क्षेपयति अचिक्षिपत् क्षिप्त्वा क्षिप्तः क्षिपः ) इगुपधलक्षणः कः ( क्षिपका ) अज्ञातादौ कः "प्रत्ययस्थात्'' इतीत्वस्य [ क्षिपकादीनां चोपसङ्क्यानम् ] इति निषेधः (क्षिप्नुः) "त्रसिगृधिधृषि'' इत्यादिना क्नुः ( परिक्षेपी ) सम्पृचादिना घिनुण् ( परिक्षेपकः ) "निन्दहिंस'' इत्यादिना वुञ् ( क्षिपा ) भिदादिपाठाद्ङ् ( क्षिप्रम् ) "ऋज्रेन्द्र'' इत्यादिना रक् ( क्षेपिष्ठः, क्षेपीयान् क्षिपयति ) "स्थूलदूर'' इत्यादिना "णाविष्ठवत्'' इतियणादेर्लोपः, पूर्वस्य च गुणः सम्पृचादिसूत्रे वृत्तिकारेण "क्षिपिर्दिवादिस्तुदादिश्च गृह्यते' इतियणादेर्लोपः, पूर्वस्य च गुणः सम्पृचादिसूत्रे वृत्तिकारण "क्षिपिर्दिवादिस्तुदादिश्च गृह्यते' इत्युक्तत्वात् "द्वाविमावम्भसि क्षिप्य गले बध्वा महाशिलाम्' इति, "संक्षिप्य संरम्भमसद्विपक्षम्' इति, भारते भट्टिकाव्ये च दर्शनात्, पारायणिकैः क्षिपेरिह पाठो ऽयुक्त इति भूसूत्रे सुधाकरेण प्रतिपादितम् कृतस्तौदादिकस्यापि समानः "स्वरेण नीचेन शपिं छुपि क्षपिम्' इति पाठादयमनिट् अयं चुरादावदन्त इत्यात्रेयः 14
