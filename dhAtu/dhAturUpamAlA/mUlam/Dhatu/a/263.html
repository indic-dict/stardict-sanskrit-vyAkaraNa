हिंसार्थाः
 तोपतीत्यादि चुपिवत् ( तुम्पति तुतुम्पतुः ) संयोगान्तत्वाल्लिटो न कित्त्वम् ( तुम्पिता तुम्पिष्यति ( तुम्पतु अतुम्पत् तुप्यात् ) "किदाशिषि'' इति कित्त्वात् अनिदितामिति नलोपः ( अतुम्पीत् अतुम्पिष्यत् तुतुम्पिषति तोतुप्यते तोतुम्पीति तोतुम्प्ति तोतुप्तः तुम्पयति अतुतुम्पत् प्रस्तुम्पति गौः ) "प्रात्तुम्पतौ गवि कर्त्तरि''इति सुट् तुम्पतेस्तुम्पताविति धातुनिर्द्देशो न तिङन्तानुकारणमिति प्रस्तुम्पः प्रस्तुम्पक इत्यादावपि भवति, प्रतोतुम्पीतीत्यादावत एव श्तिपा निर्द्देशान्न भवति ( त्रोपति त्रुम्पति तोफति त्रुम्फति ) इत्यादि तुपितुम्फिवत् तत्रारेफसरेफयोः सानुषङ्गवद्वितीयान्तयोः सेट्त्वाद् "नोपधात्थफान्ताद्वा'' इति कित्त्वविकल्पनात् पक्षे नलोपे ( त्रुपित्वा त्रुम्पित्वा त्रुफित्वा त्रुम्फित्वा ) इति भवति अरेफाश्चत्वारस्तुदादावपि आद्यस्तु चुरादावपि तत्र त्वर्दनमर्थः 408
