उदात्त आत्मनेपदी।
वृङ्
 
सम्भक्तौ
 उदात्त आत्मनेपदी ( वृणीते वृणाते वृणीषे वृणे वृणीवहे वव्रे ववृषे ववृढ्वे वव्रे ववृवहे वरिता वरीता वरिष्यते वरीष्यते वृणीताम् वृणाताम् वृणीष्व वृणै अवृणीत अवृणाताम् अवृणीथाः अवृणि वृणीत वृणयीताम् ) इत्यादि वृणोतिवत्प्रक्रिया, प्रपञ्चस्तत एवावागन्तव्यः ( वर्या ) "अवद्यपण्यवर्या'' इति स्त्रियां यति निपात्यते पुंसि ण्यति ( वार्या ) ऋत्विज इति भवति वर्येति निपातनमस्येति स्थितम् ( वरेण्यः ) "वृङ एण्य'' इत्येण्यः 38
