उक्तं ह्मेवं स्वामिना ( कायति चकौ काता काकः ) "इण्नीक''इति कन् ( गायती जगौ गातेत्यादि ) "घुमास्था'' इत्यत्र गामादाग्रहणेष्वविशेष इति लाक्षणिकगारूपस्य गायतेरपि ग्रहणात् आशीर्लिङि एत्वं गेयादिति ) भवति "गातिस्था'' इत्यत्र गापोर्ग्रहणे इण्पिबत्योर्ग्रहणमिति वृत्तावनुक्तत्वात् अस्मात्_सिवचोलुगभावात् ( अगासीदिति ) भवति ( गाथकः गायनः ) "गस्थकन्'' इति "ण्युट् च'' इति कर्त्तरि थकन्, ण्युटो णित्त्वात् "आतो युक्'' इति युक्, टित्त्वात् स्त्रियां ( गायनी ) वासरूपेण ण्वुलतृचौ, ( गायकः, गातेति ) भवति (उद्गाता) तस्य भावकर्म्मणी ( औद्गात्रम् ) "उद्गात्रादिभ्योञ्'' इति अञ् साम गायतीति, ( सामगः ) गापोष्टक्इति कर्मण्युपपदे टक् टित्त्वात् स्त्रियां ( सामगी गीत्वा गीतम् ) "घुमास्था''इति त्वम् ( प्रगीतिः ) "स्थागापापचो भावे'' इति स्त्रियां भावे क्तिन् "स्त्रियां क्तिन्'' इत्येव सिद्धे पुनर्वचनं "आतश्चोपसर्गे'' इति स्थन् ( अवगथः ) प्रातस्सवनं सोमो वा "गश्चोदि'' इति उदुपसृष्टोत्कथन् "घुमास्था'' इतीत्वम् 901
