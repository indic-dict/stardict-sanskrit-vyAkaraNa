शंसु
 
स्तुतौ
 अयं दुर्गः द्वावपि तालव्योष्मादी ( शंसति शशंस शंसितेत्यादि ) आशिषि कित्त्वात् "अनिदिताम्'' इत्युपधालोपे (शस्यात् शिशंसिषति शाशस्यते ) ङित्त्वान्न्लोपः(शाशंस्ति) यङो लुका लुप्तत्वात् प्रत्ययलक्षणाभावान्नलोपाभावः (शाशसस्तः) इत्यादौ तु तसादीनां ङित्त्वान्न्लोपः (शासधि) इत्यत्र हेरपित्त्वान् ङित्त्वान्नलोपे "धि च'' इति सलोपः (शंसयति अशशंसत् शस्यम् ) [शंसिदुहिगुहिभ्यो वेति वक्तव्यम् ] इति नलोपः क्यबभावे ण्यति (शंस्यम् शंसित्वा शस्त्वा ) उदित्त्वादिङ्विकल्प इति "न क्त्वा सेट्'' इत्यकित्त्वान्न्लोपाभावः (शस्तम्) "यस्य विभाषा'' इति नेट् नॄन् शंसतीति (नृशंसः) कर्मण्यण् नरा एनं शंसन्ति नरा अस्मिव्रासीनाः शंसन्तीति वा (नराशंसः,) घञ् अग्रिर्द्देशविशेषो वा वनस्पत्यादिषु पाठात्पूर्वपदस्य दीर्घः ब्रह्मणानि शंसतीति (ब्राह्मणाच्छंसी) रूढिरियमृत्विग्विशेषस्य "सुप्यजातौ'' इति णिनिः [ ब्राह्मणाच्छंसिन उपसंख्यानम् ] इत्यलुक् अस्मादेव निर्देशात् द्वतीयार्थे पञ्चमी ( श्रेष्ठः ज्येष्ठः श्रेयान् ज्यायान् ) प्रशस्यस्य श्रःज्य चइत्यतिशायनिकयोरजाद्योः श्रज्यावादेशौ भवतः आदेशयोरकारोच्चारणसमार्थ्याट्टिलोपो न भवति ज्यायानित्यत्र "ज्यादादीयसः'' इति ज्यशब्दात्परस्य ईयसुन आदेराकारः श्रेयसोपत्वं (श्रायसः) "देविकार्शिशपादित्यवाङ्दीर्घसत्रश्रेयसामात्'' इति वृद्धिनिमित्तात्परत्वादेकारस्य स्थाने आकारादेशः प्रशस्यमाचष्टे (श्रापयति ज्यापयति) [ तत्करोति ] इति णिचि णाविष्ठवदिति श्रज्यादेशयोर्वृद्धौ पुक् 714
