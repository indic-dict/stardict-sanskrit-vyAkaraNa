भूतप्रादुर्भावे
 
भूतप्रादुर्भावोऽतिक्रान्तोत्पत्तिः
( खच्ञाति ) श्चुत्वेन ञकारः, ( चखाच चखचिथ चखचिव खचिता खचिष्यति खच्ञातु चखान खच्ञानि अखच्ञात् खच्ञोयात् खच्यात् अखचीत् अखाचीत् ) "अतो हलादेः'' इति वा वृद्धिः ( चिखचिषति चाखच्यते चाखक्ति खाचयति अचीखचत् खचित्त्वा खवितः खव इत्येकइति मैत्रेयः स्वाम्यपि खच्ञातीत्युक्त्वा खौनातिरिति सभ्या इति ( खैनातिं खौनातु खौनाहि ) अनुनासिकादावूठ्, हौ च श्नः शानचः पूर्वे परत्त्वादूठि पश्चाद्धलन्तत्त्वाभावान्न शानच्, ( अखौनात् खौनीयात् ) आशिषि ( खव्यात् ) शेषं खचिवत्, अत्र क्वचिद्धेट् चेति पठ्यते, तत्र हेठतीत्यादि, हेठ विबाधायामिति शपि, श्रन्थादय उदात्ता पिषिवर्जम् क्लिशिस्तु विभाषितेड इषिरपि तकारादौ वार्त्तिकमतेन, सर्वे परस्मैपदिनः 59
