 ( उचयति उवोच, ऊचतुः, उवोचिथ, ) पिद्वचनेषु परत्वाद्गुणे, पश्चाद् द्विर्वचने "अभ्यासस्यासवर्णे'' इत्युवङ् यद्यपि द्विर्वचने गुणस्य स्थानिवत्त्वम्, तथाप्युत्तरखण्डस्य "वार्णादाङ्गं बलीय'' इति सवर्णदीर्घात्पूर्वे गुणे ऽभ्यासस्योवङ् ओचिता ओचिष्यति उच्यतु औच्यत्, उच्च्येत्, उच्यात्, ) लिडो ( मा भवानुचिचत् ओचिचिषति ) इत्यादावपि द्विर्वचनात्पूर्वं गुणः अत्र च ज्ञापकं "रलो व्युपधात्'' इत्यत्र हलादिग्रहणेन अजादेर्गुणार्थः कित्वपर्युदासः यदि च पूर्वं द्विर्वचनं स्याद् अधिकत्त्वेऽप्यनुपधात्वाद् गुणो न भविष्यतीति किं तेन ( ओचित्वा ) "न क्त्त्वा सेट्'' इत्यकित्त्वम् ( उचितः, औचित्यम् ) ब्राह्मणादित्वात् ष्यञ् ( औचिती ) ष्यञन्तात् षित्त्वान्ङीषि "हलस्तद्धितस्य'' इति यलोपः नियमेनोच्यतीति ( न्योकः ) शंकुः इगुपधलक्षणः कः नियमेनास्मि शकुन्तोः समवयन्तीति ( न्योजो ) वृक्षः, घञर्थे कः ( ओकः ) अच् "ओक उचः के'' इति निपातनात्कुत्वम् ( उल्बम् ) गुणश्च ( ओकः, ओकसी ) गृहम् असुनि बाहुलकात्कुत्वम् ( उल्बम् "उल्बादयश्च'' इति बप्रययः चकारस्य च लकारो गुणाभावश्च निपात्यते 115
