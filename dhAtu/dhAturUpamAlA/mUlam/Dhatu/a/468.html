प्रथ
 
प्रख्याने
 ( प्रथते पप्रथे प्रथितेत्यादि प्रथयति अपप्रथत् ) अत् स्मृदृत्वरप्रथम्रदस्तॄस्पशाम्इति चङ्परे णावभ्यासस्यात्वं मित्त्वापवादः ( अप्रथि अप्राथीत्यादि) घटिवत् ( पृथिवी ) "प्रथेः षिवन्सम्प्रसारणं च'' इति षिवन् षित्त्वान् ङीष् पृथिव्या निमित्तं संयोग उत्पातो वा वार्थिवः "सर्वभूमिवृथिवीभ्यामणौ'' इति अण् पृथिव्या ईश्वरः पृथिव्यां विदित इति वा पार्थिवः "तस्येश्वरः'' "तत्र विदितइति च'' इत्यण्, "पृथिव्या ञाञौप्राग्दीव्यतीयेष्वर्थेषु ञाञौ (पार्थिवम् ) अनयोः स्त्रियां विशेषः ( पार्थवा पार्थिवी पृथुः ) "प्रथिप्रदिम्भ्रस्जां सम्प्रसारणं सलोपश्च'' सलोपश्च'' इति कुप्रत्ययः सम्प्रसारणं च सलोपवचनं भृजत्यर्थम्_ पृथोर्भावः ( प्रथिमा ) "पृथ्वादिभ्य इमजिज्वा'' इति इमनिच् "र ऋतो हलोदः'' इति ऋकारस्य रशब्दः इमनिजभावे "इगन्ताच्च लघुपूर्वात्'' इत्यणि ( पार्थिवम् ) अत्र लघोः पूर्वत्वमिगपेक्षम् तच्चेह द्वयोरचोरानन्तर्येणावस्थानासम्भवादेकहला व्यवधानमाश्रीयते "वोतो गुणवचनात्'' इति ङीषि (पृथ्वी) गुणमुक्त्वा गुणिनि वर्त्तमानो गुणवचनः अयं कथादावपि अत्र के चित्पूथेति ऋककारोपधमपि पठन्ति यदाह वर्द्धमानः पृथिकन्क्षिन्दिक्षन्दिदृक्षादीनां घटादौ पाठः चिण्णमुलोर्वृद्ध्यर्थ इति अत्र स्वामी प्रथेः सम्प्रसारणविधानमार्ष मन्यते पुरुषकारोपि यदप्रथयत् तत्पृथिव्याः पृथिवीत्वमिति श्रौतस्य निर्वचनस्यार्षपक्षेणवाञ्जस्यमिति अन्यथा यदपर्थयदित्येव ब्रूयात् पुनश्चेक्तम् अथापि पृथेर्घटादिपाठे तत्फलं घटादयः षित इति षित्त्वादङि सति पृथेति रूपं द्रष्टव्यमिति व्यथादिस्थान्तः, पृथ प्रक्षेपइति चुरादौ 753
