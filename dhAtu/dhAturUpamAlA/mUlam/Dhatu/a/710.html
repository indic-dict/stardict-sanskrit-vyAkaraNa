प्रकथने
 ( ख्याति ) इत्यादि यातिवत् लुङि "अस्यतिवक्ति'' इत्यङि ( अख्यद् ) इति भवति तिपा निर्देशाद्यङ्लुकि अङ्भावात् सिचि (अचाख्यासीद्) इति भवति अस्यापि चक्षिङादेशवत् क्शादित्वं द्रष्टव्यम् इदमार्धधातुविषयमिति आत्रेयादयः अये तु सामान्येन मन्यन्ते (सखा) "समाने ख्यस्य चोदात्त'' इति समानशब्दउपपदे इन् प्रत्ययः, समानस्य सभावः, उदात्तश्च चशब्दाद्धातोर्यलोपः, प्रत्ययस्य डित्त्वं च "अनङ्सौ'' इत्यसम्बुद्धौ सौ अनङादेशो ङित्त्वादन्त्यस्य सम्बुद्धौ तु "ह्रस्वस्य गुणः'' इति गुणः अन्यत्र सर्वनामस्थाने "सख्युरसम्बुद्धौ'' इति णिद्वद्भावाद् वृद्धावायादेशः ( सखायौ ) इत्यादि सखीन् इत्यादि पतिशब्दवत् घिसंज्ञाऽसखीति पर्युदासान्न भवति स्त्रियां ( सखी ) "सख्यशिश्वीति भाषायाम्'' इति निपातनाद् ङीष् ( सख्या ) इत्यत्र "ख्यत्यात्परस्य'' इत्युत्वात्परत्वाद् नित्यत्वाच्च ङसिङसोर् "आणनद्याः'' इत्याटि कृतेऽकारस्य खीशब्दानन्तरत्वाभावाद् उत्वं न भवति (परमसखः) "राजाहस्ससिभ्यष्टच्'' इति टच् बहुव्रीहौ तु (बहुसखा, बहुसखायौ) इति "बहुव्रीहौ सङ्ख्येये'' इति डज् न भवति, अनङ्णिद्वद्भावौ "अङ्गाधिकारे तस्य तदुत्तरपदस्य (1) ( 1 )तदन्तस्य इति पाठो बहुषु पुस्तकेषु वर्तते च'' इति सख्यन्तस्यापि भवतः असखीति पर्युदासः केवलस्यैवेति तदन्तस्य घिसंज्ञायां बहुसखिनेत्यादौ नाभावादि भवति ( सुसखा, अतिसखा, किंसखा, असखा ) इत्यत्र "न पूजनात्'' "किमः क्षेपे'' "नञस्तत्पुरुषात्'' इति समासान्ताभावः लिङ्गविशिष्टपरिभाषया सखीशब्दान्तादस्मिन् समासान्ते कृते चाकृते च न विशेषः, यतः कृतेऽपि "भस्याढे तद्धिते'' इति पुंवद्भावे टित्त्वान्ङीपि ( परमसखी ) इत्येव रूपम् सख्युर्भावकर्मणी ( सख्यम् ) "सख्युर्य'' इति यः लिङ्गविशिष्टात्प्रत्ययेऽपीदमेव रूपम्, यतः पूर्ववत्पुंवद्भावेन भाव्यम् सम्पूर्वस्य सख्युः प्रयोगो नेति"समि ख्यः'' इत्यत्र न्यासादावुक्तम् 50
