पवने
 
पवनं पूतीकरणं
, सकर्मकोऽयम् तथा च पुनात्यग्निं, पुनीते आत्मानं कर्षकेण, व ( 2 ) ( 2 ) शालजान् इति पुस्तकान्तरे पाठः लजं पुपूषतइत्यादौ सकर्मकत्वं दृश्यते ( पुनाति पनासि पुनामि पुनीते पुनीषे पुने ) "प्वादीनां ह्रस्वः'' इति शिति परे ह्रस्वः ( पुपाव पुपुवतुः पुपविथ पुपुविव पुपुवे पुपुविरे पुपुविषे पुपुवे पविता पविष्यति पविष्यते पुनातु पुनीताम् पुनीष्व पुनै अपुनात् अपुनीताम् अपुनाः अपुनाम् अपुनीव अपुनीत अपुनाताम् अपुनीथाः अपुनिअपुनीवहि पुनीयात् पुनीयाः पुनीयाम् पुनीत पुनीयाताम् पुनीथाः पनीय ) आशिषि ( पूयात् पूयास्ताम् पविषीष्ट अपावीत् अपाविष्टाम् अपाविषुः अपविष्ट पुपूषति पुपूषते ) उगन्तत्वादनिट्त्वम् (पोपूयते पोपोति पावयति अपीवत् ) "ओः पुयण्जि'' इत्यभ्यासस्येत्वम् "दीर्घो लघोः'' इति दीर्घः ( पूत्वा पूतः पूनाः ) यवाः विनष्टा इत्यर्थः "पूञो विनाशे'' इति निष्ठानत्वम् शेषाः कृतः पवतिवत् 10
