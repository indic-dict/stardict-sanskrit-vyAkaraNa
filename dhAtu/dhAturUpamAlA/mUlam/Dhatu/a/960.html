वृहू
 
उद्यमने
 दन्त्योष्ठ्यादिरयम् तथा हि "मूलमस्या बर्हि'' "सक्थ्या बृहन्मा मे बलं विवृहत' इति प्रयुज्यते अन्ये तु बकरादिं पठन्ति (वृहति ववर्ह ववृहतुः ववर्हिथ ववर्ढ ) ऊदित्वादिड्विकल्पः झलि "हो ढः'' "झषस्तथोर्धो धः'' "ष्टुना ष्टुः'' "ढोढे लोपः'' ( वर्हिता वर्ढा वर्हिष्यति वर्क्ष्यति) "षढोः कस्सि'' इति कत्वम्, ( वृहतु अवृहत् वृह्यात् अवर्हीत् अवर्हिष्टाम् ) अनिट्पक्ष"शल इगुपधाद्'' इति क्सः ( अवृक्षत् अवृक्षताम् विवर्हिषति विवृक्षति ) "हलन्ताच्च'' इति कित्वान्न गुणः ( वरीवृह्यते वरीवर्ढि, ) इत्यादि ( वर्हयति अवीवृहत् अववर्हत् ) उरृद्वा ( वर्हित्वा वृड्ढ्वा ) "न क्त्वा सेट्'' इत्यकित्त्वमिटि ( वृढः वृढवान् ) "यस्य विभाषा'' इत्यानिट् "ढ्रलोप'' इति दीर्घः पूर्वाण इत्यत्र न भवति ( वहति ) वृंहतीति वृद्धौ शपि वर्हतइति तत्रैव प्राधान्ये त्रय एते ओष्ठ्यादयः 66
